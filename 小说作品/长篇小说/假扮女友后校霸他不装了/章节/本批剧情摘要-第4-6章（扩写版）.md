# 本批剧情摘要（第4-6章 扩写版）

**【本批范围】**
- 章节范围：第4章 – 第6章（共3章）
- 本批标题：奶茶与习惯、剥虾与入戏、画展与清水

**【主要情节推进】**
1. 江晚星回宿舍被室友盘问真假女友，她嘴硬说"还在演"，但心里反复回放霍景深的话。
2. 次日早上霍景深开始在宿舍楼下/教室门口送奶茶（三分糖去冰），并说"从今天起每天一杯"，江晚星从别扭到习惯不到一周。
3. 周三下午江晚星实验课晚了，霍景深等她并说"我要等"；周五约她周末校外吃饭，让她"穿好看点"。
4. 周六江晚星认真打扮（换裙子）赴约，两人在校外餐厅；霍景深记得她不吃辣、给她剥虾投喂、发现同校同学时搂她肩膀"别回头"。
5. 餐厅出门后霍景深约下周画展（林牧之巡回展），精准踩中江晚星喜好；她差点问"你是不是喜欢我"又咽回。
6. 周六画展约会，霍景深记得江晚星去年微博转发"在将爱未爱时最迷人"、朋友圈转发《春日》配文"春天会来的"。
7. 画展上霍景深说"以后不会让你一个人觉得孤独"，江晚星差点问出口时被学弟打断。
8. 画展散场，赵凯拉霍景深到一边说"你俩太清水了，不像情侣""大家觉得江晚星怕你""挡箭牌"。
9. 霍景深听后表情失落，问江晚星"真的怕我？"；送她到楼下说"老子不强迫女人，你要是觉得不自在随时可以说"。
10. 周一早上霍景深没出现，江晚星失落但自我说服"这很正常"。

**【人物状态 / 关系变化】**
- **江晚星**：被室友盘问时嘴硬"敬业而已"；连续几天被送奶茶从抗拒到习惯、甚至期待；周六认真打扮赴约；餐厅被投喂时心跳加速；画展上差点问"你是不是喜欢我"；被说"怕你"后尴尬；霍景深说"老子不强迫女人"后心酸又胀；周一他没出现感到失落。
- **霍景深**：每日送奶茶刷存在感；知道她口味（三分糖去冰、不吃辣）、记得她说冷；记得她微博/朋友圈转发内容；画展上说"以后不会让你一个人觉得孤独"；被赵凯说"晚星怕你"后表情失落、问"真的怕我？"；送她到楼下说"老子不强迫女人"；之后几天消失。

**【伏笔与未解悬念】**
- 霍景深记得江晚星去年微博转发、朋友圈转发，说明他关注她很久了，为后续"高中暗恋"真相做铺垫。
- 霍景深说"以后不会让你一个人觉得孤独"，暗示他想一直陪在她身边。
- 赵凯说"太清水""怕你""挡箭牌"，导致霍景深失落，为他之后几天消失埋下伏笔。
- 霍景深说"老子不强迫女人，你要是觉得不自在随时可以说"，暗示他在试探她的真实想法，也为后续消失做铺垫。
- 周一早上霍景深没出现，为第7章"他被人打了"埋下伏笔。

**【本批爽点 / 钩子简要】**
- **第4章**：爽点 小——校霸只对女主好、每日奶茶、记得口味、等她下课、说"我要等"；章尾钩子 情感——周末约会、穿好看点、入戏太深。
- **第5章**：爽点 小——记得口味、剥虾投喂、搂肩护短、记得画家喜好、记得她说冷；章尾钩子 情感——约画展、江晚星差点问"喜欢我"、心动但自我说服"不能当真"。
- **第6章**：爽点 小——记得女主微博与转发、朋友圈、画展约会、"以后不会让你一个人觉得孤独"；章尾钩子 情感/悬念——"真的怕我？"、霍景深失落、"老子不强迫女人"、江晚星不敢当真；周一早上他没出现。

**【时间线 / 场景线】**
- **时间跨度**：第二天早上开始送奶茶 → 接下来几天连续送奶茶 → 周五约周末 → 周六校外餐厅约会 → 周一送奶茶记得她说冷 → 周六画展约会 → 周一早上霍景深消失。
- **主要场景顺序**：宿舍（室友盘问） → 宿舍楼下/教室门口（连日送奶茶） → 校外餐厅（剥虾投喂、搂肩"别回头"、约画展） → 回宿舍室友八卦 → 周一送奶茶 → 周六画展（林牧之巡回展） → 画展外赵凯与霍景深对话 → 送她回宿舍楼下说"老子不强迫女人" → 周一早上他没出现。
