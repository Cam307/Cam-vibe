# 本批剧情摘要（第1-3章 扩写版）

**【本批范围】**
- 章节范围：第1章 – 第3章（共3章）
- 本批标题：Zero、嫂子好、他不配

**【主要情节推进】**
1. 江晚星在校园论坛评论霍景深照片为"Zero"被顶成热评，次日被霍景深在教学楼走廊堵住，要求假扮女友辟谣。
2. 霍景深带江晚星去二食堂"官宣"，被一群男生齐喊"嫂子好"；食堂内眼镜男向霍景深表白，霍景深当众"假吻"江晚星（实为吻自己手指）破谣。
3. 次日早上霍景深在计科楼接江晚星上课，两人关系初步绑定；下午送回宿舍时，渣男前任周锐纠缠并言语侮辱江晚星。
4. 霍景深暴揍周锐，撂下"你不配"；事后江晚星在校医室给霍景深上药，霍景深解释打人有分寸，并叮嘱她"遇到事要告诉我"。
5. 论坛发酵"校霸当众示爱"帖，霍景深称"你更得对我负责了吧"加深绑定；次日早上霍景深又来接她并送奶茶，日常绑定开始。

**【人物状态 / 关系变化】**
- **江晚星**：从论坛吃瓜路人到被迫假扮女友，经历社死场面（"嫂子好"/当众"假吻"）；最初抗拒想结束关系，被霍景深反问"就这么迫不及待躲开我"后继续演；周锐事件后对霍景深产生好感，从担心他背处分到被他"原来是心疼我"撩到；次日早上习惯了他的接送和奶茶，心动萌芽但自我说服是"演戏需要"。
- **霍景深**：表面校霸、清冷不好惹，实则是从高中就暗恋江晚星的套路王；以假扮女友为由将江晚星绑在身边，对别人冷淡、对女主双标（接送、打饭、挡桃花、揍渣男、送奶茶）；周锐事件护短人设立住，以"心疼我"试探江晚星反应，并强调"遇到事要告诉我"；章末开始日常接送送奶茶，将假扮关系持续下去。
- **周锐**：渣男前任，出轨江晚星前室友；纠缠江晚星时被霍景深暴揍，被警告"你不配"；后续不会再骚扰。

**【伏笔与未解悬念】**
- 霍景深说"以后有机会再告诉你"关于他如何成校霸的事，为后续身世线和高中暗恋线埋伏笔。
- 霍景深知道江晚星喜欢三分糖去冰奶茶，但他解释为"猜的"，暗示他其实关注她很久了。
- 霍景深说"我以前……练过。知道哪儿打得多疼，但不容易留疤"，暗示他的身手和"校霸"名号有背景故事。
- 论坛上"校霸当众示爱"的帖子持续发酵，为后续舆论反转和身份揭秘做铺垫。

**【本批爽点 / 钩子简要】**
- **第1章**：爽点 小——女主嘴硬又怂的反差（"卖艺不卖身"/吐槽怕他被传gay）；章尾钩子 情感——江晚星"不那么讨厌明天的到来了"，暗示心动萌芽。
- **第2章**：爽点 小——官宣场面、"嫂子好"、男主当众假吻破谣护短；章尾钩子 危机——周锐出现，霍景深眼神冷了下来，为第3章打脸渣男做铺垫。
- **第3章**：爽点 小——打脸渣男前任"你不配"、男主护短、女主被撩"原来是心疼我"；章尾钩子 情感——次日早上霍景深又来接她并送奶茶，江晚星从抗拒到习惯，心动萌芽，为后续日常绑定做铺垫。

**【时间线 / 场景线】**
- **时间跨度**：第一天晚上论坛热评 → 第二天早上堵门、中午食堂"嫂子好"、下午眼镜男表白与假吻、傍晚周锐事件与校医室上药、晚上论坛发酵 → 第三天早上接送送奶茶。
- **主要场景顺序**：宿舍刷论坛 → 教学楼走廊堵门 → 二食堂"嫂子好"与假吻破谣 → 次日早上计科楼接上课 → 宿舍区路口周锐纠缠 → 校医室上药 → 宿舍楼下看论坛 → 次日早上宿舍楼下接上课送奶茶。
