# 《假扮女友后校霸他不装了》长篇创作项目

## 项目说明

本长篇基于**对标短篇**《校霸套路深》（知乎盐选·因你心动：你与月亮，皆是浪漫）的关系模式与情节套路进行扩展创作，严格遵循**长篇小说-process**流程。

- **人名与场景**：与短篇**全部不重复**，采用新创名称。
- **目标字数**：30–50 万字，150–200 章。
- **单章字数**：不少于 2800 字（不含标点），建议 3000 字。
