# 第34–36章详细细纲

> 承接本批剧情摘要（第31–33章）；第33章结尾霍景深被父亲断经济来源、身无分文，江晚星说不在乎、愿意和他一起面对。
> 大纲对应：第34–36章为两人面对没有经济来源的生活、霍景深打工赚钱、感情更坚定。

---

## 第34章：身无分文

### 本章基本信息
- 章节号：第34章
- 章节标题：身无分文
- 在主线中的位置：霍景深被断经济来源后的第一天，面对现实困境
- 章节类型：现实困境+情感考验
- 承接上章：第33章霍景深被断经济来源、身无分文
- 开启下章：霍景深开始找工作、打工赚钱
- 预计字数：不少于4000字（不含标点）

### 核心目标
- 情节目标：霍景深第二天醒来，发现自己真的身无分文——银行卡被停、钱包里没钱、连早餐都买不起。江晚星来给他送早餐，发现他的窘迫。
- 情感目标：霍景深的自尊受挫但不想让江晚星担心；江晚星心疼他、想帮他但怕伤他自尊。
- 爽点：小暖——江晚星主动帮忙、两人互相体谅。
- 钩子：霍景深决定找工作、打工赚钱。

### 人物状态
- **霍景深**：第一次体验没钱的生活，自尊受挫但不想让江晚星知道，开始想解决办法。
- **江晚星**：担心霍景深，想帮他但怕伤他自尊，小心翼翼地照顾他。
- **赵凯**：得知霍景深被断经济来源，想借钱但被拒绝。

### 场景要点
1. 男生宿舍：霍景深醒来，发现自己真的没钱了。
2. 买早餐尴尬：霍景深想买早餐发现手机支付被停，现金也没有。
3. 江晚星出现：江晚星来送早餐，发现他的窘迫。
4. 午餐困境：霍景深没钱吃饭，赵凯想帮他但被他拒绝。
5. 江晚星心疼：偷偷给霍景深充饭卡、买奶茶，霍景深发现后感动又愧疚。

### 写作重点
1. 霍景深的窘迫——第一次体验没钱的生活，从富家子弟到身无分文。
2. 自尊与现实的冲突——不想让江晚星看到自己狼狈的样子，但又无法隐瞒。
3. 江晚星的体谅——想帮他但怕伤他自尊，小心翼翼地照顾。
4. 情感深化——两人一起面对困难，感情更坚定。

## 第35章：打工

### 本章基本信息
- 章节号：第35章
- 章节标题：打工
- 在主线中的位置：霍景深开始找工作、打工赚钱
- 章节类型：现实线+成长线
- 承接上章：第34章霍景深身无分文、面对现实困境
- 开启下章：霍景深打工过程中的困难和江晚星的陪伴
- 预计字数：不少于4000字（不含标点）

### 核心目标
- 情节目标：霍景深开始找兼职工作——他从未打过工，不知道怎么找工作，处处碰壁。最后找到一份奶茶店兼职。
- 情感目标：霍景深放下身段、从零开始；江晚星陪他找工作、鼓励他。
- 爽点：小暖——两人一起面对困难、江晚星不离不弃。
- 钩子：霍景深开始工作、遇到困难。

### 人物状态
- **霍景深**：第一次找工作、第一次打工，从笨拙到慢慢适应。
- **江晚星**：陪霍景深找工作、鼓励他、在他工作累时陪着他。
- **奶茶店老板**：一开始看霍景深不像打工的，后来发现他很认真。

### 场景要点
1. 找工作：霍景深不知道怎么找工作，在网上搜、到处问。
2. 处处碰壁：很多店不招人，或者觉得霍景深不像打工的。
3. 江晚星陪伴：江晚星陪他一起找工作、鼓励他。
4. 找到工作：终于找到一家奶茶店招兼职。
5. 第一天工作：霍景深第一次打工，手忙脚乱、被老板骂。
6. 江晚星心疼：等霍景深下班、给他买水、问他累不累。

### 写作重点
1. 霍景深的转变——从富家子弟到打工仔，放下身段。
2. 找工作的困难——他从未打过工，什么都不知道。
3. 江晚星的陪伴——陪他找工作、鼓励他、心疼他。
4. 第一天工作的笨拙——手忙脚乱、被骂、慢慢适应。
5. 情感深化——两人一起面对困难、感情更坚定。

## 第36章：累了但值得

### 本章基本信息
- 章节号：第36章
- 章节标题：累了但值得
- 在主线中的位置：霍景深打工一周后的疲惫与坚持
- 章节类型：日常线+情感深化
- 承接上章：第35章霍景深开始打工
- 开启下章：霍景深逐渐适应打工生活、两人感情更稳定
- 预计字数：不少于4000字（不含标点）

### 核心目标
- 情节目标：霍景深打工一周后，每天上课+工作，很累但咬牙坚持。江晚星心疼他、想劝他休息但知道他不会停。某天霍景深下班很晚，江晚星等他，两人一起走回学校。
- 情感目标：霍景深虽然累但很充实——他靠自己赚钱养自己、也能养江晚星；江晚星心疼又骄傲。
- 爽点：小暖+小爽——霍景深第一次拿到工资、给江晚星买礼物。
- 钩子：霍景深拿到工资、给江晚星买她喜欢的东西。

### 人物状态
- **霍景深**：每天上课+打工，很累但坚持，慢慢适应打工生活。
- **江晚星**：心疼霍景深、想帮他但知道他不会接受，只能默默陪伴。
- **奶茶店老板**：对霍景深改观，夸他认真、能吃苦。

### 场景要点
1. 一周后的疲惫：霍景深每天上课+打工，黑眼圈重了、瘦了。
2. 江晚星心疼：想劝他休息但知道他不会停，只能默默陪伴。
3. 工作中的小插曲：有顾客刁难、老板维护霍景深、霍景深学会应对。
4. 第一次发工资：霍景深拿到工资，虽然不多但很有成就感。
5. 给江晚星买礼物：霍景深用工资给江晚星买她喜欢的东西（画画用的颜料/画笔）。
6. 两人走回学校：深夜，江晚星等霍景深下班，两人一起走回学校，霍景深说"累了但值得"。

### 写作重点
1. 霍景深的疲惫——每天上课+打工，很累但咬牙坚持。
2. 江晚星的心疼——想劝他休息但知道他不会停，只能默默陪伴。
3. 第一次发工资——虽然不多但很有成就感、靠自己的努力。
4. 给江晚星买礼物——记得她喜欢什么、用自己赚的钱给她买。
5. 情感深化——两人一起面对困难、感情更坚定。
6. 钩子——霍景深说"累了但值得"。

江晚星："你……你累不累？"
