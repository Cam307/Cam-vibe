# 第28–30章详细细纲

> 承接本批剧情摘要（第25–27章）；第27章结尾江晚星站在窗帘后看着霍景深、心疼但不敢下去、心更乱了。
> 大纲对应：第28–30章为程慕远告诉江晚星更多真相（霍景深处分为她、卫生巾和纸条也是他送的）、江晚星重新审视和霍景深的关系、赵凯继续助攻。

---

## 第28章：真相

### 本章基本信息
- 章节号：第28章
- 章节标题：真相
- 在主线中的位置：程慕远告诉江晚星霍景深处分为她、她震惊
- 章节类型：情节推进+情感转折
- 承接上章：第27章江晚星心疼霍景深但不敢去看他
- 开启下章：江晚星开始重新审视与霍景深的关系
- 预计字数：不少于4000字（不含标点）

### 核心目标
- 情节目标：程慕远找到江晚星、告诉她霍景深处分的真相——入学时打了学长是为了帮她、他背了处分。江晚星震惊、不知道该信谁。
- 情感目标：江晚星从"他利用我"转向"他为我做了很多"，矛盾加剧。
- 爽点：中爽——原来霍景深处分是为了她。
- 钩子：江晚星开始重新审视与霍景深的关系、想知道更多真相。

### 人物状态
- **江晚星**：心疼霍景深、程慕远告诉她真相、她震惊矛盾。
- **程慕远**：助攻型、告诉江晚星霍景深处分的真相、让她重新审视关系。
- **赵凯**：助攻型、告诉江晚星霍景深的伤情、让她心疼。

### 场景要点
1. 宿舍：江晚星在宿舍发呆、许悠悠和孙晓劝她去看霍景深、她矛盾。
2. 程慕远来找：程慕远在楼下等江晚星、让她下来一趟、有重要的事要告诉她。
3. 揭示真相：程慕远告诉江晚星、霍景深处分是因为打了学长、那个学长当年骚扰过她。
4. 江晚星震惊：江晚星想起当年的事、原来霍景深是为她出头、她不知道该说什么。
5. 矛盾：江晚星想起自己拉黑了霍景深、心疼又愧疚、想去找他但又怕。

### 写作重点
1. 程慕远的话要真诚——他不是要帮霍景深追她、只是不想她误会。
2. 江晚星的反应要层层递进——从震惊到怀疑到愧疚。
3. 回忆杀——当年学长骚扰她的场景、霍景深打学长的场景（程慕远转述）。
4. 心理描写——江晚星不知道该信谁、矛盾加剧。

## 第29章：还有更多

### 本章基本信息
- 章节号：第29章
- 章节标题：还有更多
- 在主线中的位置：程慕远告诉江晚星卫生巾和纸条也是霍景深送的、她彻底震惊
- 章节类型：情节推进+情感爆发
- 承接上章：第28章江晚星知道霍景深处分为她、震惊矛盾
- 开启下章：江晚星决定去找霍景深、两人对峙和好
- 预计字数：不少于4000字（不含标点）

### 核心目标
- 情节目标：程慕远继续告诉江晚星、卫生巾和纸条也是霍景深送的、不是他送的。江晚星震惊、原来自己一直误会了。
- 情感目标：江晚星从"他利用我"彻底转向"他一直对我好"、愧疚和心疼交织。
- 爽点：大爽——原来霍景深一直默默对她好。
- 钩子：江晚星决定去找霍景深、把话说清楚。

### 人物状态
- **江晚星**：知道卫生巾和纸条也是霍景深送的、彻底震惊、愧疚心疼、决定去找他。
- **程慕远**：继续助攻、澄清卫生巾和纸条不是他送的、是霍景深。
- **赵凯**：助攻型、告诉江晚星霍景深的近况、让她心疼。

### 场景要点
1. 继续对话：程慕远和江晚星继续聊、他提到卫生巾和纸条的事。
2. 澄清误会：程慕远说卫生巾和纸条不是他送的、是霍景深、他从未承认过。
3. 江晚星回忆：江晚星想起当年的事、她一直以为是程慕远送的、原来自己一直误会了。
4. 心理爆发：江晚星愧疚、心疼、想去找霍景深、问问他为什么不告诉她。
5. 决定：江晚星决定去找霍景深、把话说清楚。

### 写作重点
1. 卫生巾和纸条的回忆——江晚星当年生理期尴尬、有人塞了卫生巾和纸条、她以为是程慕远。
2. 程慕远的澄清——他从未承认过、是江晚星自己误会的。
3. 江晚星的反应——从震惊到愧疚到心疼。
4. 心理描写——她不知道该怎么面对霍景深、愧疚又心疼。

## 第30章：我想见你

### 本章基本信息
- 章节号：第30章
- 章节标题：我想见你
- 在主线中的位置：江晚星去找霍景深、两人在宿舍楼下对峙、和好
- 章节类型：情节高潮+情感爆发
- 承接上章：第29章江晚星决定去找霍景深
- 开启下章：两人和好、关系从假扮转为真在一起
- 预计字数：不少于4000字（不含标点）

### 核心目标
- 情节目标：江晚星去找霍景深、在宿舍楼下找到他、两人对峙。江晚星问他"处分是你为了我背的""卫生巾和纸条是你送的对吗"、霍景深承认。江晚星说"为什么不告诉我"、霍景深"我不想你觉得欠我"。江晚星哭、说"我不想欠你、我喜欢你"。两人和好。
- 情感目标：江晚星从误会到理解、从愧疚到心动；霍景深从被误解到被理解、两人关系突破。
- 爽点：大爽+情感高潮——江晚星表白、两人和好。
- 钩子：两人真正在一起、关系从假扮转为真恋人。

### 人物状态
- **江晚星**：去找霍景深、问清楚所有事、表白、两人和好。
- **霍景深**：在宿舍楼下等江晚星、以为她不会来了、她来了、两人对峙、和好。

### 场景要点
1. 江晚星去找霍景深：她在宿舍楼下等他、香樟树下、她站了很久。
2. 霍景深出现：霍景深从宿舍出来、看见江晚星、愣住。
3. 对峙：江晚星问他"处分是你为了我背的""卫生巾和纸条是你送的对吗"。
4. 霍景深承认：霍景深承认、说"我不想让你知道""我不想你觉得欠我"。
5. 江晚星表白：江晚星哭、说"我不想欠你、我喜欢你"。
6. 和好：霍景深愣住、然后抱住她、说"我也喜欢你"。

### 写作重点
1. 对峙的张力——江晚星的问题层层递进、霍景深的回答真诚。
2. 霍景深的反应——他以为江晚星不会来了、她来了、他愣住。
3. 表白的场景——江晚星的表白要真诚、不是冲动、是经过思考的。
4. 和好的场景——霍景深抱住她、两人的情感爆发。

### 第30章对话示例（供写章时参考）

```
江晚星："处分是你为了我背的，对吗？"

霍景深愣住："谁告诉你的？"

江晚星："这不重要。重要的是，你为什么不告诉我？"

霍景深："我不想让你知道。"

江晚星："为什么？"

霍景深："我不想让你觉得欠我。"

江晚星："可你不说，我怎么知道？"

霍景深："你知道了又能怎么样？"

江晚星："我知道了，我就能……"她顿了顿，"我就能告诉你，我不想欠你。"

霍景深："那你来干什么？"

江晚星："我来告诉你……"她深呼吸，"我喜欢你。"

霍景深彻底愣住。

江晚星："不是假扮女友的那种喜欢，是真的喜欢。"

霍景深没说话，只是盯着她。

江晚星："你……你不信吗？"

霍景深突然伸手，把她拉进怀里。

"我信。"他说，声音有些哑，"我信。"

江晚星靠在他怀里，听见他的心跳——很快，像她的心一样。

"我也喜欢你。"他说。

江晚星的眼泪掉下来，打湿了他的衣服。

```
