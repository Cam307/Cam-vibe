# 第22–24章详细细纲

> 承接本批剧情摘要（第16–18章）及第19–21章；第21章结尾江晚星决定为霍景深准备生日惊喜、问赵凯。
> 大纲对应：第22–24章为误会爆发线，江晚星准备生日惊喜→偷听订婚对话→心沉拉黑。

---

## 第22章：问赵凯

### 本章基本信息
- 章节号：第22章
- 章节标题：问赵凯
- 在主线中的位置：生日准备+偷听前夕
- 章节类型：情节铺垫+小甜
- 承接上章：第21章江晚星决定为霍景深准备生日惊喜
- 开启下章：江晚星准备礼物，决定去霍景深宿舍
- 预计字数：不少于4000字（不含标点）

### 核心目标
- 情节目标：江晚星找到赵凯询问霍景深喜好；赵凯透露霍景深很在意她；江晚星开始准备礼物。
- 情感目标：江晚星对霍景深的在意加深，准备礼物时心里是甜的；与赵凯对话中侧面反映霍景深的真心。
- 爽点：无，偏铺垫和小甜。
- 钩子：江晚星准备完礼物，打算10月3号前送到他宿舍——为第23章去宿舍路上偷听做铺垫。

### 人物状态
- **江晚星**：在意霍景深，想给他准备生日惊喜；从赵凯处得知霍景深很在意她，心里更甜。
- **赵凯**：被江晚星问霍景深喜好；说漏嘴透露霍景深很在意江晚星；助攻型。
- **霍景深**：本章未直接出场，通过赵凯的台词侧面体现他的在意。

### 场景要点
1. 校园某处（篮球场边/食堂）：江晚星找到赵凯，询问霍景深喜好。
2. 赵凯透露：霍景深最近忙但总念叨她；霍景深在意她。
3. 江晚星心里甜，开始准备礼物——决定送亲手做的礼物（手织围巾/手工钱包/等）。
4. 时间推进：几天过去，生日将近，江晚星准备完毕，打算送去宿舍。

## 第23章：生日前夕

### 本章基本信息
- 章节号：第23章
- 章节标题：生日前夕
- 在主线中的位置：礼物准备完成→去宿舍路上→偷听前夕
- 章节类型：情感铺垫+过渡
- 承接上章：第22章江晚星准备礼物
- 开启下章：江晚星去霍景深宿舍，路过楼后公园
- 预计字数：不少于4000字（不含标点）

### 核心目标
- 情节目标：10月2日晚/10月3号清晨，江晚星准备完礼物；决定提前去霍景深宿舍给他惊喜；出发去男生宿舍区。
- 情感目标：江晚星满怀期待和甜蜜，想给他惊喜；心情从紧张到期待。
- 爽点：无，偏情感铺垫。
- 钩子：江晚星走到男生宿舍楼后，准备拐进去时听见公园里的声音——为第24章偷听做铺垫。

### 人物状态
- **江晚星**：满怀期待，礼物准备完毕；想去霍景深宿舍给惊喜；心情甜又紧张。
- **霍景深**：本章未直接出场，但他的声音会在偷听场景出现。

### 场景要点
1. 宿舍：江晚星包装礼物，心里想着霍景深收到会是什么表情。
2. 出发：傍晚/晚上，江晚星拿着礼物往男生宿舍区走。
3. 路上：心情甜蜜，想着怎么给他惊喜（直接敲门？等他回来？）。
4. 到达宿舍区：快到宿舍楼时，路过楼后小公园；准备拐进去时听见说话声。
5. 停步：声音熟悉（白雨薇、赵凯、霍景深），江晚星好奇停下脚步。

## 第24章：偷听

### 本章基本信息
- 章节号：第24章
- 章节标题：偷听（或"随便抓个人玩玩"）
- 在主线中的位置：误会爆发点
- 章节类型：情节爆发+情感转折
- 承接上章：第23章江晚星路过楼后公园，听见声音
- 开启下章：江晚星心沉拉黑霍景深
- 预计字数：不少于4000字（不含标点）

### 核心目标
- 情节目标：江晚星躲在公园树后/灌木后，偷听白雨薇、赵凯和霍景深的对话。
- 对话内容：
  - 白雨薇："那个女生怎么回事？你最近天天往她那边跑。"
  - 霍景深：（沉默片刻）
  - 白雨薇："霍爷爷那边我帮你压着，但你得知道，订婚典礼如期举行。"
  - 霍景深："嗯。"
  - 白雨薇："那她呢？江晚星？"
  - 霍景深：（冷漠）"随便抓个人玩玩罢了。"
- 情感目标：江晚星从期待到震惊到心沉；手里的礼物变得沉重；转身离开，拉黑霍景深。
- 爽点：无，偏虐和误会。
- 钩子：江晚星拉黑霍景深，心寒离开——为第25-26章江晚星躲霍景深、程慕远出现做铺垫。

### 人物状态
- **江晚星**：满怀期待去送礼物→偷听到对话→震惊→心沉→失望→转身离开→拉黑霍景深。
- **霍景深**：和白雨薇对话；说"随便抓个人玩玩罢了"（实际是在演戏/保护江晚星，但她不知道）。
- **白雨薇**：试探霍景深对江晚星的态度；确认订婚如期举行。
- **赵凯**：在场，欲言又止，知道霍景深是在演戏但不敢说破。

### 场景要点
1. 江晚星停在公园外，躲在树后/灌木后。
2. 听见对话：
   - 白雨薇的声音："那个女生怎么回事？"
   - 霍景深的沉默
   - 白雨薇："订婚典礼如期举行。"
   - 霍景深："嗯。"
   - 白雨薇："那她呢？江晚星？"
   - 霍景深："随便抓个人玩玩罢了。"
3. 江晚星反应：
   - 震惊，不敢相信
   - 心沉到谷底
   - 手里的礼物变得沉重
   - 想冲进去质问，但腿像灌了铅
   - 最终转身离开
4. 回宿舍路上/宿舍内：
   - 江晚星把礼物收起来/扔掉
   - 拉黑霍景深的微信、电话
   - 心里冰冷，原来自己真的只是"随便抓的人"
5. 霍景深不知情，还在和白雨薇周旋（保护江晚星的手段）。

## 写作重点提示

### 第22章重点
1. 赵凯助攻：通过赵凯的台词体现霍景深的在意
2. 江晚星的心理：从"想准备礼物"到"准备过程中心里甜"
3. 小细节：江晚星准备礼物时的用心（选礼物、包装、写卡片等）

### 第23章重点
1. 江晚星的期待：心里想着霍景深的反应
2. 时间推进：从准备到出发
3. 场景转换：从宿舍到路上到宿舍区
4. 钩子铺垫：走到楼后公园，听见声音

### 第24章重点（关键章）
1. 偷听场景的张力：江晚星躲在后面，心跳、紧张
2. 对话的杀伤力：
   - "订婚典礼如期举行"——第一刀
   - "随便抓个人玩玩罢了"——第二刀，致命
3. 江晚星的情绪变化：
   - 期待 → 震惊 → 不敢相信 → 心沉 → 失望 → 心寒
4. 礼物细节：手里准备送的礼物，此刻变得讽刺
5. 拉黑动作：回到宿舍后拉黑，决绝

### 第24章霍景深台词的设计注意
- 霍景深说"随便抓个人玩玩罢了"时，实际是在演戏/保护江晚星（不让白雨薇盯上她）
- 但江晚星不知道，她以为是真的
- 这种信息差是误会的核心
- 霍景深的声音要冷漠、不在意（演戏），但读者要通过后续知道他是为了保护她

## 第24章对话示例（供写章时参考）

```
江晚星停在灌木后，手里的礼物袋攥紧了。

公园里有三个人。

她听见了白雨薇的声音："霍景深，那个女生怎么回事？你最近天天往她那边跑。"

霍景深没说话。

白雨薇又道："霍爷爷那边我帮你压着，但你得知道，订婚典礼如期举行。"

霍景深："嗯。"

一个字，冷漠，不在意。

江晚星的心，慢慢沉了下去。

白雨薇："那她呢？江晚星？"

霍景深的声音，更冷了："随便抓个人玩玩罢了。"

江晚星握着礼物袋的手，僵住了。

……

她转身就走。

脚步很快，像在逃离什么。

回到宿舍，她拿出手机，点开霍景深的头像。

拉黑。

微信，拉黑。

电话，拉黑。

所有能联系的方式，全部拉黑。
```
