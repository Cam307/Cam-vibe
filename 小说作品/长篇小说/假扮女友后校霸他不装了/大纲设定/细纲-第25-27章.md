# 第25–27章详细细纲

> 承接本批剧情摘要（第22–24章）；第24章结尾江晚星矛盾又痛苦、最终决定不去见霍景深、但心还是放不下。
> 大纲对应：第25–27章为江晚星躲霍景深、霍景深找她、两人在校园里僵持对峙、江晚星要求分开一段时间、霍景深被父亲打伤。

---

## 第25章：躲

### 本章基本信息
- 章节号：第25章
- 章节标题：躲
- 在主线中的位置：江晚星躲霍景深、霍景深开始找她
- 章节类型：情节推进+情感拉扯
- 承接上章：第24章江晚星决定不去见霍景深、但心放不下
- 开启下章：霍景深在校园里找到江晚星、两人僵持
- 预计字数：不少于4000字（不含标点）

### 核心目标
- 情节目标：江晚星开始躲霍景深——不去他常去的地方（食堂、球场、宿舍楼下）、不接陌生电话、看到他就绕道走。霍景深发现江晚星在躲他，开始找她。赵凯告诉他江晚星拉黑了他，霍景深才知道江晚星误会了。
- 情感目标：江晚星内心矛盾——想躲他但又放不下；霍景深察觉她在躲他、开始着急。
- 爽点：无，偏情感拉扯。
- 钩子：霍景深在图书馆/教学楼找到江晚星、两人对峙。

### 人物状态
- **江晚星**：躲霍景深，不去他常去的地方，看到他就绕道走；内心矛盾——想躲但又放不下。
- **霍景深**：发现江晚星在躲他，开始找她；赵凯告诉他江晚星拉黑了他，他才知道江晚星误会了；想解释但找不到她。
- **赵凯**：助攻型，告诉霍景深江晚星拉黑了他、建议他主动去找她。

### 场景要点
1. 宿舍/校园：江晚星开始躲霍景深——不去食堂（让室友带饭）、不去球场附近、看到他就绕道走。
2. 霍景深视角：霍景深在宿舍楼下等江晚星、她没下来；他去食堂找她、她故意避开；他找赵凯问怎么回事、赵凯说她拉黑了他。
3. 霍景深反应：霍景深才知道江晚星误会了；他开始在校园里找她。
4. 找到她：霍景深在图书馆/教学楼找到江晚星、两人对峙。

## 第26章：找到你了

### 本章基本信息
- 章节号：第26章
- 章节标题：找到你了
- 在主线中的位置：霍景深找到江晚星、两人僵持对峙
- 章节类型：情节高潮+情感爆发
- 承接上章：第25章霍景深在校园里找江晚星
- 开启下章：江晚星要求分开一段时间、霍景深被父亲打伤
- 预计字数：不少于4000字（不含标点）

### 核心目标
- 情节目标：霍景深在图书馆找到江晚星、两人在图书馆外对峙。霍景深问她为什么躲他、江晚星说他有未婚妻为什么要来找她。霍景深想解释、但江晚星不想听、转身离开。霍景深拉住她、她甩开、两人不欢而散。
- 情感目标：霍景深着急但不知道怎么解释；江晚星心寒又心软、但最终还是决定分开。
- 爽点：无，偏虐和僵持。
- 钩子：霍景深被父亲叫回家、被打伤。

### 人物状态
- **江晚星**：被霍景深找到、不想听他解释、心寒又心软、最终转身离开。
- **霍景深**：找到江晚星、想解释但她不想听、被拒绝后失魂落魄、被父亲叫回家。

### 场景要点
1. 图书馆：江晚星在图书馆看书、霍景深找到她。
2. 图书馆外：两人走出来、霍景深问她为什么躲他。
3. 对峙：江晚星说"你有未婚妻为什么要来找我"、霍景深想解释、江晚星说"我不想听"。
4. 不欢而散：江晚星转身离开、霍景深想拉住她、她甩开、两人不欢而散。
5. 霍景深被叫回家：霍景深失魂落魄、接到父亲电话、被叫回家、暗示会被打。

## 第27章：受伤

### 本章基本信息
- 章节号：第27章
- 章节标题：受伤
- 在主线中的位置：霍景深被父亲打伤、江晚星从赵凯处得知、心疼但矛盾
- 章节类型：情节高潮+情感转折
- 承接上章：第26章两人不欢而散、霍景深被叫回家
- 开启下章：江晚星得知霍景深受伤、想去看他但又矛盾
- 预计字数：不少于4000字（不含标点）

### 核心目标
- 情节目标：霍景深回到家、父亲因为白雨薇的事和他吵架、父亲动手打他（但打得不重、霍景深没还手）。赵凯找到江晚星、告诉她霍景深被父亲打了、受伤了。江晚星心疼但矛盾、最终决定不去看他。
- 情感目标：霍景深被打但没还手、心灰意冷；江晚星得知后心疼、想去但又忍住。
- 爽点：无，偏虐。
- 钩子：江晚星最终还是想去看看他——为第28章及后续做铺垫。

### 人物状态
- **江晚星**：从赵凯处得知霍景深被父亲打了、受伤了；心疼但矛盾、最终决定不去看他、但心更乱了。
- **霍景深**：被父亲打、没还手、心灰意冷；伤得不重但心里受伤。

### 场景要点
1. 霍景深家：霍景深回到家、父亲因为他和白雨薇的事发火、父亲动手打他、霍景深没还手、被打伤。
2. 江晚星视角：江晚星在宿舍、赵凯来找她、告诉她霍景深被父亲打了、受伤了。
3. 江晚星反应：江晚星心疼、想去看他、但又忍住、最终决定不去。
4. 钩子：江晚星躺在床上睡不着、脑子里全是霍景深被打的画面、她最终还是想去看看他。

## 写作重点提示

### 第25章重点
1. 江晚星躲霍景深的细节——不去食堂、绕道走、看到他就躲。
2. 霍景深的视角——在宿舍楼下等她、她没来；去食堂找她、她避开；找赵凯问怎么回事。
3. 赵凯助攻——告诉霍景深江晚星拉黑了他、建议他主动去找她。
4. 霍景深的反应——他才知道江晚星误会了、开始着急。

### 第26章重点（关键章）
1. 霍景深找到江晚星的场景——图书馆、两人对峙。
2. 对话的张力——霍景深问她为什么躲他、她说"你有未婚妻"。
3. 霍景深想解释——但江晚星不想听、她说"我不想听"。
4. 两人不欢而散——江晚星转身离开、霍景深想拉住她、她甩开。
5. 霍景深的反应——失魂落魄、接到父亲电话、被叫回家、暗示会被打。

### 第27章重点（关键章）
1. 霍景深被打的场景——父亲因为他和白雨薇的事发火、父亲动手打他、霍景深没还手。
2. 打得不要太重——但要打出伤、让赵凯能告诉江晚星。
3. 赵凯找江晚星——告诉她霍景深被父亲打了、受伤了。
4. 江晚星的反应——心疼、想去看他、但又忍住、最终决定不去、但心更乱了。
5. 钩子——江晚星最终还是想去看看他。

### 第26-27章对话示例（供写章时参考）

**第26章对峙场景：**
```
霍景深："你为什么要躲我？"

江晚星："我没躲。"

霍景深："那你为什么不接我电话？为什么不回我消息？"

江晚星："我拉黑你了。"

霍景深愣住："为什么？"

江晚星："你不是有未婚妻吗？为什么还要来找我？"

霍景深："晚星，你听我解释……"

江晚星："我不想听。"

她转身就走。

霍景深想拉住她，她甩开。

"别碰我！"

霍景深的手僵在半空。

江晚星头也不回地走了。
```

**第27章赵凯告知场景：**
```
赵凯："嫂子，霍哥受伤了。"

江晚星："什么？"

赵凯："他被家里打了。他爸因为他和白雨薇的事发火，动了手。"

江晚星："他……怎么样了？"

赵凯："伤得不重，但……他在医院。"

江晚星握紧手机，心口一紧。

赵凯："嫂子，你不去看看他吗？"

江晚星："不去。"

赵凯："可是……"

江晚星："我说了不去。"

可挂了电话，她坐在椅子上，眼泪掉下来。

她想去。

可她不能去。

```
