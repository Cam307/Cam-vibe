# 第31–33章详细细纲

> 承接本批剧情摘要（第28–30章）；第30章结尾江晚星和霍景深互相表白、两人和好、关系从假扮转为真在一起。
> 大纲对应：第31–33章为两人确立关系后的甜蜜日常、霍家内部矛盾的处理、霍父施压等。

---

## 第31章：官宣

### 本章基本信息
- 章节号：第31章
- 章节标题：官宣
- 在主线中的位置：两人确立关系后的甜蜜日常、公开关系
- 章节类型：甜蜜日常+小爽
- 承接上章：第30章两人互相表白、和好
- 开启下章：霍景深带江晚星见霍老、霍父施压
- 预计字数：不少于4000字（不含标点）

### 核心目标
- 情节目标：第二天霍景深在宿舍楼下等江晚星、送她上课、在食堂公开秀恩爱。论坛上炸了、都在传两人真的在一起了。
- 情感目标：江晚星从假扮女友到真恋爱的甜蜜；霍景深公开示爱、护短。
- 爽点：小爽——公开秀恩爱、论坛炸了。
- 钩子：霍景深接到霍父电话、被叫回家。

### 人物状态
- **江晚星**：和霍景深确立关系、甜蜜害羞；第一次真恋爱、不习惯公开秀恩爱但心里开心。
- **霍景深**：公开示爱、护短；论坛上官宣、让所有人知道江晚星是他的人。
- **赵凯、室友们**：助攻+吃瓜；论坛炸了、都在讨论。

### 场景要点
1. 宿舍楼下：霍景深等江晚星、送她上课、路上牵手。
2. 食堂：霍景深在食堂公开秀恩爱、给她剥虾、喂饭。
3. 论坛反应：赵凯拍照片发论坛、论坛炸了、都在讨论。
4. 电话：霍景深接到霍父电话、被叫回家。

### 写作重点
1. 甜蜜细节——霍景深牵她的手、给她剥虾、喂她吃饭。
2. 江晚星的反应——害羞、不习惯但心里开心。
3. 论坛反应——同学们的讨论、酸话、祝福。
4. 钩子——霍景深接到父亲电话、暗示霍家内部矛盾。

## 第32章：见爷爷

### 本章基本信息
- 章节号：第32章
- 章节标题：见爷爷
- 在主线中的位置：霍景深带江晚星见霍老、霍老认可
- 章节类型：情节推进+家族线
- 承接上章：第31章霍景深被父亲叫回家
- 开启下章：霍父施压、霍景深与父亲正面冲突
- 预计字数：不少于4000字（不含标点）

### 核心目标
- 情节目标：霍景深带江晚星去见霍老。霍老慈祥、询问江晚星情况、认可她。江晚星紧张但霍老很好相处。
- 情感目标：江晚星被霍老认可、心里踏实；霍景深高兴霍老喜欢江晚星。
- 爽点：中爽——霍老认可江晚星。
- 钩子：霍父出现、对江晚星不满意。

### 人物状态
- **江晚星**：紧张去见霍老、霍老慈祥让她放松、被认可很开心。
- **霍景深**：带江晚星见爷爷、很高兴爷爷喜欢她。
- **霍老**：慈祥、喜欢江晚星、认可她。
- **霍父**：对江晚星不满意、认为她配不上霍家。

### 场景要点
1. 霍家：江晚星紧张、霍景深安慰她。
2. 见霍老：霍老慈祥、询问江晚星情况、聊得很开心。
3. 霍老认可：霍老说"我很喜欢你"、江晚星很开心。
4. 霍父出现：霍父对江晚星冷淡、认为她配不上霍家。

### 写作重点
1. 霍老的慈祥——让江晚星放松、聊家常。
2. 江晚星的紧张→放松——从紧张到被认可很开心。
3. 霍父的反差——对江晚星冷淡、认为她配不上。
4. 霍景深的护短——维护江晚星。

## 第33章：我不娶她

### 本章基本信息
- 章节号：第33章
- 章节标题：我不娶她
- 在主线中的位置：霍景深与父亲正面冲突、拒绝娶白雨薇
- 章节类型：情节高潮+冲突
- 承接上章：第32章霍父对江晚星不满意
- 开启下章：霍景深被断经济来源、两人感情更坚定
- 预计字数：不少于4000字（不含标点）

### 核心目标
- 情节目标：霍父单独找霍景深谈话、逼他娶白雨薇。霍景深拒绝、说"我不娶她，我只娶江晚星"。霍父大怒、说要断他经济来源。
- 情感目标：霍景深为了江晚星与父亲对抗；江晚星知道后心疼但坚定。
- 爽点：中爽——霍景深为了江晚星拒绝联姻。
- 钩子：霍景深被断经济来源、江晚星知道后心疼。

### 人物状态
- **霍景深**：与父亲正面冲突、拒绝娶白雨薇、被断经济来源也不后悔。
- **霍父**：逼霍景深娶白雨薇、被拒绝后大怒、断他经济来源。
- **江晚星**：知道霍景深为她与父亲冲突、心疼但坚定、愿意和他一起面对。

### 场景要点
1. 霍家书房：霍父找霍景深谈话、逼他娶白雨薇。
2. 霍景深拒绝：说"我不娶她，我只娶江晚星"。
3. 霍父大怒：说要断他经济来源。
4. 江晚星知道：霍景深告诉她、她心疼但坚定、愿意和他一起面对。

### 写作重点
1. 霍父的施压——用家族、利益、前途逼霍景深。
2. 霍景深的拒绝——为了江晚星拒绝联姻。
3. 霍父大怒——断经济来源。
4. 江晚星的反应——心疼但坚定、愿意和他一起面对。

### 第33章对话示例（供写章时参考）

```
霍父："你必须娶白雨薇。"

霍景深："我不娶。"

霍父："你说什么？"

霍景深："我说，我不娶白雨薇。我只娶江晚星。"

霍父："你……你为了那个女生，要和白家作对？"

霍景深："不是和她作对，是我根本不喜欢白雨薇。"

霍父："喜欢能当饭吃？两家的利益呢？"

霍景深："我不在乎。"

霍父："好……好……那我就断你经济来源，我看你怎么养她！"

霍景深："断就断。我有手有脚，能养活她。"

霍父："你……"

霍景深："我说了，我只娶江晚星。"

```
