> 写作前填写 → 写作中对照 → 写作后勾选

---

```
章节：第___章

季节：___
林深年龄：___岁

相识时间：___年（当前年龄-18）
关系阶段：[ ]暧昧期 [ ]破裂期 [ ]重逢期 [ ]追妻期 [ ]修复期
```

---

## ✅ 写作前检查（30秒）

- [ ] 知道本章发生在什么时间
- [ ] 知道人物当前年龄
- [ ] 知道两人认识了几年
- [ ] 知道当前关系处于哪个阶段
- [ ] 知道本章要推动什么情节

---

## 🚨 敏感词警戒（写作时注意）

| 出现的词 | 立即核对 |
|----------|----------|
| "十年/二十年/一辈子" | 具体几年？算了没？ |
| "从小/童年" | 从几岁开始？ |
| "大学时期" | 大几？ |
| "工作了X年" | 毕业几年了？ |
| "认识X年了" | 当前年龄-18=？ |

---

## ✅ 写作后检查（1分钟）

- [ ] 时间线正确（已核对"X年"类词汇）
- [ ] 人物行为合理（已对照人物设定）
- [ ] 无明显矛盾（已检查与前文）
- [ ] 伏笔记录（如有新伏笔已记录）

---

## 🔍 逻辑自检（3问）

1. **时间对吗？**
   ```
   相识年数 = 当前年龄 - 18
   我的答案：___ = ___ - 18 ✓
   ```

2. **行为合理吗？**
   ```
   林深现在的性格：[ ]温和隐忍 [ ]清冷独立
   陆行舟现在的性格：[ ]傲慢逃避 [ ]偏执追妻
   本章行为符合：___ ✓
   ```

3. **前文呼应了吗？**
   ```
   前文相关情节：___________
   本章是否矛盾：[ ]是 [ ]否 ✓
   ```
