## 🎯 三种使用方式

### 方式一：章节模板（推荐）⭐

**最适合：** 每次创建新章节时使用

**步骤：**
1. 打开 `【章节模板-复制使用】.md`
2. 全选复制（Ctrl+A → Ctrl+C）
3. 创建新章节文件
4. 粘贴（Ctrl+V）
5. 填写顶部的"章节信息"
6. 开始写作
7. 写完后勾选"写作后检查"

**优点：**
- ✅ 自检信息整合在文件内
- ✅ 不用切换窗口
- ✅ 写完即可勾选

---

### 方式二：独立清单

**最适合：** 写作习惯分屏操作的用户

**步骤：**
1. 打开章节文件（主窗口）
2. 打开 `章节自检清单模板.md`（侧边窗口）
3. 按照清单逐项核对
4. 完成后关闭清单

**优点：**
- ✅ 章节文件干净简洁
- ✅ 清单可复用

---

### 方式三：参考手册查阅

**最适合：** 遇到具体问题时使用

**何时查阅 `写作质量控制指南.md`：**
- ❓ 不确定时间线 → 查"一、时间线管理"
- ❓ 不确定人物行为 → 查"二、人物设定一致性"
- ❓ 不确定关系阶段 → 查"2.3 人物关系检查"
- ❓ 发现了矛盾 → 查"三、情节逻辑检查"

## 💡 推荐工作流

### 创建新章节（2分钟）

```
1. 复制【章节模板-复制使用】.md
        ↓
2. 创建文件：第X章-xxx.md
        ↓
3. 填写顶部信息（30秒）
   - 章节号、标题
   - 人物年龄、相识年数
   - 关系阶段
        ↓
4. 开始写作
```

### 写作中（随时）

```
出现敏感词 → 停下来核对
"十年/二十年" → 算：当前年龄-18=？
"从小/童年" → 确认：从几岁开始？
```

### 写作完成后（1分钟）

```
1. 搜索敏感词（可选）
   Ctrl+F 搜索："十年"、"从小"

2. 勾选"写作后检查"

3. 从读者角度快速读一遍
```

## ⚡ Obsidian技巧

### 技巧1：固定参考手册

将 `写作质量控制指南.md` 固定在侧边栏：
1. 打开文件
2. 点击右上角"更多选项"
3. 选择"固定到侧边"

### 技巧2：快速打开模板

创建模板的快捷方式：
1. 设置 → 核心插件 → 模板
2. 指定模板文件夹：`小说作品/章节草稿/`
3. 设置快捷键（如：Ctrl+T）

### 技巧3：用Dataview自动检查

如果会用Dataview，可以添加元数据自动检查：

```yaml
---
章节: 1
标题: "最好的朋友"
林深年龄: 25
陆行舟年龄: 25
相识年数: 7
关系阶段: "暧昧期"
写作日期: 2026-01-27
---
```

### 技巧4：用Pin同时打开多个参考

Ctrl+点击文件名 → 在新窗口打开
推荐组合：
- 主窗口：当前章节
- 侧边1：人物档案.md
- 侧边2：写作质量控制指南.md

## 🔍 常见问题

### Q1: 每次都要填这么多信息吗？

**A:** 不是！
- **必须填：** 章节号、标题、年龄、相识年数（30秒）
- **可选填：** 写作后检查（1分钟勾选）
- **查手册：** 遇到问题时才查

### Q2: 会不会影响写作灵感？

**A:** 不会！
- 顶部注释不影响正文
- 填写只需30秒，比改逻辑错误省时间多了
- 习惯了会变成下意识行为

### Q3: 必须用这个模板吗？

**A:** 不是必须，但强烈推荐：
- 第1-3章：严格使用，养成习惯
- 第4-10章：可以简化，只填关键信息
- 第10章后：熟练了，可以不用模板

### Q4: 写作质量控制指南.md 什么时候看？

**A:** 按需查阅，不需要背诵：
- 写作前：快速看一眼"时间线速查表"
- 遇到问题：查对应章节
- 每10章：通读检查整体逻辑

## ✅ 最简版本（懒人方案）

如果你觉得太复杂，最少要记住这三点：

```
1. 写新章节前，花30秒确认：
   □ 现在是哪一年？人物多少岁？
   □ 相识几年？（当前年龄-18）
   □ 关系处于哪个阶段？

2. 出现"十年/二十年/从小"时，停下来算一算

3. 不确定的时候，查人物档案.md和大纲
```
