## 👥 主要人物

### 受：林深

#### 基本信息

| 项目 | 内容 |
|------|------|
| 姓名 | 林深 |
| 年龄 | 25岁（开始）→28岁（结局） |
| 生日 | 11月15日（天蝎座） |
| 相识时间 | 18岁大一认识，至今**七年** |
| 身高 | 178cm |
| 体重 | 62kg（三年后58kg，更瘦了） |
| 身份 | 室内设计师→知名设计师 |
| 公司 | 林氏设计工作室（自己创立） |
| 学历 | A大设计系本科 |
| 家庭 | 父母早逝，由奶奶抚养长大 |

#### 外貌描写

**整体气质：**
- 清冷美人，气质温润如玉
- 三年后更瘦更苍白，更有距离感
- 眼神清冷，但眼底深处藏着温柔

**五官：**
- 眉眼：眉形柔和，眼睛是琥珀色，睫毛很长
- 鼻子：鼻梁高挺，鼻尖微微上翘
- 嘴唇：唇形优美，不笑的时候很淡
- 脸型：瓜子脸，轮廓分明，皮肤白皙

**身材：**
- 身材修长，骨架小但不瘦弱
- 手指修长，适合画画和设计
- 三年后因为苦难变得更瘦，锁骨明显

**穿着风格：**
- 简约大方，偏好浅色系
- 喜欢穿白色、浅灰色、浅蓝色
- 三年后偏爱黑色，更有距离感

#### 性格特点

**前期（25岁）：**
- 温和隐忍，不善于表达自己的感情
- 对陆行舟温柔体贴，事事为他着想
- 内心敏感，善于察觉别人的情绪
- 有些自卑，觉得自己配不上陆行舟
- 对感情胆怯，不敢表白

**后期（28岁）：**
- 清冷独立，心墙高筑
- 不轻易相信他人
- 对感情更加谨慎
- 但内心深处依然温柔，只是被深深埋藏
- 对陆行舟有复杂的心结

**性格关键词：**
- 敏感、温柔、隐忍、清冷、独立
- 重感情、心软、固执（对感情）

#### 背景故事

**童年：**
- 出生在一个普通家庭
- 父母在他10岁时出车祸去世
- 由奶奶抚养长大
- 奶奶是他唯一的亲人

**大学时期（18-22岁）：**
- A大设计系学生
- 大一认识陆行舟（18岁）
- 对陆行舟一见钟情
- 但不敢表白，只是以朋友的身份陪在他身边
- **四年暗恋，认识七年**，却始终没有说出口

**工作时期（22-25岁）：**
- 毕业后成为陆行舟的私人设计师
- 进入陆氏集团工作
- 朝夕相处，感情更深
- 陆行舟开始意识到自己对林深的感情，但不敢面对
- 林深以为陆行舟只把他当朋友，不敢表白

**国外三年（25-28岁）：**
- 被骗走所有积蓄
- 生病没钱治疗，差点死掉
- 奶奶去世，未能见最后一面
- 被黑工打，伤痕累累
- 在宋远的帮助下慢慢恢复
- 创立自己的设计工作室

#### 成长弧线

```
隐忍暗恋 → 心碎离开 → 国外苦难 → 独立成长 → 心墙高筑 → 慢慢修复
```

#### 情感历程

**对陆行舟的感情：**
- 18岁大一认识，一见钟情，开始暗恋
- **认识七年，暗恋七年**
- 四年默默付出，不敢表白
- 工作后朝夕相处，感情更深
- 期待陆行舟开口，但陆行舟始终不说
- 订婚宴心碎，决定离开
- 国外三年，慢慢放下（但从未真正放下）
- 三年后重逢，心墙高筑
- 最终选择原谅，慢慢修复

#### 金手指

**设计才华：**
- 设计天赋极高
- 作品风格独特，备受业界认可
- 获得过多个国际设计大奖
- 三年后成为知名设计师

#### 弱点

1. **对陆行舟的感情**
   - 十年暗恋，无法真正放下
   - 即使被伤害，还是心软
   - 最终选择原谅

2. **奶奶**
   - 奶奶是他唯一的亲人
   - 奶奶去世是他最大的遗憾
   - 未能见最后一面

3. **心软**
   - 即使被伤害，还是心软
   - 看到陆行舟痛苦，还是会心疼

#### 习惯与爱好

**习惯：**
- 工作的时候喜欢听轻音乐
- 思考的时候喜欢咬嘴唇
- 紧张的时候喜欢转笔
- 睡不着的时候会画画

**爱好：**
- 画画、设计
- 听音乐
- 看书
- 喝茶

**不喜欢：**
- 喧闹的地方
- 被人欺骗
- 背叛
- 失去

#### 语言风格

- 说话温和，声音好听
- 不善言辞，但说到做到
- 前期：对陆行舟温柔体贴
- 后期：冷淡疏离，但对陆行舟还是会心软

#### 经典台词

1. "陆总，请你自重。"
2. "我们已经不可能了。"
3. "有些伤害，是无法原谅的。"
4. "我不爱你了。"（假的）
