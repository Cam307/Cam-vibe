# 30万字长篇小说创作流程指南

> 本文档提供完整的30万字长篇小说创作流程，从选题到完稿的全流程指导。

---

## 📊 30万字小说概览

### 字数分配参考

| 项目 | 字数 | 章节数 | 占比 |
|------|------|--------|------|
| **开篇钩子** | 3-5万字 | 6-10章 | 10-15% |
| **前期发展** | 5-8万字 | 10-16章 | 15-25% |
| **中期推进** | 10-12万字 | 20-24章 | 35-40% |
| **高潮爆发** | 5-8万字 | 10-16章 | 15-25% |
| **结局收尾** | 3-4万字 | 6-8章 | 10-15% |
| **总计** | **30万字** | **60-80章** | **100%** |

### 章节标准

- **每章字数**：3000-5000字（建议4000字）
- **章节总数**：60-80章
- **更新频率**：建议每天2章（8000字），约40天完成初稿
- **完稿周期**：2-3个月（含修改）

## 🎯 第一阶段：前期准备（1-3天）

### 1.1 选题确定

#### 步骤1：选择题材组合

从《小说选题灵感库.md》中选择或组合：

**示例选择：**
```
题材：古代言情
核心梗：追妻火葬场 + 替身 + 宫廷权谋
风格：虐文（前期虐妻，后期虐夫）
参考：别枝头、冰山之心、别枝头
目标字数：30万字
```

**填写选题表：**

| 项目 | 内容 |
|------|------|
| 小说名 | （暂定）金蝉脱壳 |
| 题材类型 | 古言 + 宫廷 + 追妻火葬场 |
| 核心梗 | 替身 + 白月光 + 权谋 |
| 风格定位 | 虐文（中虐偏重） |
| 目标字数 | 30万字 |
| 目标读者 | 18-35岁女性 |
| 参考作品 | 别枝头、冰山之心 |

#### 步骤2：确定核心卖点

列出3-5个核心卖点：

```
【金蝉脱壳】核心卖点：
1. 替身女主觉醒，华丽离开
2. 追妻火葬场，男主痛不欲生
3. 宫廷权谋，各方势力博弈
4. 女主独立成长，不再依附
5. 反套路设计，出人意料
```

#### 步骤3：一句话简介

用一句话概括小说：

```
替身贵妃江离，在白月光归来后被弃，和离去后入朝堂，
三年后再遇，悔恨交加的皇帝能否追回前妻？
```

### 1.3 世界观设定

**时代背景：**
- 架空古代王朝（大周）
- 皇帝集权，丞相掌权
- 宫廷政治斗争激烈
- 女子地位较低，但可入朝为官

**地理设定：**
- 京城（皇宫）
- 江府（相府）
- 边境（战场）
- 江南（女主和离后生活）

**势力分布：**
- 皇权（皇帝）
- 相权（丞相府）
- 后宫（皇后/嫔妃）
- 边军（萧云）

### 1.4 大纲搭建

#### 三幕式结构

**第一幕：替身生涯（0-25%）**
```
开篇：江离入宫为妃（0-5%）
发展：三年替身生活，看似受宠（5-15%）
转折：白月光萧雪归来，江离被弃（15-20%）
激励事件：江离求和离（20-25%）
```

**第二幕：独立成长（25-75%）**
```
前期：和离去江南，开始新生活（25-35%）
中期：入朝堂为女官，展现才华（35-55%）
后期：各方势力斗争，女主成长（55-70%）
高潮：边境危机，女主出征/智退敌军（70-75%）
```

**第三幕：追妻与结局（75-100%）**
```
追妻：皇帝后悔，开始追妻（75-85%）
抉择：女主是否回头（85-90%）
结局：女主选择自己，成为独立女性（90-95%）
尾声：三年后，各自安好（95-100%）
```

## ✍️ 第三阶段：分章创作（30-40天）

### 3.1 创作准备

#### 每日创作计划

**标准日程：**
```
早上（2小时）：写第1章
下午（2小时）：写第2章
晚上（1小时）：检查修改，规划明天
```

**每日进度：** 2章 × 4000字 = 8000字

**完稿周期：** 80章 ÷ 2章/天 = 40天

#### 每章创作步骤

```
步骤1：回顾大纲（5分钟）
步骤2：列本章要点（10分钟）
步骤3：撰写正文（90分钟）
步骤4：检查修改（15分钟）
步骤5：设置悬念（5分钟）
```

### 3.2 章节创作模板

#### 章节结构标准

**开头（300-500字）：**
- 承上启下
- 设置本章场景
- 引出本章事件

**发展（2500-3000字）：**
- 核心事件展开
- 人物对话
- 情感推进

**高潮（800-1000字）：**
- 本章最大冲突/转折
- 情绪顶点

**结尾（200-400字）：**
- 收束本章事件
- 设置悬念钩子

#### 悬念钩子类型

```
类型1：危机钩子
"就在这时，门外传来了急促的脚步声..."

类型2：对话钩子
"江离，我们，和离吧。"

类型3：转折钩子
"江离万万没有想到，来人竟是..."

类型4：悬念钩子
"那一刻，江离做出了改变她一生的决定。"

类型5：情感钩子
"萧景终于明白，他失去了什么。"
```

### 3.3 质量控制

#### 每日检查清单

```
□ 字数达标（4000字左右）
□ 情节推进
□ 人物性格一致
□ 有悬念钩子
□ 语言流畅
□ 错别字检查
```

#### 每周复盘

```
□ 本周章节完成情况
□ 情节是否有遗漏
□ 人物发展是否合理
□ 节奏是否合适
□ 下周调整计划
```

## ⚡ 实战示例

### 示例：第1章创作

#### 步骤1：回顾大纲

```
第1章：入宫
核心事件：大婚之夜，皇帝提起白月光
字数目标：4000字
虐点设置：大婚之夜，皇帝提起另一个女人
```

#### 步骤2：列要点

```
1. 大婚典礼场景
2. 江离入洞房
3. 皇帝到来，表现冷淡
4. 皇帝提起白月光
5. 江离心碎，但表面平静
6. 结尾钩子：江离独自流泪
```

#### 步骤3：撰写（示例开头）

```
大周宣和三年，春。

桃花开得正艳，丞相府的三小姐江离，嫁入皇宫为妃。

十里红妆，满城皆知。

然而，当夜幕降临，红烛摇曳的寝殿内，等待江离的，并非新婚的温存……

……

萧景坐在床边，目光却越过她，望向虚空。

"江离，"他忽然开口，声音低沉，"你知道雪儿吗？"

江离的心，猛地一沉。

"她就像这桃花一样，美好却易碎……"

大婚之夜，他在她的面前，思念着另一个女人。

那一刻，江离终于明白，她只是一个替身。

……

夜深了。

萧景已经离开，只留下江离一人。

她坐在床边，看着摇曳的红烛，一滴泪，终于落下。

这就是她的新婚之夜。

```

#### 步骤4：检查

```
✓ 字数：约4000字
✓ 情节：大婚之夜，提白月光
✓ 虐点：新婚之夜提另一个女人
```
