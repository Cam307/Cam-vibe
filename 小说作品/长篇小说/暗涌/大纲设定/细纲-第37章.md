# 细纲-第37章：奶奶墓地

## 【基本信息】

- **章节标题**：奶奶墓地
- **目标字数**：3,750字
- **时间**：第36章结束后第二天
- **地点**：公墓
- **天气**：阴沉、微风
- **人物状态**：
  - 林深：去祭拜奶奶
  - 陆行舟：偷偷跟随，忏悔

---

## 【上下文约束】

**前情提要**：
第36章结尾，陆行舟向林深道歉，告诉他"我相信你"。林深冷淡回应："现在说这些，已经没有意义了。"陆行舟痛苦离开，但没有放弃。第二天，他得知林深要去公墓祭拜奶奶，决定偷偷跟随。

**核心冲突**：陆行舟的忏悔 vs 无法弥补的遗憾

---

## 【核心情节】

林深去公墓祭拜奶奶，在墓前说出了自己的心里话。陆行舟躲在远处，听到了林深的话——"奶奶，我想你了......奶奶，我后悔来巴黎......奶奶，我后悔没有见他最后一面......"陆行舟听到这些，心碎。等林深离开后，陆行舟走到墓前，跪下忏悔。

## 【对话示例】

### 示例一：林深祭拜
```
林深拿着一束白菊花，慢慢走到奶奶的墓前。

他放下花，蹲下，抚摸墓碑上的照片。

照片里的奶奶笑得很慈祥。

"奶奶，我来看你了。"林深轻声说。

风吹过，树叶沙沙作响。

"你还好吗？"林深说，"在那边......一个人......孤单吗？"

林深的眼泪掉下来。

"我很好......真的很好......"

"我现在有自己的工作室了......"

"我也认识了新的朋友......"

"但奶奶......我想你......"

"真的很想你......"

林深哭出声来。

"奶奶，对不起......"

"对不起我没有见你最后一面......"

"对不起我让你一个人......"

"奶奶，我真的......对不起......"
```

### 示例二：林深的诉说
```
林深坐在墓前，继续说。

"奶奶，我后悔来巴黎......"林深说。

"如果我没有来......如果我没有走......"

"我就能见你最后一面了......"

"奶奶，你知道吗？我接到你去世的消息的时候......我在昏迷......"

"我生病了，没钱治疗，在出租屋昏迷了三天......"

"等我醒来的时候......你已经......"

林深哭到失声。

"奶奶，我后悔 deleting 他的联系方式......"

"如果我没有删除......如果我能联系到他......"

"或许我能借到钱......我就能回来了......"

"我就能见你最后一面了......"

"奶奶，我真的......后悔了......"

风吹过，带着林深的哭声。

远处，陆行舟躲在树后，听着林深的话。

他的心，狠狠地抽痛。

"我后悔 deleting 他的联系方式......"

"他"......是谁？

陆行舟的心狠狠一抽。

那个"他"......是自己吗？

林深后悔......删除了自己的联系方式？

陆行舟闭上眼睛，泪流满面。

原来......林深后悔了。

原来......林深在奶奶去世的时候，想联系自己......

但找不到自己了。

因为林深删除了所有的联系方式。

而自己......一无所知。

陆行舟跪在地上，崩溃大哭。
```

### 示例三：陆行舟忏悔
```
林深离开后，陆行舟走到墓前。

他跪下，抚摸墓碑上的照片。

"奶奶，对不起......"陆行舟说。

"我没有保护好他......"

"他那么痛苦......而我......我什么都不知道......"

"奶奶，你知道吗？他在巴黎......"

"他被骗光了所有积蓄......"

"他生病差点死掉......"

"他一个人......一个人承受了所有的痛苦......"

"而我......我在哪里......"

"我在和苏婉订婚......"

"我在......我在过我的生活......"

"奶奶，我真的......对不起......"

陆行舟跪在地上，泪流满面。

"奶奶，请保佑他......"

"保佑他不再痛苦......"

"保佑他......能够原谅我......"

"哪怕......哪怕只有一点点......"

陆行舟闭上眼睛，心中涌起无法言说的痛。

他欠林深的，太多了。

欠得......他这辈子都还不清。

风吹过，树叶沙沙作响。

仿佛奶奶在回应他。

"傻孩子。"风中仿佛传来奶奶的声音，"如果你真的想弥补......"

"那就去弥补。"

"不要放弃。"

"因为他......他还爱你。"

陆行舟睁开眼睛。

风吹过，带着温暖。

陆行舟深吸一口气，站起来。

奶奶说得对。

不要放弃。

林深还爱自己。

从林深刚才的话就能听出来——

"我后悔 deleting 他的联系方式......"

"如果我还能联系到他......"

这些话，说明林深还在乎。

林深还没有完全放下。

那他......就不能放弃。

陆行舟擦掉眼泪，转身离开。

他还有机会。

他一定要抓住。
```

## 【情感基调】

- **整体基调**：悲伤、忏悔、希望
- **情感层次**：
  - 表层：林深对奶奶的思念
  - 深层：陆行舟的忏悔
  - 隐层：一丝希望

- **情感节奏**：
  - 开场：林深祭拜奶奶
  - 中段：林深的诉说
  - 高潮：陆行舟听到后的崩溃
  - 收尾：陆行舟的忏悔和决心

---

## 【写作注意事项】

1. **《针锋对决》风格要点**：
   - 林深的诉说要真实、隐忍
   - 陆行舟的忏悔要真诚、有力
   - 情感层次要丰富

2. **细节把控**：
   - 林深抚摸墓碑照片的动作
   - 风吹过树叶的声音
   - 陆行舟跪在墓前的画面

3. **节奏控制**：
   - 本章是情感重点章节，节奏要平稳
   - 情感层次要丰富

4. **伏笔设置**：
   - 林深"后悔删除"的伏笔
   - 为两人关系的修复做铺垫

5. **避免事项**：
   - 不要让林深显得过于歇斯底里
   - 不要让陆行舟显得过于煽情
   - 保持克制，情感要真实
