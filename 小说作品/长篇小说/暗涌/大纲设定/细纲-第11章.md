# 细纲-第11章：决定

## 【基本信息】

- **章节标题**：决定
- **目标字数**：3,750字
- **时间**：奶奶葬礼后第二天，周日下午至晚上
- **地点**：老家 → 回程飞机上 → 林深公寓
- **天气**：晴朗，阳光刺眼
- **人物状态**：
  - 陆行舟：刚与苏家谈完合作，得知林深奶奶去世的消息后愧疚万分
  - 林深：心力交瘁，对陆行舟已经不抱期待
  - 苏婉音：与陆行舟通电话，商议订婚事宜

---

## 【上下文约束】

**前情提要**：
第10章结尾，奶奶去世后，陆行舟姗姗来迟。林深说了一句"反正……你也来不了"，决绝地转身离开。陆行舟站在病房里，面对着已经去世的奶奶，心中充满了愧疚。第二天，奶奶的葬礼举行，陆行舟全程陪同，但林深始终没有和他说过一句话。

**核心冲突**：陆行舟的愧疚与无法挽回 vs 林深的彻底失望

---

## 【核心情节】

奶奶葬礼后，陆行舟回到公司，发现与苏家的合作已经基本谈妥，苏家提出的条件是——订婚。陆行舟陷入两难，但最终，为了公司的利益，他做出了决定。晚上，他给林深打电话，告诉他自己要订婚的消息。

---

## 【详细场景分解】

### 场景一：葬礼后的沉默（约900字）
- **时间**：周日下午
- **地点**：老家墓地 → 回程车上
- **人物**：陆行舟、林深、司机
- **情节描述**：

  - 陆行舟的心狠狠一抽——"陆总"

## 【对话示例】

### 示例一：葬礼后的沉默
```
葬礼结束，林深站在墓碑前，久久不肯离开。

陆行舟站在远处，看着林深的背影。

风吹过，林深的外套被吹起，露出单薄的身形。

陆行舟想上前，想把手放在林深的肩膀上，想说点什么。

但他最终没有动。

因为他知道，这个时候，任何安慰都是多余的。

林深转身，径直走向车边，经过陆行舟身边时，没有停留。

陆行舟叫住了他："林深。"

林深停下脚步，但没有回头。

"怎么了？"

声音很平静，平静得让陆行舟心慌。

"我……"陆行舟张了张嘴，却不知道该说什么，"你……还好吗？"

林深沉默了很久。

"我很好。"林深说，"谢谢陆总关心。"

陆行舟的心狠狠一抽——"陆总"

"林深，我……"

"陆总。"林深打断了他，"如果您要说对不起，就不用了。反正……您也不欠我什么。"

说完，林深继续往前走，留下陆行舟站在原地，心头满是苦涩。
```

### 示例二：苏家的条件
```
手机在桌上震动，陆行舟看了眼来电显示，接起。

"喂，婉音。"

"行舟，你那边忙完了吗？"苏婉音的声音很轻快。

"嗯，刚处理完一些事情。"陆行舟说，"合作的事，你那边有进展吗？"

"有呢。"苏婉音说，"爸爸说，基本可以定下来了，但……"

"但什么？"

"爸爸说，希望我们能先订婚，这样他才能放心把项目交给陆氏。"

陆行舟沉默了。

"婉音，我……"

"我知道这很突然。"苏婉音说，"但我们本来就是打算订婚的，只是时间早晚的问题，不是吗？"

陆行舟无法反驳。

他和苏婉音的婚约，是两家早就商定好的。只是因为种种原因，一直拖延到现在。

现在，苏家提出这个条件，并不算过分。

"给我一点时间考虑。"陆行舟说。

"好。"苏婉音顿了顿，"但爸爸说，希望这周内能有答复。毕竟项目拖得越久，变数越多。"

"我知道了。"

"行舟……"苏婉音的声音柔和下来，"我知道你可能有顾虑，但……我们结婚，不是迟早的事吗？"

陆行舟闭上眼睛。

"我知道。"

"那就好。"苏婉音说，"我等你消息。"

挂断电话，陆行舟靠在椅背上，长长地叹了口气。
```

### 示例三：决定后的电话
```
陆行舟拿起手机，找到林深的号码。

这个时候打给林深，说什么呢？

告诉他自己要订婚了？告诉他自己……对不起？

陆行舟的手指悬在拨号键上方，久久没有落下。

但他知道，必须打。

林深会从别人那里听到这个消息。与其让他从别人那里知道，不如自己告诉他。

陆行舟最终按下了拨号键。

电话响了很久，才被接通。

"喂？"

林深的声音很哑，带着哭过后的沙哑。

陆行舟的心狠狠一抽。

"林深……是我。"

电话那头沉默了几秒。

"陆总，有事吗？"

又是"陆总"。

陆行舟的心像被人狠狠掐了一下。

"林深，我……"陆行舟深吸一口气，"我有件事要告诉你。"

"什么？"

"我和婉音……"陆行舟顿了顿，"我们要订婚了。"

电话那头陷入了沉默。

长久的沉默。

沉默到陆行舟以为电话已经断了。

"林深？"

"恭喜。"

林深的声音很平静，平静得让人害怕。

"林深，我……"

"陆总，我这边有点事，先挂了。"林深说，"恭喜你和苏小姐。"

"林深，等等……"

"嘟——嘟——"

电话被挂断了。

陆行舟握着手机，久久没有动。

他不知道的是，电话那头的林深，靠在墙上，手机从手中滑落，眼泪无声地流下来。
```

## 【情感基调】

- **整体基调**：压抑、决绝、无可挽回
- **情感层次**：
  - 表层：陆行舟的纠结与决定
  - 深层：两人关系的不可逆转
  - 隐层：陆行舟对林深的感情，但他自己可能还没有意识到

- **情感节奏**：
  - 开场：葬礼后的冷漠
  - 中段：陆行舟的纠结
  - 高潮：决定后的电话
  - 收尾：电话挂断后的空虚

---

## 【写作注意事项】

1. **《针锋对决》风格要点**：
   - 情感要克制，不要过度渲染
   - 陆行舟的决定要有合理性，不是因为他不爱林深，而是因为现实的考量
   - 林深的反应要符合他此时的状态——已经失望透顶

2. **细节把控**：
   - 葬礼场景要庄重，不要过度煽情
   - 陆行舟的纠结要通过细节体现：握紧的手、紧锁的眉头、反复查看手机
   - 林深的冷漠要通过细节体现：平静的声音、回避的眼神、礼貌而疏离的态度

3. **节奏控制**：
   - 本章是转折性章节，节奏要从缓慢到压抑再到爆发
   - 电话场景是高潮，要写得足够虐心

4. **伏笔设置**：
   - 为后续林深的离开做铺垫
   - 为陆行舟的后悔埋下伏笔

5. **避免事项**：
   - 不要让陆行舟显得冷血，他有合理的理由
   - 不要让林深显得过于歇斯底里，他的悲伤是内敛的
   - 不要过度渲染悲剧感，保持克制
