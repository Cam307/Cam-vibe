## 【上下文约束】

**前情提要**：
第34章结尾，陆行舟醉酒崩溃，承认自己爱林深。林深那边，赵泽透露自己是"假未婚夫"，但两人没有进一步讨论。第二天，苏婉找到陆行舟，决定告诉他订婚宴的真相。

**核心冲突**：真相的揭露 vs 陆行舟的承受

---

## 【核心情节】

苏婉找到陆行舟，告诉他订婚宴的真相——那天她指控林深抄袭，是故意的。因为她嫉妒林深，嫉妒陆行舟对林深的特殊感情。她想让林深离开，想让陆行舟看清自己的心。但她没想到，她的做法伤害了林深那么深。陆行舟听后崩溃，质问苏婉为什么不早说。苏婉说："因为我也后悔了。"

## 【对话示例】

### 示例一：苏婉出现
```
陆行舟坐在办公室里，头痛欲裂。

昨晚喝太多了。

助理敲门进来："陆总，苏家小姐来了。"

"不见。"陆行舟头也不抬。

"她说......"助理顿了顿，"她说有关于林深的事情要告诉您。"

陆行舟的手顿住了。

"让她进来。"

助理点头，退出去。

几分钟后，苏婉推门而入。

她穿着一件简单的白色连衣裙，脸色有些苍白。

"行舟。"苏婉轻声叫道。

陆行舟抬起头，看着苏婉。

"你来做什么？"陆行舟问，语气冷淡。

"我......"苏婉顿了顿，"我是来告诉你的......关于订婚宴的事。"

陆行舟的心狠狠一跳。

"订婚宴......怎么了？"
```

### 示例二：订婚宴真相
```
苏婉坐在陆行舟对面，深吸一口气。

"订婚宴那天......"苏婉说，"是我故意指控林深抄袭的。"

陆行舟愣住了。

"什么？"

"那个设计，确实是林深的原创。"苏婉说，"没有抄袭。"

"那你为什么要......"

"因为我嫉妒。"苏婉说，声音颤抖，"我嫉妒你对林深的特殊感情。"

陆行舟坐在那里，说不出话。

"我知道，你对他......不一般。"苏婉说，"但我以为......我以为如果我和你订婚了，你会忘记他。"

"但订婚宴那天......我看到你看着林深的眼神......"

"我就知道，我错了。"

"所以我想......如果林深离开了，你会怎么做？"

"于是我指控他抄袭。"

"我想让他当众出丑，想让他离开......"

"但我没想到......"苏婉的眼泪掉下来，"我没想到伤害他那么深。"

陆行舟坐在那里，脸色越来越苍白。

他终于明白了。

订婚宴那天，林深被当众指控抄袭。

他看向自己，期待自己为他说话。

但他没有。

他没有说话，任由林深被羞辱。

然后林深离开了。

去了巴黎，受尽了苦难。

而这一切......

都是因为苏婉的嫉妒。

因为自己没有保护好林深。
```

### 示例三：陆行舟的愤怒
```
陆行舟站起身，双手撑在桌上。

"你知不知道你做了什么？"陆行舟吼道。

"我......"

"你当众羞辱他......"陆行舟的声音在颤抖，"你让他失去了所有的尊严......"

"而我......我没有为他说话......"

"我就站在那里，看着他被羞辱......"

"我什么都没有做......"

"我知道。"苏婉的眼泪掉下来，"所以我后悔了。"

"后悔有什么用？"陆行舟冷笑，"伤害已经造成了！"

"行舟，我真的不知道会这样......"

"你不知道？"陆行舟看着苏婉，"苏婉，你是成年人，你应该知道你的行为会有什么后果！"

"我......"

"你毁了他。"陆行舟说，"你也毁了我。"

"行舟......"

"你毁了我唯一......唯一爱过的人。"

苏婉愣住了。

"爱？"苏婉喃喃道，"你......你爱他？"

陆行舟没有回答。

他只是闭上眼睛，心中涌起剧烈的痛。

是的，他爱林深。

他爱了七年。

但他从来没有承认过。

直到现在。

直到一切都无法挽回的时候。
```

### 示例四：苏婉的解释
```
苏婉擦掉眼泪，深吸一口气。

"我当时以为......"苏婉说。

"你以为什么？"陆行舟冷冷地问。

"我以为如果林深离开了，你会看清自己的心。"苏婉说。

"看清我的心？"陆行舟笑了，笑声讽刺，"苏婉，你毁了他，也毁了我。"

"我知道。"苏婉说，"所以我来告诉你真相。"

"这样，你至少知道......他真的是被冤枉的。"

"冤枉又如何？"陆行舟说，"伤害已经造成了。"

"至少......至少你可以告诉他......"苏婉说，"告诉他，你是相信他的。"

"告诉他......"陆行舟顿住了。

是的。

他可以告诉林深。

告诉他，"我相信你。"

告诉他，"你是被冤枉的。"

但这有什么用呢？

伤害已经造成了。

林深的痛苦，已经无法挽回了。

"谢谢你告诉我。"陆行舟最终说。

"行舟......"

"你走吧。"陆行舟说，"我想一个人待一会儿。"

苏婉看着陆行舟，最终点了点头。

"对不起。"苏婉轻声说。

然后她转身离开。

办公室里只剩下陆行舟一个人。

他闭上眼睛，心中涌起复杂的情绪。

真相......

终于真相大白了。

但这真相，只会让他更痛。
```

## 【情感基调】

- **整体基调**：真相、愤怒、痛苦
- **情感层次**：
  - 表层：苏婉的坦白
  - 深层：陆行舟的愤怒和痛苦
  - 隐层：无法挽回的遗憾

- **情感节奏**：
  - 开场：苏婉的到来
  - 中段：订婚宴真相的揭露
  - 高潮：陆行舟的愤怒爆发
  - 收尾：真相的无力

---

## 【写作注意事项】

1. **《针锋对决》风格要点**：
   - 苏婉的坦白要有层次，不是单纯的认错
   - 陆行舟的愤怒要有力，但不要过于歇斯底里
   - 对话要有目的

2. **细节把控**：
   - 苏婉的愧疚和后悔
   - 陆行舟听到真相时的反应
   - 两人对话时的气氛

3. **节奏控制**：
   - 本章是真相揭露章节，节奏要紧凑
   - 情感层次要丰富

4. **伏笔设置**：
   - 为陆行舟向林深道歉做铺垫
   - 为两人关系的转折做铺垫

5. **避免事项**：
   - 不要让苏婉显得过于可恶
   - 不要让陆行舟显得过于完美
   - 保持平衡，每个人都有自己的立场
