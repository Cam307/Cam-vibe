## 【上下文约束】

**前情提要**：

## 【对话示例】

### 示例一：看到伤疤
```
林深洗完澡，穿着浴袍出来。

陆行舟在床上看书，抬头看到林深。

"洗好了？"陆行舟问。

"嗯。"林深点头，走到衣柜前，准备拿睡衣。

浴袍滑落，露出肩膀和背上的伤疤。

陆行舟的心狠狠一痛。

那些伤疤......是林深在巴黎受的苦。

是那些人打他时留下的。

陆行舟记得，陈默说过，林深被几个人打，差点死掉。

从那以后，林深的身上就留下了这些伤疤。

永远地留下了。

"你......在看什么？"林深转身，发现陆行舟在看他。

"没......没什么。"陆行舟说。

但他的眼中，满是心疼。

林深顺着陆行舟的目光，低头看了看自己的肩膀。

他明白了。

陆行舟在看他身上的伤疤。

"很丑，对吧？"林深自嘲地笑了。

"不。"陆行舟摇头，"不丑。"

"怎么会不丑？"林深说，"这些伤疤......"

"这些伤疤，是你坚强的证明。"陆行舟说。

林深愣住了。

"你......你是认真的？"

"我是认真的。"陆行舟说，"这些伤疤，证明了你熬过来了。"

"证明了......你很坚强。"

林深的心狠狠一跳。

陆行舟......不觉得他丑？

陆行舟......在心疼他？
```

### 示例二：询问伤疤
```
陆行舟走到林深身后。

"这些伤疤......还疼吗？"陆行舟问。

"不疼了。"林深说，"早就好了。"

"但它们......会一直提醒我那段日子。"林深说。

陆行舟的心中一痛。

"对不起。"陆行舟说。

"不怪你。"林深转身，看着陆行舟。

"是我没有保护好你。"陆行舟说。

"那时候，你应该在A市，过着安稳的生活。"

"而不是在巴黎，被人打，差点死掉。"

"这都是我的错。"

"不怪你。"林深摇头，"是我自己选择的。"

"我选择离开，选择去巴黎。"

"所有的后果，我都承担。"

"但如果你不是为了躲我......"陆行舟说。

"不是为了躲你。"林深打断陆行舟，"是为了......放过我自己。"

"我需要离开，需要重新开始。"

"所以，我去了巴黎。"

"那些苦难，是我自己选择的。"

"不怪你。"

陆行舟看着林深，眼中满是心疼。

林深......还在为他开脱。

还在......不怪他。

但陆行舟知道，那些伤疤，是他造成的。

如果不是他......林深不会受那么多苦。
```

### 示例三：亲吻伤疤
```
"我能......吻它们吗？"陆行舟问。

林深愣住了。

"什么？"

"我说，我能吻它们吗？"陆行舟重复道。

林深沉默了一会儿，然后点头。

"好。"

陆行舟轻轻拉下林深的浴袍。

露出肩膀上的伤疤。

陆行舟低下头，轻轻吻了上去。

林深浑身一颤。

"疼吗？"陆行舟抬起头。

"不疼。"林深说。

陆行舟继续吻，一个一个伤疤，温柔地吻。

肩膀上的，背上的，手臂上的......

每一个伤疤，陆行舟都温柔地吻。

林深的眼泪掉了下来。

"别哭。"陆行舟说。

"我......我没哭。"林深说。

但眼泪止不住地流。

陆行舟抬起头，擦去林深的眼泪。

"为什么哭？"陆行舟问。

"因为......"林深哽咽道，"因为从来没有人......这样对我。"

"从来没有。""

陆行舟的心狠狠一痛。

"以后会有。"陆行舟说，"以后，我会一直这样对你。"

"每一个伤疤，都是我欠你的。"陆行舟说。

"我会用余生来弥补。"

"不需要弥补。"林深摇头。

"需要。"陆行舟说，"这是我心甘情愿的。"

"因为我爱你。"

"所以，我想弥补。"

"想让你......不再受伤。"

林深看着陆行舟，眼泪又掉了下来。

"我爱你。"林深说。

"我也爱你。"陆行舟说。

两人紧紧相拥，在月光下。

第66章，完。
```
