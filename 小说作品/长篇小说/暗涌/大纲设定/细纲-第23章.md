# 细纲-第23章：奶奶去世

## 【基本信息】

- **章节标题**：奶奶去世（回忆篇，超虐）
- **目标字数**：4,500字（超虐）
- **时间**：三年前，林深生病康复后一个月
- **地点**：巴黎出租屋、医院（回忆中）
- **天气**：阴雨连绵
- **人物状态**：
  - 林深：刚康复，准备给奶奶打电话
  - 奶奶：在老家病重，想见孙子最后一面

---

## 【上下文约束】

**前情提要**：
第22章回忆林深生病后，邻居宋远救了他。第23章继续回忆，一个月后，林深刚康复，准备给奶奶打电话报平安，却接到国内电话——奶奶病危，想见他最后一面。但林深没钱买机票，只能隔着电话，听着奶奶去世的消息。

**核心冲突**：林深的无力 vs 奶奶的思念

---

## 【核心情节】

林深刚康复，准备给奶奶打电话报平安，却接到国内电话——奶奶病危。奶奶想见他最后一面，但林深没钱买机票。他疯狂地借钱、打工攒钱，但还是晚了一步。奶奶去世了，最后一面也没见到。林深在巴黎的出租屋里，对着电话，痛哭失声。

## 【对话示例】

### 示例一：准备给奶奶打电话
```
林深坐在出租屋里，手里拿着手机。

"宋远，谢谢你的帮助。"林深说，"医药费我一定会还你的。"

"不急，你先养好身体。"宋远说，"对了，你家里……有亲人吗？"

"有，奶奶。"林深说，"我还没告诉她我生病的事，怕她担心。"

"那你要不要给她打个电话？"

"好，我今晚就打。"林深说。

晚上，林深拿着手机，犹豫着要不要给奶奶打电话。

他怕奶奶听出他的声音不对，怕奶奶担心。

但最终还是拨通了号码。

"喂？"

奶奶虚弱的声音从电话那头传来。

林深的心狠狠一抽。

"奶奶，是我，深儿。"

"深儿啊……"奶奶的声音有些颤抖，"你……你在那边还好吗？"

"我很好，奶奶。"林深强忍着眼泪，"我找了工作，认识了新朋友，一切都好。"

"那就好……那就好……"奶奶说，"奶奶担心你……"

"奶奶，您别担心，我很好的。"林深说，"您要注意身体，多休息……"

"奶奶知道……"奶奶说，"深儿，奶奶……想你了……"

林深的眼泪差点掉下来。

"我也想您，奶奶。"林深说，"等我有时间了，就回去看您。"

"好……奶奶等你……"奶奶说。

林深不知道的是，这是他和奶奶最后一次通话。
```

### 示例二：接到噩耗
```
就在林深准备再次给奶奶打电话的时候，手机响了。

是国内的老家号码。

林深的心猛地一紧，不好的预感涌上心头。

他接起电话。

"喂？"

"是林深吗？"电话那头是二叔的声音，"你奶奶……病危了。"

林深的手一抖，手机差点掉在地上。

"什么？"

"你奶奶……一直昏迷，医生说……可能撑不过今晚了。"二叔说，"她……她一直在念叨你，想见你最后一面。"

林深的心狠狠一抽。

能回来吗？

他现在身无分文，连买机票的钱都没有。

"我……我尽量。"林深说，声音在颤抖。

"尽快吧……"二叔叹了口气，"不然……可能来不及了。"

挂断电话，林深坐在那里，久久没有动。

奶奶病危。

想见他最后一面。

但他……回不去。

林深握紧拳头，眼泪流了下来。

"一定要回去……一定要……"
```

### 示例三：疯狂的尝试
```
林深疯狂地想办法回国。

他查了机票价格——最便宜的也要七八千人民币。

他现在连七百都没有。

林深给所有认识的人打电话借钱。

"喂，老王，能不能借我点钱？我奶奶病危，我要回国……"

"抱歉，我也没钱……"

"喂，李姐，能不能……"

"你被骗了？那更不能借你了，万一你还不上了怎么办？"

一个又一个拒绝。

林深绝望地坐在出租屋里。

他又去打黑工，试图在短时间内赚到机票钱。

但三天能赚到的钱，远远不够。

他去航空公司问能不能赊账，被拒绝了。

"抱歉，先生，我们这里不提供赊账服务。"

他去中介问有没有更便宜的票，还是被拒绝了。

"先生，这已经是最便宜的了。"

林深绝望了。

他坐在出租屋里，看着窗外的巴黎夜景。

埃菲尔铁塔在远处闪烁，美丽而浪漫。

但这一切，都不属于他。

他只是这里的囚徒，一个被贫穷困住的囚徒。

林深闭上眼睛，眼泪滑落。

"奶奶……等着我……我一定会回去的……"
```

### 示例四：最后的告别
```
第四天，林深的手机响了。

是二叔的电话。

林深颤抖着接起。

"林深……"二叔的声音很沉重，"你奶奶……走了。"

林深的手机从手中滑落，掉在地上。

"什么时候？"

"两个小时前。"二叔说，"她……她一直在等你，直到最后……"

林深的喉咙发紧，说不出话来。

"她说……让你好好活着，别担心她……"

林深的眼泪决堤般流下来。

他在巴黎的出租屋里，对着电话，痛哭失声。

"奶奶……对不起……对不起……"

"林深，别这样……"二叔说，"奶奶走得很安详……"

"我……我没能见到她最后一面……"林深哽咽道，"我……我没能……"

"不怪你，奶奶知道的。"二叔说，"奶奶说，让你好好活着，这就是她最后的愿望。"

林深跪在地上，眼泪流了一地。

奶奶走了。

最后一面也没见到。

最后的话也没听到。

奶奶一直等他，等到最后……

但他，让奶奶失望了。

"奶奶……对不起……"林深痛哭失声，"对不起……"

窗外，巴黎下起了雨。

就像老天也在为奶奶哭泣。

林深跪在地上，久久没有动。

他知道，从今以后，他再也没有奶奶了。

再也没有人会无条件地爱他，再也没有人会担心他冷不冷、饿不饿……

再也没有奶奶了。

林深闭上眼睛，在雨声中，痛哭失声。
```

## 【情感基调】

- **整体基调**：绝望、无力、超虐
- **情感层次**：
  - 表层：奶奶去世的悲痛
  - 深层：林深的自责
  - 隐层：命运的残酷

- **情感节奏**：
  - 开场：平静的日常
  - 中段：接到噩耗的震惊
  - 高潮：疯狂的尝试
  - 收尾：绝望的告别

---

## 【写作注意事项】

1. **《针锋对决》风格要点**：
   - 奶奶去世要虐心，但不要过度渲染
   - 林深的无力要通过细节体现
   - 最后的告别要写得足够催泪

2. **细节把控**：
   - 林深的疯狂尝试要真实：借钱、打工、询问赊账
   - 奶奶的临终遗言要简单而有力
   - 巴黎的雨景要烘托气氛

3. **节奏控制**：
   - 本章是超虐章节，节奏要从平静到震惊再到绝望
   - 情感层次要丰富，不要单一

4. **伏笔设置**：
   - 为林深的坚强做铺垫
   - 为第24章的被打埋下伏笔

5. **避免事项**：
   - 不要让林深显得过于歇斯底里
   - 不要让奶奶显得过于悲惨，她是安详地走的
   - 不要过度渲染悲剧感，保持克制
