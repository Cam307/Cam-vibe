# 细纲-第65章：同居

## 【基本信息】

- **章节标题**：同居
- **目标字数**：3,500字
- **时间**：第64章之后一周
- **地点**：陆行舟家
- **天气**：晴朗
- **人物状态**：
  - 林深：准备搬入陆行舟家
  - 陆行舟：期待同居生活

---

## 【上下文约束】

**前情提要**：
第64章，两人公开了关系。一周后，陆行舟提议同居。林深思考后同意。本章描写两人同居的第一天。

**核心冲突**：同居的新鲜感 vs 生活的磨合

---

## 【核心情节】

陆行舟提议同居，林深同意。林深搬入陆行舟家，两人开始同居生活。本章描写同居第一天的种种——一起做饭、一起收拾、一起睡觉。结尾，两人在新家相拥而眠。

## 【对话示例】

### 示例一：提议同居
```
"林深。"陆行舟忽然开口。

"嗯？"林深抬头，正在吃饭。

"搬过来和我住吧。"陆行舟说。

林深愣住了："什么？"

"我说，搬过来和我住。"陆行舟说，"我们同居吧。"

林深沉默了。

陆行舟的心提了起来，怕林深拒绝。

"这......太快了吧？"林深问。

"不快。"陆行舟说，"我们已经认识十年了。"

"可是......"林深想说同居是大事。

"我知道你在担心什么。"陆行舟说，"但我想和你一起生活。"

"我想每天醒来第一眼看到你，睡之前最后一眼也看到你。"

"我想和你一起做饭，一起收拾，一起......生活。"

林深的心狠狠一跳。

"你......你真的想吗？"林深问。

"想。"陆行舟点头，"非常想。"

林深看着陆行舟，眼中闪过一丝温柔。

"好。"林深说，"我搬过来。"

陆行舟狂喜，一把抱住林深。

"谢谢你，谢谢你......"陆行舟激动地说。

"不用谢。"林深在陆行舟怀中说，"我也想......和你一起生活。"

两人紧紧相拥，心中充满期待。

从今以后，他们就要住在一起了。

就要......一起生活了。
```

### 示例二：布置新家
```
两人一起布置新家。

"你的书放这里。"陆行舟指着书架。

"好。"林深把书放上去。

"衣服放这个衣柜。"陆行舟打开衣柜，里面已经腾出了一半空间。

"谢谢。"林深笑了。

"不客气。"陆行舟说，"衣柜一人一半，公平。"

林深笑了。

两人继续收拾，很快就收拾好了。

站在收拾好的房间里，两人相视一笑。

"这就是......我们的家了。"陆行舟说。

"嗯。"林深点头，"我们的家。"

"从今以后，这里就是我们两个人的家。"陆行舟说。

"你会在这里吃饭，在这里睡觉，在这里......生活。"

"我知道。"林深笑了。

"你开心吗？"陆行舟问。

"开心。"林深点头，"很开心。"

"我也是。"陆行舟笑了，"我也很开心。"

两人相视而笑，心中充满幸福。

这个家，从此以后就有两个人了。

不再是冷清的一个人。

而是......温暖的两个人。

"林深。"陆行舟忽然说。

"嗯？"

"我爱你。"陆行舟说。

"我也爱你。"林深笑了。

两人相拥，在新家里。

这一刻，是如此美好。
```

### 示例三：第一夜
```
两人洗漱完毕，准备睡觉。

站在床边，两人都有些紧张。

这是他们第一次......一起睡觉。

"你......睡哪边？"陆行舟问。

"随便。"林深笑了，"你呢？"

"我也随便。"陆行舟说。

两人对视，然后都笑了。

"那我睡左边。"陆行舟说。

"好，那我睡右边。"林深说。

两人躺下，关灯。

房间里陷入黑暗，只有窗外的月光洒进来。

两人都能听到对方的呼吸声。

有些紧张，有些......期待。

"晚安。"陆行舟说。

"晚安。"林深说。

沉默了一会儿，陆行舟伸出手。

林深愣了一下，然后自然地靠进陆行舟怀里。

陆行舟的心狠狠一跳。

他搂住林深，林深的头靠在他胸前。

"这样......睡得着吗？"陆行舟问。

"睡得着。"林深的声音很轻，"很安心。"

陆行舟的心中一暖。

"我也是。"陆行舟说，"也很安心。"

两人不再说话，就这样静静地抱着。

很快，林深的呼吸变得平稳。

他睡着了。

陆行舟轻轻吻了吻林深的额头。

"晚安，我的爱人。"陆行舟轻声说。

月光洒在两人身上，温暖而美好。

这就是同居的第一夜。

也是他们......新生活的开始。

第65章，完。
```

- **情感层次**：
  - 表层：同居的过程
  - 深层：一起生活的期待
  - 隐层：家的温暖

- **情感节奏**：
  - 开场：提议同居
  - 中段：搬家和布置
  - 高潮：布置完成
  - 收尾：第一夜

   - 本章是同居章节，节奏要温馨
   - 情感层次要丰富

4. **伏笔设置**：
