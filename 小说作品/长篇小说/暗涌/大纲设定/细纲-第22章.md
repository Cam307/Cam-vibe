# 细纲-第22章：生病

## 【基本信息】

- **章节标题**：生病（回忆篇）
- **目标字数**：3,750字
- **时间**：三年前，林深被骗后三个月
- **地点**：巴黎出租屋、药店
- **天气**：寒冷的冬天
- **人物状态**：
  - 林深：被骗后身无分文，只能打黑工，过度劳累导致生病

---

## 【上下文约束】

**前情提要**：
第21章回忆林深刚到巴黎时被骗走积蓄的经历。第22章继续回忆，三个月后，林深身无分文，只能打黑工维生。过度劳累加上营养不良，林深生病了，却没钱治疗。

**核心冲突**：林深的病痛 vs 无钱治疗的困境

---

## 【核心情节】

被骗后，林深身无分文，只能打黑工维生。他在中餐馆洗碗、在工地搬砖、在街头发传单……每天工作十几个小时，却只能赚到微薄的收入。过度劳累加上营养不良，林深生病了。他躺在床上，发着高烧，却连买药的钱都没有。

---

## 【对话示例】

### 示例一：打黑工的生活
```
林深在中餐馆后厨洗碗，水很凉，手冻得通红。

"快点，客人等着呢！"老板催促道。

"知道了，马上就好。"林深忍着冻疮的疼痛，继续洗碗。

洗完碗，林深马不停蹄地去工地搬砖。

水泥袋很重，肩膀磨破了皮，但他不能停。

"中国人？"工头问，"打工？"

"嗯，需要钱。"林深说。

"干活小心点，出了工伤没人负责。"工头提醒。

"好，谢谢。"林深继续搬砖。

晚上，林深在街头发传单，寒风刺骨。

"先生，看看这个餐厅……"

没有人理会，匆匆走过。

林深站在街头，看着巴黎的夜景。

埃菲尔铁塔在远处闪烁，浪漫而美丽。

但这一切，都不属于他。

他只是这里的过客，一个身无分文、打黑工维生的过客。

林深叹了口气，继续发传单。

他一天打三份工，却只能赚到微薄的收入。

回到出租屋，林深累得连洗澡的力气都没有。

他躺在床上，闭上眼睛。

"林深，加油。"他对自己说，"会好起来的。"
```

### 示例二：生病的前兆
```
林深开始感到不舒服，头很晕，身体发冷。

他强撑着去中餐馆打工。

"你怎么了？脸色这么差。"餐馆老板问。

"没事，可能有点累。"林深强撑着说，手还在洗碗。

"休息一下吧，别到时候晕在我店里。"

"不行，我需要钱。"林深继续洗碗。

老板叹了口气，没有再说什么。

晚上，林深回到出租屋，量了体温——39度。

"没事，睡一觉就好。"林深对自己说。

他吃了一片之前剩的退烧药，躺在床上。

但第二天，体温更高了，林深起床都困难。

他挣扎着起来，准备去打工，但走到门口就晕倒了。

林深倒在地上，意识模糊。

"不能晕……"他对自己说，"还要打工……还要赚钱……"

但身体不听使唤，林深再次失去了意识。

再次醒来时，他躺在地上，浑身冷汗。

林深爬起来，回到床上。

他知道，自己病了，病得很重。

但他没有钱看病，没有钱买药。

只能硬扛。

"林深，你会没事的。"他对自己说，"一定要没事。"
```

### 示例三：奇迹般的好转
```
林深躺在床上，意识模糊。

房间里很冷，他裹着所有被子，还是觉得冷。

"我会不会就这样死在这里？"

这个念头在林深脑海里闪过。

死了也好，就不会痛苦了……

但很快，他又骂自己："林深，你疯了吗？你还有奶奶，你还有……"

奶奶。想到奶奶，林深的心狠狠一痛。

奶奶在国内，不知道他现在的情况。

如果奶奶知道他病成这样，会多担心……

突然，有人敲门。

"你好，我是你的邻居，听到你屋里一直没动静……"

林深想回答，但发不出声音。

门被敲得更响，"有人在吗？需要帮忙吗？"

林深用尽最后的力气，爬到门口，打开了门。

一个年轻的中国男人站在门口，惊讶地看着他。

"天啊，你怎么了？发烧了？"

林深倒在地上，失去了意识。

再次醒来时，他躺在医院里。

"你终于醒了。"邻居宋远说，"你发烧40度，差点死了。"

"谢谢……"林深虚弱地说。

"医药费我帮你垫了，以后慢慢还。"宋远说，"不过你真的要注意身体，差点就……"

"谢谢。"林深说，"我一定会还你的。"

"别想这些了，先养好身体。"宋远说，"对了，我叫宋远，也是来巴黎留学的。"

"我叫林深。"

"我知道，你房门上写着呢。"宋远笑了，"好好休息吧。"

林深闭上眼睛，心中涌起感激。

如果不是宋远，他可能真的会死在出租屋里。

这个世界，还是有好人的。
```

## 【情感基调】

- **整体基调**：艰难、绝望、转机
- **情感层次**：
  - 表层：生病的痛苦
  - 深层：无助与绝望
  - 隐层：人性的温暖

- **情感节奏**：
  - 开场：打黑工的艰辛
  - 中段：生病的前兆
  - 高潮：躺在床上的绝望
  - 收尾：奇迹般的获救

---

## 【写作注意事项】

1. **《针锋对决》风格要点**：
   - 生病的痛苦要真实，不要过度渲染
   - 打黑工的艰辛要通过细节体现
   - 邻居的帮助要自然，不要过于戏剧化

2. **细节把控**：
   - 黑工的场景要真实：洗碗、搬砖、发传单
   - 生病的症状要真实：发烧、头晕、无力
   - 出租屋的环境要写出寒酸和孤独

3. **节奏控制**：
   - 本章是回忆性章节，节奏要从艰辛到危机再到转机
   - 情感层次要丰富，不要单一

4. **伏笔设置**：
   - 为宋远的出现做铺垫（后续会成为重要配角）
   - 为第23章的奶奶去世埋下伏笔

5. **避免事项**：
   - 不要让林深显得过于悲惨
   - 不要让邻居显得过于完美，只是正常的善意
   - 不要过度渲染悲剧感，保持克制
