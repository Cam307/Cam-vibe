# 细纲-第12章：质问

## 【基本信息】

- **章节标题**：质问
- **目标字数**：3,750字
- **时间**：第11章结束后第二天，周三下午至晚上
- **地点**：陆氏集团 → 林深公寓
- **天气**：阴沉，偶有雷声
- **人物状态**：
  - 林深：听到订婚消息后，内心翻涌，决定最后问一次
  - 陆行舟：愧疚，但已经做出决定，无法回头
  - 公司同事：配角，议论订婚消息

---

## 【上下文约束】

**前情提要**：
第11章结尾，陆行舟打电话告诉林深自己要订婚了。林深只说了两个字"恭喜"，就挂断了电话。陆行舟握着手机，心中涌起复杂的情绪。而林深靠在墙上，手机从手中滑落，眼泪无声地流下来。

**核心冲突**：林深的质问 vs 陆行舟的沉默

---

## 【核心情节】

第二天，公司里传遍了陆行舟要订婚的消息。同事们议论纷纷，有人祝贺，有人好奇。林深坐在工位上，听着周围的议论，内心翻涌。下班后，他决定最后问一次陆行舟——为什么？为什么要订婚？是因为苏家的合作？还是因为……别的什么？陆行舟没有正面回答，只是说"对不起"。林深明白了，转身离开。

---

## 【详细场景分解】

### 场景一：公司里的议论（约900字）
- **时间**：周三上午
- **地点**：陆氏集团设计部办公室
- **人物**：林深、同事们
- **情节描述**：
  - 林深来到公司，发现气氛异常
  - 同事们三三两两聚在一起议论，看到林深来，立刻散开
  - "听说了吗？陆总要和苏家大小姐订婚了！"
  - "真的假的？什么时候？"
  - "这周末！订婚宴据说要办得很盛大呢！"
  - 林深假装没听见，坐在工位上打开电脑
  - 但屏幕上的内容，他一个字也看不进去
  - 同事小王凑过来，"林设计师，听说你要升职了？恭喜啊！"
  - 林深愣了一下，"什么？"
  - "陆总说了，这次项目你出力最多，项目结束后就升你做设计总监。"

### 场景二：陆行舟的召见（约900字）
- **时间**：周三下午
- **地点**：陆氏集团总裁办公室
- **人物**：陆行舟、林深
- **情节描述**：
  - 秘书通知林深，陆总叫他
  - 林深来到总裁办公室，陆行舟正在批文件
  - "坐。"陆行舟头也不抬地说
  - 林深坐在对面，看着陆行舟
  - 陆行舟终于放下笔，抬起头，与林深对视
  - "订婚的事……你知道了？"
  - "知道了。"林深平静地说，"恭喜陆总。"
  - 陆行舟的表情僵了一下
  - "林深，我……"
  - "陆总，您叫我上来，是为了升职的事吗？"林深打断他
  - 陆行舟沉默了一会儿，"是的。这次项目你做得很好，公司决定升你做设计总监。"
  - "谢谢陆总。"林深站起身，"如果没别的事，我先出去了。"
  - "林深。"
  - "还有什么事？"
  - 陆行舟张了张嘴，最终只说了两个字："对不起。"

### 场景三：电梯里的相遇（约900字）
- **时间**：周三傍晚
- **地点**：公司电梯
- **人物**：林深、陆行舟
- **情节描述**：
  - 下班时间，林深走进电梯，准备离开
  - 电梯门要关上时，一只手挡住了门
  - 陆行舟走进电梯，站在林深旁边
  - 电梯里只有他们两个人
  - 气氛安静得让人窒息
  - "林深……"
  - "陆总还有事吗？"林深看着电梯门上的数字变化
  - "你……生气了？"
  - 林深转过头，看着陆行舟的眼睛
  - "我为什么要生气？陆总要订婚，是好事。我作为下属，应该为您高兴才对。"
  - 陆行舟被这句话刺得心头一痛
  - "林深，你知道我不是这个意思……"
  - "那您是什么意思？"林深的声音依然平静，"陆总，您已经做出了决定，不是吗？"

## 【对话示例】

### 示例一：公司里的议论
```
林深刚走进办公室，就感觉到气氛异常。

几个同事聚在一起，看到他来，立刻散开，窃窃私语。

"听说了吗？"一个同事小声说，"陆总要和苏家大小姐订婚了！"

"真的假的？什么时候？"

"这周末！订婚宴据说要办得很盛大呢！苏家可是豪门中的豪门，这次陆氏可真是攀上高枝了！"

"那林设计师怎么办？"

"嘘！小声点！人家还没走远呢……"

林深假装没听见，径直走向自己的工位。

打开电脑，屏幕上的设计图映入眼帘，但他一个字也看不进去。

脑海里反复回荡着那两个字——订婚。

订婚。

陆行舟要订婚了。

和苏家大小姐。

林深的手指在键盘上悬着，久久没有落下。

"林设计师！"同事小王凑过来，"听说你要升职了？恭喜啊！"

林深愣了一下，"什么？"

"你还不知道？"小王惊讶地说，"陆总说了，这次项目你出力最多，项目结束后就升你做设计总监！这可是好事啊！"

升职。

设计总监。

林深不知道该笑还是该哭。

陆行舟要订婚了，然后……给他升职？

这就是陆行舟的安排吗？
```

### 示例二：电梯里的质问
```
电梯门缓缓关上，狭小的空间里只有林深和陆行舟两个人。

数字从顶楼慢慢往下跳。

安静。

安静得让人窒息。

陆行舟站在林深旁边，没有说话。

林深看着电梯门上的倒影，两人并排站着，却像隔着一个世界。

"林深。"

陆行舟终于开口。

林深没有回头，"陆总还有事吗？"

"你……生气了？"

林深转过头，看着陆行舟的眼睛。

"我为什么要生气？"

林深的声音很平静，平静得让陆行舟心慌。

"陆总要订婚，是好事。我作为下属，应该为您高兴才对。"

"林深，你知道我不是这个意思……"

"那您是什么意思？"林深看着陆行舟，"陆总，您已经做出了决定，不是吗？"

陆行舟沉默了。

"订婚是两家的意愿，是公司的利益，是……最好的选择。"

"最好的选择。"林深重复这句话，"对谁来说是最好的选择？"

"对所有人。"

"所有人？"林深轻笑一声，"包括我吗？"

陆行舟的心狠狠一抽。

"林深，我……"

叮——

电梯门开了，一楼到了。

林深走出电梯，没有回头。

"恭喜陆总，订婚快乐。"

陆行舟站在电梯里，看着林深远去的背影，电梯门缓缓关上。
```

### 示例三：最后的质问
```
两人坐在楼下的长椅上，夜风吹过，有些凉。

林深低着头，看着自己的影子。

"为什么要订婚？"

终于，林深问出了这个问题。

陆行舟沉默了很久。

"因为……这是最好的选择。"

"最好的选择？"林深抬起头，看着陆行舟，"对谁来说是最好的选择？"

"对公司，对苏家，对……所有人。"

"对你自己呢？"

陆行舟没有回答。

林深看着陆行舟，等待一个答案。

但陆行舟始终没有说话。

"是因为苏家的合作吗？"林深问，"因为苏家提出订婚的条件？"

陆行舟依然沉默。

沉默就是答案。

林深明白了。

"所以我猜对了。"林深轻笑一声，"是为了合作，所以你要订婚。"

"林深，情况很复杂……"

"复杂到必须用订婚来换取合作？"林深问，"陆行舟，你有没有问过自己，这真的是你想要的吗？"

陆行舟转过头，看着林深。

"什么意思？"

"你真的爱苏小姐吗？"林深问，"还是……你只是在履行一个承诺？"

陆行舟沉默了。

"我不知道。"许久，陆行舟说，"但我知道，这是我应该做的。"

"应该做的。"林深重复这句话，"陆行舟，你这辈子，做了多少'应该做的'事？"

陆行舟没有回答。

"算了。"林深站起身，"我不该问这些。"

"林深。"

陆行舟叫住他。

"对不起。"

林深的脚步顿住了。

"别再说对不起。"林深背对着陆行舟，"这三个字，我听够了。"

"林深，我……"

"陆总，您要订婚了。"林深打断他，"恭喜您。"

说完，林深头也不回地走进楼里。

陆行舟坐在长椅上，看着林深消失的楼道入口，久久没有动。
```

## 【情感基调】

- **整体基调**：压抑、质问、无奈
- **情感层次**：
  - 表层：林深的质问
  - 深层：林深的绝望
  - 隐层：陆行舟的动摇

- **情感节奏**：
  - 开场：公司里的议论
  - 中段：陆行舟的愧疚
  - 高潮：电梯里的对话
  - 收尾：最后的质问

---

## 【写作注意事项】

1. **《针锋对决》风格要点**：
   - 质问要克制，不要歇斯底里
   - 陆行舟的沉默要有深意，不是逃避，而是无法回答
   - 林深的平静比爆发更虐心

2. **细节把控**：
   - 公司场景要真实：同事的窃窃私语、工作被打断的烦躁
   - 电梯场景要写出压抑感：狭小的空间、沉默的对峙
   - 夜景场景要写出孤独感：凉风、空荡的街道、遥远的路灯

3. **节奏控制**：
   - 本章是质问性章节，节奏要从压抑到爆发再到收尾
   - 情感层次要丰富，不要单一

4. **伏笔设置**：
   - 为后续林深的离开做铺垫
   - 为陆行舟的后悔埋下伏笔

5. **避免事项**：
   - 不要让林深显得过于歇斯底里
   - 不要让陆行舟显得冷血，他有自己的无奈
   - 不要过度渲染悲剧感，保持克制
