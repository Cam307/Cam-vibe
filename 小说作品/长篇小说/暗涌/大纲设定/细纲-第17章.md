# 细纲-第17章：告别

## 【基本信息】

- **章节标题**：告别
- **目标字数**：3,750字
- **时间**：第16章结束后第二天，周日上午至下午
- **地点**：林深公寓、咖啡厅
- **天气**：晴朗，阳光温暖
- **人物状态**：
  - 林深：经过一夜的思考，决定离开
  - 陆行舟：心慌意乱，想挽回

---

## 【上下文约束】

**前情提要**：
第16章结尾，林深在洗手间崩溃后回到公寓。陆行舟追来，在门外解释，但林深没有开门。陆行舟说"对不起"，林深在门内说"这三个字，我听够了"。陆行舟最终离开，林深无眠一夜。

**核心冲突**：林深的告别 vs 陆行舟的挽留

---

## 【核心情节】

经过一夜的思考，林深做出了决定——他要离开。不是辞职，而是离开这座城市，离开陆行舟的世界。他约陆行舟见面，做最后的告别。陆行舟听到林深说要离开，慌了，试图挽留，但林深心意已决。

---

## 【对话示例】

### 示例一：一夜无眠后的决定
```
林深坐在沙发上，看着窗外的天慢慢变亮。

一夜没睡，但他的精神却异常清醒。

他翻看着手机里的照片，一张一张删除。

这张是在公司年会上拍的，陆行舟笑着，他也笑着。

这张是在加班的深夜，陆行舟拍的，两人头靠在一起，都在看电脑。

这张是陆行舟发来的自拍，附文："在出差，好无聊，想你了。"

林深看着这些照片，眼泪又流了下来。

但他还是删除了，一张不留。

每删除一张，心就痛一下。

但他还是删除了。

林深打开电脑，搜索"巴黎设计学院"。

屏幕上出现录取通知书的信息：

"亲爱的林深先生，恭喜您被巴黎设计学院录取……"

林深之前申请过，被录取了，但当时因为陆行舟，他拒绝了。

现在，他重新打开这封录取通知书。

"林深，够了。"他对自己说，"你为他做了那么多，现在……该为自己活了。"

林深收拾行李，把衣服、书、设计稿装进箱子。

他只带走必需品，其他的，都不要了。

那个陆行舟送他的杯子，不要了。

那件陆行舟送他的外套，不要了。

那些和陆行舟有关的回忆，也不要了。

林深看着收拾好的箱子，深吸一口气。

"林深，新的开始。"他对自己说。
```

### 示例二：约陆行舟见面
```
林深给陆行舟打电话。

电话响了三声，被接通。

"喂？林深？"陆行舟的声音带着焦急，"你昨晚……还好吗？"

"我很好。"林深平静地说，"陆总，有空吗？我想见你。"

"有，当然有！"陆行舟立刻说，"你在哪里？我去接你。"

"不用了，老地方见吧。"林深说，"中午12点，星光咖啡厅。"

"好，好，我一定到。"陆行舟说，"林深，昨天的事……我真的很抱歉……"

"见面再说吧。"林深打断他，"陆总，中午见。"

"好，中午见。"

林深挂断电话，看着手机屏幕。

陆行舟的声音里有焦急，有歉意，有……挽留？

但林深不想再听这些了。

他决定要走了。

离开这座城市，离开陆行舟的世界，离开这段没有结果的感情。

林深收拾好最后的行李，看着空荡荡的房间。

这个房间，他住了三年。

三年里，陆行舟来过无数次。

他们在这里吃饭，在这里聊天，在这里……

够了，真的够了。

林深提起箱子，走出房间。

门关上的时候，他没有回头。
```

### 示例三：咖啡厅里的告别
```
陆行舟早早就到了，坐在靠窗的位置，焦急地等待。

他不停地看手机，又看门口，心中涌起强烈的不安。

林深的态度……太平静了。

平静得让人害怕。

陆行舟想起昨晚，林深没有开门，他在门外站了很久才离开。

那一夜，林深在想什么？

陆行舟不敢想。

门口传来响动，陆行舟抬头，看到林深走进来。

林深穿着简单的白衬衫和牛仔裤，手里拿着一个文件袋。

陆行舟注意到，林深的脸色有些苍白，眼圈有些发红。

"林深……"陆行舟站起来。

"坐。"林深坐下，"我有事要告诉你。"

陆行舟坐下，心中涌起强烈的不安。

"什么事？"

"我决定辞职。"林深平静地说。

陆行舟愣住了，"什么？"

"我要去巴黎。"林深继续说，"去进修设计。"

"为什么？"陆行舟慌了，"林深，是因为昨天的事吗？我……我可以解释……"

"不是因为昨天的事。"林深打断他，"是因为我想了很久，觉得……我该为自己活了。"

"林深……"陆行舟握住林深的手，"别走，好吗？"

林深抽回手，"陆总，这是我的辞职信。"他把文件袋推过去。

陆行舟看着桌上的文件袋，手在颤抖。

"林深，别走。"陆行舟说，"你要什么条件我都答应你。加薪？升职？还是……"

"这不是条件的问题。"林深说，"陆总，我已经决定了。"

"是因为苏婉音吗？"陆行舟问，"是因为我要订婚吗？"

"不是。"林深摇头，"陆总，您要订婚，是您的事。我做什么，是我的事。"

"林深……"陆行舟的声音有些颤抖。

"陆总，祝您订婚快乐。"林深站起来，"也祝您……幸福。"

"林深！"陆行舟站起来，抓住林深的手，"别走，求你。"

林深看着陆行舟，眼中闪过一丝痛苦。

"陆总，放手吧。"林深说，"我们之间……从来就没有开始过，又何谈结束？"

林深抽回手，转身离开。

陆行舟站在原地，看着林深的背影，心狠狠地抽痛。
```

### 示例四：最后的对话
```
"林深，等等！"陆行舟追出门。

林深停下脚步，但没有回头。

"陆总还有事吗？"

"林深，我……"陆行舟不知道该说什么，"我真的不知道昨天该怎么办。"

"我知道。"林深平静地说，"陆总，我不怪您。"

"那为什么要走？"

"因为……"林深顿了顿，"因为我不想再等了。"

"等什么？"

"等一个永远不会到来的人。"林深轻声说。

陆行舟的心狠狠一抽。

"林深，我……"

"陆总，您不需要解释什么。"林深说，"我们都是成年人，都明白有些事……是不可能改变。"

"什么不可能改变？"

"所有。"林深说，"您的订婚，我的离开，都是……必然的。"

"林深……"

"陆总，保重。"林深说完，头也不回地走了。

陆行舟站在原地，看着林深的背影消失在人群中。

心中涌起强烈的恐慌。

林深真的要走了。

离开他的世界，离开这座城市，去巴黎。

陆行舟突然意识到，他可能……真的要失去林深了。

这一次，不是暂时的，不是可以挽回的。

而是……永远的失去。
```

## 【情感基调】

- **整体基调**：决绝、遗憾、无可挽回
- **情感层次**：
  - 表层：告别
  - 深层：放弃
  - 隐层：双方的不舍与无奈

- **情感节奏**：
  - 开场：一夜无眠后的决定
  - 中段：约陆行舟见面
  - 高潮：咖啡厅里的告别
  - 收尾：最后的对话

---

## 【写作注意事项】

1. **《针锋对决》风格要点**：
   - 告别要克制，不要歇斯底里
   - 陆行舟的挽留要真诚，但不要显得太卑微
   - 林深的决绝要从痛苦中诞生

2. **细节把控**：
   - 一夜无眠的场景要真实：黑眼圈、干涩的眼睛、疲惫的身体
   - 咖啡厅场景要细节：阳光、咖啡香、窗外的景色
   - 告别的场景要虐心：平静的语气、颤抖的手、闪躲的眼神

3. **节奏控制**：
   - 本章是告别性章节，节奏要从思考到决定再到执行
   - 情感层次要丰富，不要单一

4. **伏笔设置**：
   - 为第18章的机场场景做铺垫
   - 为后续的三年分离埋下伏笔

5. **避免事项**：
   - 不要让林深显得过于歇斯底里
   - 不要让陆行舟显得冷血
   - 不要过度渲染悲剧感，保持克制
