# 细纲-第6章：醉酒

## 【基本信息】

- **章节标题**：醉酒
- **目标字数**：3,750字
- **时间**：周六早上，7:00-11:00
- **地点**：林深公寓
- **天气**：雨后初晴，阳光透过窗帘缝隙洒进来
- **人物状态**：
  - 陆行舟：宿醉头痛，记忆模糊，在陌生环境醒来感到困惑和尴尬
  - 林深：一夜未眠好，照顾陆行舟后心力交瘁，但表面保持克制

---

## 【上下文约束】

**前情提要**：
第5章结尾，陆行舟醉酒后在林深公寓过夜。醉酒的他在林深面前暴露了脆弱，甚至说了一些模棱两可的话。林深默默照顾，在陆行舟睡着后在他额头轻轻印下一个吻。但林深知道，明天醒来，一切都会回到原点——陆行舟会将这一切归为酒后失态，不会当真。

**核心冲突**：陆行舟的尴尬与逃避 vs 林深的克制与隐忍

---

## 【核心情节】

陆行舟在林深公寓醒来，发现自己酒后失态，内心充满尴尬。他试图回忆昨晚发生的事情，但记忆支离破碎。林深表现得克制而专业，将一切都解释为"下属的应尽职责"。陆行舟虽然感激，但内心深处有一丝说不清道不明的失落。两人都小心翼翼地维持着表面的平静。

## 【对话示例】

### 示例一：清晨的尴尬相遇
```
陆行舟走出房间时，闻到了一股淡淡的粥香。

循着香味来到客厅，他看到林深正在开放式厨房里忙碌。清晨的阳光从落地窗洒进来，给那个身影镀上了一层柔和的金边。

听到脚步声，林深转过身。

两人的目光在空中相遇，时间仿佛凝固了几秒。

"醒了？"林深率先开口，语气平静得像是任何普通的工作日早晨。

陆行舟清了清嗓子，声音沙哑："昨晚……"

"您醉得很厉害。"林深转过身继续搅拌锅里的粥，"酒吧的酒保给我打电话，我去接了您。"

陆行舟努力回想，但记忆一片混乱，只有零星的片段——酒吧的灯光、摇晃的视野、还有一双温暖的手扶住他。

"抱歉，给你添麻烦了。"他低声说。

"不麻烦。"林深将粥盛好，端到餐桌上，"喝点粥，对胃好。"

陆行舟走过去，看着那碗熬得恰到好处的皮蛋瘦肉粥，心中涌起一种复杂的情绪。

"你……没睡好？"他注意到林深眼底淡淡的青黑。

"还好。"林深避开他的目光，"您昨晚吐了几次，一直折腾到凌晨。"

陆行舟的脸微微发热。他在林深面前吐了？还折腾到凌晨？

"抱歉。"他又说了一遍，声音更低了。

"吃吧。"林深将筷子递给他，"凉了就不好吃了。"

陆行舟接过筷子，低头喝粥。温热的液体滑入胃部，缓解了宿醉的不适。

但他心里清楚，有些不适，这碗粥是缓解不了的。
```

### 示例二：记忆碎片与逃避
```
"昨晚……"陆行舟放下筷子，犹豫了片刻，"我说了什么吗？"

林深正在收拾碗筷的手顿了一下。

"您醉了，说了一些胡话。"林深平静地说，没有回头。

"什么胡话？"陆行舟追问。

"没有什么重要的。"林深转过身，脸上看不出任何表情，"都是醉话，不必放在心上。"

陆行舟盯着林深的眼睛，试图从那双平静的眼眸中读出些什么。但林深的目光平静无波，像一潭深不见底的湖水。

"我有没有……"陆行舟顿了顿，"有没有说什么不合适的话？"

林深的睫毛微微颤动了一下，但很快恢复了平静。

"没有。"他说，"您只是说了些公司的事，还有……家里的压力。"

陆行舟的心中闪过一丝失落。

"那就好。"他站起身，"麻烦你了，林深。"

"应该的。"林深低下头继续收拾碗筷，"您是老板，照顾您是下属的职责。"

下属的职责。

这五个字像一把刀，将两人之间刚刚建立起来的某种暧昧联系彻底斩断。

陆行舟点点头，"那我先回去了。"

"我送您。"林深放下手中的碗筷，"您现在开车不安全。"

"不用……"

"我送您。"林深的语气不容置疑，"这也是下属的职责。"

陆行舟看着林深，最终没有再拒绝。

但他心里清楚，有些东西已经变了。而这些变化，他不知道该如何面对。
```

### 示例三：临别前的未尽之言
```
车停在陆行舟家楼下时，两人都沉默了片刻。

"谢谢。"陆行舟解开安全带，"昨晚……还有今天早上，麻烦你了。"

"不麻烦。"林深目视前方，"应该的。"

陆行舟的手放在车门把手上，却迟迟没有推开。

"林深。"他突然开口。

"嗯？"林深转过脸。

陆行舟看着林深，张了张嘴，想说些什么，却又不知从何说起。

"如果……"他顿了顿，"如果以后有什么需要帮忙的，可以告诉我。"

林深的嘴角勾起一抹淡淡的微笑，"好的，谢谢陆董。"

又是这种客套的称呼。陆行舟心中一紧，但很快压了下去。

"那……周一见。"

"周一见。"

陆行舟推开车门下车，站在路边看着林深的车缓缓驶离。

直到红色的尾灯消失在拐角，他才转身走进楼道。

电梯上升的数字不断跳动，陆行舟靠在轿厢壁上，闭上眼睛。

昨晚的记忆碎片又开始闪回——他拉着林深的手不肯松开，说"别走"，说"我不想娶什么苏婉"，还有那句未完的话："我只想……"

我只想什么？

陆行舟摇了摇头，驱散那些混乱的思绪。

都是醉话。正如林深所说，都是醉话，不必当真。

但为什么，他的心里会有一种说不清道不明的失落？
```

## 【情感基调】

- **整体基调**：尴尬、克制、暧昧后的疏离
- **情感层次**：
  - 表层：上司与下属之间的客套
  - 深层：陆行舟对昨晚失态的尴尬和困惑
  - 隐层：两人都清楚昨晚说了什么，但都选择装作不知

- **情感节奏**：
  - 开场：陆行舟的困惑与尴尬
  - 中段：两人之间的微妙试探
  - 收尾：回归日常后的空虚感

---

## 【写作注意事项】

1. **《针锋对决》风格要点**：
   - 人物对话要克制，不要有过多的情绪外露
   - 尴尬要通过细节体现：视线回避、话语停顿、动作僵硬
   - 林深的克制要贯穿始终，即使内心翻江倒海，表面也要平静

2. **细节把控**：
   - 陆行舟的宿醉状态要真实：头痛、口干、记忆模糊
   - 林深的状态也要体现：一夜未睡好的疲惫、眼底的黑眼圈
   - 环境描写要烘托氛围：雨后初晴的阳光、安静的公寓

3. **节奏控制**：
   - 本章节奏较慢，重点在于两人之间的心理博弈
   - 每一个对话都要暗藏深意

4. **伏笔设置**：
   - 陆行舟对昨晚记忆的不完整，为后续回忆更多细节埋下伏笔
   - 林深的过度克制，暗示他内心的真实情感

5. **避免事项**：
   - 不要让陆行舟显得过于软弱，他的尴尬是有尊严的
   - 不要让林深显得卑微，他的克制是因为尊重
