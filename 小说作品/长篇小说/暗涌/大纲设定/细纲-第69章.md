# 细纲-第69章：订婚

## 【基本信息】

- **章节标题**：订婚
- **目标字数**：3,500字
- **时间**：第68章之后一周
- **地点**：家中
- **天气**：晴朗
- **人物状态**：
  - 林深：幸福，期待订婚
  - 陆行舟：骄傲，准备订婚宴

---

## 【上下文约束】

**前情提要**：
第68章，陆行舟向林深求婚成功。一周后，两人准备订婚宴。这次和第15章的订婚宴完全不同——这次是林深和陆行舟的订婚宴。

**核心冲突**：过去的阴影 vs 现在的幸福

---

## 【核心情节】

两人准备订婚宴，这次是真正的订婚——林深和陆行舟的订婚宴。家人朋友都来祝福。结尾，两人在订婚宴上跳舞，林深说"这次，是真的了"。

---

## 【对话示例】

### 示例一：准备订婚宴
```
"请哪些人？"陆行舟问。

"亲朋好友就好。"林深说，"不想太张扬。"

"好，听你的。"陆行舟说。

两人一起选择场地、布置、菜单。

"这个怎么样？"陆行舟指着一张图片。

"太华丽了。"林深说，"我喜欢简单的。"

"那就选这个。"陆行舟指着另一张图片。

"这个不错。"林深笑了。

两人继续选择，很快就决定好了。

"这次......"林深忽然说，"和上次不一样了。"

陆行舟的心狠狠一痛。

"对不起。"陆行舟说。

"不用道歉。"林深摇头，看着陆行舟的眼睛。

"上次是......和别人的订婚宴。"

"这次是......我们的订婚宴。"

"完全不一样。"

陆行舟的心中一暖。

"这次是真的。"陆行舟说。

"嗯。"林深点头，"这次是真的。"

"是你和我。"

"对，是我和你。"陆行舟说。

两人相视一笑，继续准备订婚宴。

这一次，没有遗憾。

只有幸福。
```

### 示例二：苏婉的祝福
```
苏婉走进宴会厅，穿着一件优雅的连衣裙。

"深哥，陆总，恭喜。"苏婉笑着说。

"谢谢。"林深和陆行舟都说。

"我没想到......你们真的在一起了。"苏婉说。

"我也没想到。"林深笑了。

"但很高兴。"苏婉说，"真的很高兴。"

"上次......"苏婉顿了顿，"上次我不该说那些话。"

"没关系。"林深说，"都过去了。"

"但我还是想说对不起。"苏婉说。

"没关系，真的。"林深说。

"而且，上次那件事......"苏婉看向陆行舟，"陆总也没有怪我。"

"我没怪你。"陆行舟说。

"谢谢。"苏婉笑了，"你们真是......太好人了。"

"这次，我是真心来祝福你们的。"苏婉说。

"谢谢。"林深说，"我们也真心接受。"

三人相视而笑，过去的恩怨都放下了。

"祝你们幸福。"苏婉说。

"谢谢。"林深和陆行舟都说。

"也祝你幸福。"林深说。

苏婉笑了："我会的。"

苏婉离开后，陆行舟握住林深的手。

"你真好。"陆行舟说。

"她也不坏。"林深笑了，"只是......那时候我们都年轻。"

"现在我们都长大了。"陆行舟说。

"嗯。"林深点头，"都长大了。"
```

### 示例三：跳舞
```
音乐响起，两人进入舞池。

陆行舟搂着林深，两人慢慢起舞。

"你会跳舞吗？"林深问。

"不会。"陆行舟笑了，"但我想和你跳。"

"那随便踩。"林深笑了。

"好。"陆行舟说。

两人慢慢起舞，虽然动作不熟练，但很默契。

"这次，是真的了。"林深忽然说。

"嗯。"陆行舟点头，"这次是真的。"

"没有抄袭指控，没有......"林深想说"没有羞辱"。

"没有。"陆行舟打断林深，"只有我们。"

"只有我们两个人。"

"嗯。"林深点头，"只有我们。"

"而且......"林深抬起头，看着陆行舟的眼睛。

"什么？"

"这次你保护我了。"林深说。

陆行舟的心狠狠一跳。

"我以后都会保护你。"陆行舟说。

"我知道。"林深笑了。

两人继续跳舞，音乐温柔而浪漫。

周围的人都看着他们，眼中都是祝福。

这一刻，是如此美好。

美好得让人想永远停留。

"我爱你。"陆行舟轻声说。

"我也爱你。"林深笑了。

两人在舞池中央拥吻。

这一次，是真正的订婚。

真正的幸福。

第69章，完。
```
