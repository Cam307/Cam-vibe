## 【上下文约束】

**前情提要**：
第71章，林深向陆行舟求婚成功。今天，是他们的婚礼。在花园里，亲朋好友的见证下，两人结为夫妻。

## 【核心情节**

今天是林深和陆行舟的婚礼。在花园里，亲朋好友的见证下，两人交换戒指，宣读誓词，结为夫妻。结尾，两人拥吻，在阳光下，众人祝福。

## 【对话示例】

### 示例一：婚礼当天早上
```
"今天......就要结婚了。"陆行舟说。

"嗯。"林深点头，"有点紧张。"

"我也是。"陆行舟笑了。

两人一起洗漱，一起吃早餐。

"你后悔吗？"陆行舟忽然问。

"后悔什么？"林深问。

"后悔......和我结婚。"陆行舟说。

"不后悔。"林深摇头，看着陆行舟的眼睛。

"从来不后悔。"

"而且......"林深顿了顿，"我很期待。"

"期待......和你结婚。"

"期待......成为你的丈夫。"

陆行舟的心狠狠一跳。

"我也是。"陆行舟说。

"我也是......不后悔。"

"而且......我也很期待。"

两人相视而笑，心中充满幸福。

门铃响了，是顾阳来接林深。

"新郎官，准备好了吗？"顾阳笑着说。

"准备好了。"林深笑了。

"那走吧。"顾阳说，"陆总在等你。"

林深站起来，看着陆行舟。

"待会见。"林深说。

"待会见。"陆行舟笑了。

林深跟着顾阳离开，陆行舟目送他离开。

今天......

他们就要结婚了。
```

### 示例二：婚礼仪式
```
音乐响起，婚礼开始。

林深走在红毯上，陆行舟在尽头等他。

红毯两边，坐满了亲朋好友。

顾阳、小张、苏婉......都在。

他们脸上都带着祝福的笑容。

林深一步步走向陆行舟，心跳如鼓。

终于，他走到了陆行舟面前。

"你真帅。"陆行舟轻声说。

"你也是。"林深笑了。

司仪开始主持："今天，我们见证林深和陆行舟的结合......"

"在这美好的日子里，他们将结为夫妻，共度一生。"

"现在，请两位交换戒指。"

陆行舟拿出戒指，为林深戴上。

"这枚戒指，象征着我永恒的爱。"陆行舟说。

林深也拿出戒指，为陆行舟戴上。

"这枚戒指，象征着我永恒的爱。"林深说。

司仪继续："现在，请两位宣读誓词。"

陆行舟先开口："林深，我愿意娶你为妻，无论富贵贫穷，健康疾病，都永远爱你，保护你，珍惜你。"

"这是我的承诺，永远不变。"

林深接着说："陆行舟，我愿意娶你为夫，无论富贵贫穷，健康疾病，都永远爱你，陪伴你，支持你。"

"这是我的承诺，永远不变。"

司仪："你们，愿意吗？"

"我愿意。"两人异口同声地说。
```

### 示例三：拥吻
```
司仪："现在，你们可以亲吻新郎了。"

陆行舟捧起林深的脸，轻轻吻下去。

林深闭上眼睛，感受着这个吻。

这是......他们的婚礼之吻。

是......他们成为夫妻的吻。

众人鼓掌，洒下花瓣。

花瓣飘落，洒在两人身上。

"我爱你。"陆行舟在林深耳边说。

"我也爱你。"林深笑了。

两人分开，但手还握在一起。

"从今以后，我们就是夫妻了。"陆行舟说。

"嗯。"林深点头，"夫妻。"

"一辈子的夫妻。"

"对。"林深笑了，"一辈子的夫妻。"

两人相视而笑，眼中都是幸福。

司仪宣布："现在，我宣布，林深和陆行舟，正式结为夫妻！"

众人再次鼓掌，欢呼。

"恭喜你们！"顾阳喊道。

"谢谢。"林深和陆行舟都说。

两人手牵手，走向宾客。

阳光洒在他们身上，温暖而美好。

这一刻，是如此幸福。

幸福得让人想永远停留。

从此以后，他们就是夫妻了。

一辈子的夫妻。

第72章，完。
```
