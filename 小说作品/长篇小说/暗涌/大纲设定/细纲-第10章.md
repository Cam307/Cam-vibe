# 细纲-第10章：奶奶病重

## 【基本信息】

- **章节标题**：奶奶病重
- **目标字数**：3,750字
- **时间**：一周后，周五上午至周六下午
- **地点**：老家医院
- **天气**：阴沉转晴，象征某种转折
- **人物状态**：
  - 林深：奶奶病重，心力交瘁，对陆行舟的期待已经降到最低
  - 奶奶：病危，思念孙子，临终前想见林深一面
  - 医生、护士：配角
  - 陆行舟：本章节末尾出现，但为时已晚

---

## 【上下文约束】

**前情提要**：
第9章结尾，林深在公司会议上晕倒，陆行舟才知道他生病了。陆行舟送林深去医院，内心充满愧疚。但林深的心已经凉透了——在最需要的时候，陆行舟不在；现在知道了，又能如何？一周后，林深接到老家医院的电话，奶奶病危。他请假赶回老家，在医院陪伴奶奶度过最后的时光。而陆行舟……依然没有来。

**核心冲突**：林深的孤独无助 vs 陆行舟的缺席

---

## 【核心情节】

一周后，林深接到老家医院电话，奶奶病危。他请假赶回老家，在医院里陪伴奶奶。奶奶清醒时问起林深的感情状况，林深无法回答。奶奶说想见见林深"喜欢的人"，林深下意识地拿出手机，但最终没有拨出那个号码。奶奶在周六下午去世，而陆行舟在奶奶去世后才赶到，一切都太晚了。

## 【对话示例】

### 示例一：奶奶的询问
```
"深儿……"

奶奶的声音很轻，林深凑近了才听清。

"奶奶，我在。"林深握紧奶奶的手，"您要什么？"

"奶奶想问你……"奶奶的眼皮颤动着，费力地睁开，"你……有没有……喜欢的人？"

林深的心狠狠一痛，"奶奶，怎么突然问这个……"

"奶奶只是……想看你……幸福……"奶奶虚弱地说，"那个人……对你好吗？"

林深沉默了。病房里只有仪器的滴答声，安静得让人窒息。

"他……很好。"许久，林深低声说，"只是……我们之间……不可能。"

"为什么不可能？"奶奶问，"是因为……他不喜欢你？"

林深摇头，眼泪掉了下来，"不是……是因为……很多原因。"

"什么原因……能比……幸福更重要？"

林深哽咽了。他无法回答这个问题。

"深儿……"奶奶的手轻轻拍了拍林深的手背，"奶奶这辈子……最后悔的……就是没有……勇敢追求自己的幸福……你……不要像奶奶一样……"

林深抬起头，泪眼朦胧地看着奶奶。

"不管那个人是谁……"奶奶继续说，"只要他对你好……奶奶就……就放心了……"

"奶奶……"林深再也忍不住，趴在床边痛哭起来。

奶奶的手轻轻抚摸着他的头发，就像小时候那样。

"别哭……深儿……奶奶在呢……"
```

### 示例二：未能拨出的电话
```
"想……想见见他……"

奶奶的声音很模糊，像是梦呓，但林深听得很清楚。

他的心狠狠一震。

"奶奶，您说什么？"

"那个人……"奶奶费力地说，"想……想见……"

林深的喉咙发紧。他知道奶奶在说什么。

奶奶一直想见他"喜欢的人"，想看看那个让她孙子魂牵梦萦的人。但林深从未带任何人回家给奶奶看，因为……他带不来。

"奶奶……"林深哽咽道，"他……他很忙……"

奶奶似乎有些失望，发出一声叹息。

林深的心如刀割。他下意识地拿出手机，找到陆行舟的号码。

打过去吗？

告诉陆行舟，奶奶病危，想见他一面？

林深的手指悬在拨号键上方，颤抖着。

但他很快压下了这个念头。

陆行舟会来吗？即使来了，又能如何？以什么身份？下属？朋友？

陆行舟正忙着与苏家的合作，正忙着……走向他应该有的未来。

林深最终按下了锁屏键，将手机放回口袋。

"奶奶……他……来不了……"林深哽咽道，"真的……来不了……"

奶奶似乎听到了，眼中闪过一丝失望，但很快又柔和下来。

"没关系……"奶奶虚弱地说，"只要……只要你……幸福……就好……"

林深的泪如雨下。
```

### 示例三：奶奶去世后
```
周六下午三点，奶奶走了。

林深握着奶奶渐渐变凉的手，眼泪已经流干了。

医生进来宣布死亡时间，护士们开始撤走仪器。林深坐在那里，一动不动。

直到有人轻轻拍了拍他的肩膀。

"林深……"

熟悉的声音让林深猛地回过头。

陆行舟站在病房门口，身上还穿着出差的衣服，显然是赶了很久的路。

"你……"林深看着陆行舟，一时说不出话来。

"我看到你的信息就赶过来了。"陆行舟走近，"对不起，我来晚了……"

林深看着陆行舟，心中涌起复杂的情绪。

来晚了。

是啊，来晚了。

奶奶已经走了。带着对他的祝福，带着对他"幸福"的期待，带着……对他"喜欢的人"的好奇，永远地走了。

而陆行舟，在她去世后才赶到。

"她……已经走了。"林深平静地说，声音沙哑。

陆行舟的表情僵住了。

"什么时候？"

"半个小时前。"林深低下头，"奶奶……一直在等你。"

陆行舟的身体晃了晃，像是被人狠狠打了一拳。

"她……她说什么了吗？"

林深抬起头，看着陆行舟的眼睛。

"她说……"林深顿了顿，"她说只要我幸福就好。"

陆行舟的眼中闪过痛苦。

"林深……"

"没关系。"林深站起身，松开奶奶的手，"反正……你也来不了。"

这句话说得很平静，却让陆行舟的心狠狠地抽痛。

林深转身离开病房，留给陆行舟一个决绝的背影。

陆行舟站在原地，看着病床上那张安详的面容，心中充满了无法言说的愧疚。
```

## 【情感基调】

- **整体基调**：悲伤、遗憾、决绝
- **情感层次**：
  - 表层：奶奶病重去世的悲痛
  - 深层：林深的孤独与失望
  - 隐层：两人关系的不可逆转的裂痕

- **情感节奏**：
  - 开场：紧急情况下的焦虑
  - 中段：医院里的温情与痛苦
  - 高潮：未能拨出的电话
  - 收尾：奶奶去世后的决绝

---

## 【写作注意事项】

1. **《针锋对决》风格要点**：
   - 悲伤要克制，不要过度渲染
   - 人物对话要简洁，暗藏深意
   - 陆行舟的迟到要有合理的解释，不要让他显得冷血

2. **细节把控**：
   - 医院场景要真实：仪器的声音、消毒水的味道、病房的冷清
   - 奶奶的形象要立体：慈祥、对孙子的爱
   - 林深的情绪要通过细节体现：颤抖的手、哽咽的声音、含泪的眼睛

3. **节奏控制**：
   - 本章是悲剧性章节，节奏要从紧急到缓慢再到爆发
   - 情感层次要丰富，不要单一

4. **伏笔设置**：
   - 为林深和陆行舟关系的彻底破裂做铺垫
   - 为后续陆行舟的挽回埋下伏笔

5. **避免事项**：
   - 不要让奶奶显得可怜，她是慈祥而坚强的
   - 不要让陆行舟显得不负责任，他有合理的理由
   - 不要过度渲染悲剧感，保持克制
