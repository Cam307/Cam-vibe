## 【对话示例】

### 示例一：阳光下的早晨
```
阳光洒进房间，两人醒来。

"今天......天气真好。"林深说。

"是啊。"陆行舟笑了，"阳光明媚。"

"我们......出去走走吧。"

"好。"陆行舟说。

两人慢慢起床，准备出门。

"穿这件衣服吧。"陆行舟拿了一件外套。

"好。"林深接过外套，慢慢穿上。

"我帮你。"陆行舟说。

陆行舟帮林深扣上扣子，动作温柔而熟练。

"好了。"陆行舟说。

"谢谢。"林深笑了。

两人互相搀扶，慢慢走出家门。

阳光洒在他们身上，温暖而美好。

"今天......真的很好。"林深说。

"是。"陆行舟笑了，"因为有你在。"

"我也因为有你在，很好。"林深笑了。

两人相视而笑，继续走着。

在阳光下。

手牵手。

就像......他们年轻时一样。

只是现在......

他们老了。

但......

爱依然在。
```

### 示例二：回顾一生
```
两人坐在长椅上，回顾一生。

"从18岁到现在，我们经历了太多。"陆行舟说。

"是啊。"林深点头，"太多了。"

"暗恋、暧昧、错过、分离、重逢、在一起、结婚、白头偕老......"

"每一步，都不容易。"

"但......每一步，都值得。"

"对。"林深笑了，"每一步都值得。"

"因为我们......走到了最后。"

"走到了......一起。"

"而且......"林深顿了顿，"很幸福。"

"很圆满。"

陆行舟握住林深的手。

"老伴。"陆行舟说。

"嗯？"

"谢谢你。"陆行舟说。

"谢什么？"林深问。

"谢谢你......陪我走完这一生。"陆行舟说。

"谢谢你......让我幸福。"

"不用谢。"林深笑了，"我也谢谢你。"

"谢我什么？"

"谢谢你......爱我。"林深说。

"谢谢你......让我也幸福。"

两人相视而笑，心中充满感激。

这一生......

真的很好。

因为有彼此。

所以......

幸福。

圆满。
```

### 示例三：在阳光下
```
两人站起来，继续在阳光下散步。

手紧紧握在一起。

"老伴。"陆行舟说。

"嗯？"

"我爱你。"陆行舟说。

"我也爱你。"林深笑了。

"一直爱你。"

"从18岁到现在，一直爱你。"

"我也是。"陆行舟笑了。

"而且......"陆行舟顿了顿，"以后也会一直爱。"

"到什么时候？"林深问。

"到永远。"陆行舟说。

林深的心狠狠一跳。

"好。"林深笑了，"到永远。"

"我们也一起到永远。"

"对。"陆行舟笑了，"一起到永远。"

两人继续走着，阳光洒在他们身上。

温暖而美好。

"如果......"林深忽然说，"如果有来世......"

"如果有来世，我们还要在一起。"陆行舟打断林深。

"好。"林深笑了，"来世还在一起。"

"而且......"陆行舟说，"来世早点相遇。"

"早点......在一起。"

"好。"林深点头，"来世早点在一起。"

两人相视而笑，继续走着。

在阳光下。

手牵手。

到永远。

这就是他们的大结局。

在阳光下，手牵手。

到永远。

第79章，完。
```
