# 细纲-第38章：照顾

## 【基本信息】

- **章节标题**：照顾
- **目标字数**：3,750字
- **时间**：第37章结束后第三天
- **地点**：林深公寓
- **天气**：阴雨
- **人物状态**：
  - 林深：生病发烧，虚弱
  - 陆行舟：得知消息后赶来照顾

---

## 【上下文约束】

**前情提要**：
第37章结尾，陆行舟在奶奶墓前忏悔，听到林深说"后悔删除他的联系方式"，心中燃起一丝希望。三天后，宋远给陆行舟打电话，说林深生病了，发高烧，一个人在家。

**核心冲突**：陆行舟的照顾 vs 林深的拒绝

---

## 【核心情节】

林深生病发烧，一个人在家。陆行舟得知后赶来照顾，林深拒绝，说"我不需要你照顾"。陆行舟没有离开，而是留下来照顾林深——喂药、做饭、陪夜。林深在高烧中恍惚，叫出了陆行舟的名字。陆行舟心疼，握着林深的手，整夜未眠。

---

## 【对话示例】

### 示例一：陆行舟赶来
```
门铃响了。

林深想站起来开门，但头晕得厉害，根本站不起来。

他坐在沙发上，犹豫着要不要去开门。

门铃持续响，似乎很着急。

"进来......"林深虚弱地喊，"门没锁......"

门被推开，陆行舟冲了进来。

"林深！"

陆行舟看到林深瘫在沙发上，脸色苍白，满头大汗。

他冲过来，蹲在林深面前，摸他的额头。

"好烫......"陆行舟皱眉，"你发烧了？"

"一点......"林深虚弱地说，"没什么......"

"什么叫没什么？"陆行舟说，"你这样子像没什么？"

"你怎么来了......"林深问。

"宋远给我打电话，说你生病了。"陆行舟说，"我来看看你。"

"我没事......"林深想站起来。

"别动。"陆行舟按住他，"你躺着。"

"我真的......"

"我说别动。"陆行舟语气不容置疑。

林深看着陆行舟，最终放弃挣扎，躺回沙发上。

陆行舟的手放在林深的额头上，眉头紧皱。

"至少39度......"陆行舟说，"家里有退烧药吗？"

"没有......"林深说，"用完了......"

陆行舟叹了口气："等着，我去买。"

"不......不用......"林深想阻止，但陆行舟已经转身出门。

林深躺在沙发上，闭上眼睛。

陆行舟......

他又在照顾自己了。

就像七年前一样。

那时候林深生病，陆行舟也是这样，紧张得不行。

但现在......

现在不一样了。

他们已经不是朋友了。

但陆行舟......还是在照顾他。
```

### 示例二：照顾
```
陆行舟回来了，带着药和水。

"起来。"陆行舟说，"先吃药。"

林深挣扎着坐起来，伸手想拿药。

但他的手在发抖，根本拿不住。

"给我。"陆行舟拿过药，"张嘴。"

"我自己来......"林深说。

"你手都在抖，怎么吃？"陆行舟直接把药送到林深嘴边。

林深看着陆行舟，最终张嘴吃药。

陆行舟又喂他喝水。

"好点了吗？"陆行舟问。

"嗯......"林深点头。

"还要不要吃点东西？"陆行舟问。

"不用了。"林深说，"我不饿。"

"你生病了，需要吃东西。"陆行舟说，"我去给你煮粥。"

"你不需要这么做。"林深说。

"我想做。"陆行舟看着林深，"让我照顾你，好吗？"

林深沉默了很久。

"为什么？"林深问。

"什么？"

"为什么要照顾我？"林深问，"我们......我们已经没有关系了。"

陆行舟沉默了一会儿。

"因为......"陆行舟说，"因为我在乎你。"

"林深，不管我们是什么关系......"

"我都不想看到你生病一个人在家。"

林深看着陆行舟，说不出话。

陆行舟的眼睛里，有真诚。

有......关心。

林深的心，微微颤动。

但他很快就压下了这种感觉。

"谢谢。"林深最终说，"但真的......不需要了。"

"我需要。"陆行舟说。

林深愣住了。

"我需要照顾你。"陆行舟说，"这让我......感觉好一点。"

林深看着陆行舟，最终点了点头。

"那......麻烦你了。"

"不麻烦。"陆行舟笑了，"等着，我去煮粥。"
```

### 示例三：整夜陪伴
```
深夜，林深躺在床上，昏昏沉沉。

烧还没有退，反而更高了。

陆行舟坐在床边，用冷毛巾给林深擦脸。

"深儿......"陆行舟轻声叫道，"好点了吗？"

林深没有回应，只是昏昏沉沉地躺着。

半夜，林深开始说胡话。

"奶奶......"林深轻声说，"我想你......"

陆行舟握住林深的手。

"我在。"陆行舟说。

"不是你......"林深说，"是奶奶......"

"我知道。"陆行舟说，"奶奶在看着你呢。"

"陆行舟......"林深忽然叫道。

陆行舟的心狠狠一抽。

"我在。"陆行舟说，"我在，深儿，我在。"

"你在哪里......"林深说，"我找不到你......"

"我就在这里。"陆行舟握紧林深的手，"我哪里都不去，就在这里陪着你。"

"真的？"林深问。

"真的。"陆行舟说，"我发誓。"

林深似乎安心了，安静下来。

陆行舟坐在床边，看着林深。

林深的脸色还是很苍白，眉头微微皱着，像是在做什么梦。

陆行舟的心，疼得厉害。

林深......还在乎自己。

否则他不会在梦中叫自己的名字。

不会说"我找不到你"。

陆行舟握紧林深的手，放在自己的唇边。

"深儿......"陆行舟轻声说，"我不会再让你找不到我了。"

"我会一直在你身边。"

"一直。"

窗外，雨还在下。

但房间里，有温暖。

陆行舟没有离开，他就这样坐着，握着林深的手，整夜未眠。
```

## 【情感基调】

- **整体基调**：温暖、照顾、心动
- **情感层次**：
  - 表层：陆行舟的照顾
  - 深层：林深的动摇
  - 隐层：两人关系的微妙变化

- **情感节奏**：
  - 开场：林深生病
  - 中段：陆行舟的照顾
  - 高潮：林深高烧中叫出陆行舟的名字
  - 收尾：陆行舟整夜陪伴

---

## 【写作注意事项】

1. **《针锋对决》风格要点**：
   - 陆行舟的照顾要细致，但不要过于宠溺
   - 林深的反应要真实，不要过于抗拒
   - 对话要有目的

2. **细节把控**：
   - 林深发烧时的细节描写
   - 陆行舟照顾林深的动作
   - 最后握手的温暖画面

3. **节奏控制**：
   - 本章是转折性章节，节奏要平稳
   - 情感层次要丰富

4. **伏笔设置**：
   - 林深高烧中叫陆行舟名字的伏笔
   - 为两人关系的修复做铺垫

5. **避免事项**：
   - 不要让陆行舟显得过于卑微
   - 不要让林深显得过于矫情
   - 保持平衡，两人的情感都要真实
