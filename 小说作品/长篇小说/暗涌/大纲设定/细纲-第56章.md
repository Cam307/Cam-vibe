## 【对话示例】

### 示例一：早餐时的对话
```
两人一起吃早餐。

陆行舟做的早餐很简单——小米粥、鸡蛋、小菜。

但很好吃。

"好吃吗？"陆行舟问。

"嗯。"林深点头。

"那就好。"陆行舟笑了。

两人陷入沉默，但这次是温暖的沉默。

"林深。"陆行舟开口。

"嗯？"林深抬头。

"你还记得大学时的事吗？"陆行舟问。

林深愣了一下，然后点头。

"记得。"

"全部？"

"大部分。"林深说。

"那......"陆行舟顿了顿，"你还记得我们第一次相遇吗？"

林深的心狠狠一颤。

他当然记得。

怎么会忘记？

那天，阳光下的陆行舟，笑容灿烂。

就是那一刻，林深......喜欢上陆行舟了。

"记得。"林深说，"你带我去教室。"

"对。"陆行舟笑了，"那时候你很紧张，脸都红了。"

林深的脸......又红了。

"你......"林深想说什么。

"我只是......"陆行舟说，"我只是想知道，你还记不记得。"

"我记得。"林深说。

"那就好。"陆行舟说。

林深看着陆行舟，心中涌起复杂的情绪。

陆行舟......在试探他？

试探他还记不记得过去？

试探......他还喜不喜欢陆行舟？

林深......

他不知道答案。

他只知道......

他还在乎陆行舟。

还在乎......很多。
```

### 示例二：试探
```
晚上，两人坐在沙发上看电视。

电视里放着一部老电影，是大学时他们一起看过的。

"你还记得这部电影。"陆行舟说。

"嗯。"林深点头，"我们一起看过。"

"对。"陆行舟笑了，"那时候你睡着了，靠在我肩膀上。"

林深的脸又红了。

"我......"林深想解释。

"不用解释。"陆行舟说，"我很怀念那个时候。"

"我也是。"林深轻声说。

陆行舟转头看林深，鼓起勇气。

"林深，我有话想问你。"

"什么？"林深转头。

"你......"陆行舟顿了顿，"你还喜欢我吗？"

林深愣住了。

还喜欢吗？

他......

他还喜欢陆行舟吗？

七年的暗恋，三年的苦难......

现在......

现在林深......还喜欢陆行舟吗？

林深不知道。

他只知道......

他还在乎陆行舟。

但喜欢......

"我......"林深想说什么。

"没关系。"陆行舟说，"你不需要回答。"

"我只是......想知道。"

"知道什么？"

"知道......我还有没有机会。"陆行舟说。

林深的心狠狠一跳。

机会......

陆行舟还在等机会？

"林深。"陆行舟继续说，"我知道我错过了太多。"

"我知道我伤害了你。"

"但我想弥补。"

"我想......让你重新爱上我。"

林深听着陆行舟的话，心中涌起复杂的情绪。

陆行舟......

真的很真诚。

真的很用心。

"我......"林深最终说，"我需要时间。"

"多久都可以。"陆行舟说。

"那......"林深看着陆行舟，"给我一些时间。"

"好。"陆行舟笑了，"我会等的。"

"多久都可以。"

林深看着陆行舟，心中涌起温暖的情绪。

陆行舟......

真的......在等他。
```
