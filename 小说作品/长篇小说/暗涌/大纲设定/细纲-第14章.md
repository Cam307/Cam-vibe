# 细纲-第14章：订婚筹备

## 【基本信息】

- **章节标题**：订婚筹备
- **目标字数**：3,750字
- **时间**：第13章结束后一周，周四至下周三
- **地点**：公司、希尔顿酒店、花店、设计工作室
- **天气**：晴转多云
- **人物状态**：
  - 林深：全力投入订婚宴设计，用工作麻痹自己
  - 陆行舟：订婚筹备繁忙，但心中不安
  - 苏婉音：兴奋于即将到来的订婚宴

---

## 【上下文约束】

**前情提要**：
第13章结尾，林深的沉默让陆行舟心慌。陆行舟想找林深谈谈，但林深总是礼貌而疏离地回避。订婚的日子越来越近，陆行舟的不安越来越强烈。

**核心冲突**：林深为订婚宴设计 vs 内心的痛苦

---

## 【核心情节】

订婚宴进入筹备阶段，林深作为设计负责人，全程参与。他用工作麻痹自己，把订婚宴设计得完美无缺。苏婉音对林深的设计非常满意，邀请他参加订婚宴。林深答应了，语气平静得让陆行舟心慌。

---

## 【详细场景分解】

- **时间**：周四上午
- **地点**：希尔顿酒店宴会厅
- **人物**：林深、陆行舟、苏婉音、酒店经理
- **情节描述**：
  - 三人来到希尔顿酒店，考察订婚宴场地
  - 酒店经理介绍宴会厅的情况
  - "这个宴会厅可容纳300人，有独立舞台……"
  - 苏婉音挽着陆行舟的手臂，看起来很亲密
  - 林深拿着笔记本，认真记录场地信息

  - 陆行舟时不时看向林深，但林深始终专注于工作
  - 考察结束后，苏婉音说："林设计师，辛苦你了。订婚宴那天，你一定要来！"
  - "谢谢苏小姐邀请。"林深平静地说，"我会到的。"

### 场景二：花店选花（约900字）
- **时间**：周五下午
- **地点**：高级花店
- **人物**：林深、陆行舟、苏婉音、花店老板

  - 陆行舟站在一旁，看着林深
  - 林深的专业、专注，让陆行舟想起他们第一次见面时的场景
  - 那时的林深，也是这样专注地设计着什么
  - "行舟，你觉得呢？"苏婉音问
  - 陆行舟回过神，"什么？"
  - "花的选择，你有什么建议？"
  - "我……"陆行舟看向林深，"听林深的就好。"

### 场景三：深夜加班（约950字）
- **时间**：周一晚上
- **地点**：设计工作室
- **人物**：林深、陆行舟
- **情节描述**：

  - 陆行舟进来，带来夜宵
  - "这么晚还在工作？"
  - "嗯，还有一些细节需要调整。"林深没有抬头

  - "谢谢陆总。"林深停下手中的工作，打开夜宵
  - 陆行舟站在一旁，看着林深吃东西的样子
  - "林深……"
  - "陆总还有事吗？"
  - "订婚宴……你真的会来吗？"
  - "当然。"林深抬起头，"这是我负责的项目，我一定会到场确保万无一失。"
  - 陆行舟的心狠狠一抽——林深说的是"确保万无一失"，不是"祝福你"

## 【对话示例】

### 示例一：场地考察
```
希尔顿酒店的宴会厅宽敞明亮，水晶吊灯从天花板垂下，折射出璀璨的光芒。

"这个宴会厅可容纳300人，有独立舞台，还有专门的休息区……"酒店经理详细介绍着。

苏婉音挽着陆行舟的手臂，脸上满是笑容。

"行舟，这里怎么样？"

"很好。"陆行舟点头，但眼睛却不由自主地看向林深。

林深拿着笔记本，正在认真记录场地信息，完全没有注意到陆行舟的视线。

"林设计师，你觉得怎么样？"苏婉音问。

林深抬起头，专业地分析："场地很合适。不过我建议做一些调整——舞台的位置可以稍微移动，这样主宾席的视角会更好。还有灯光，我建议用暖色调，会更有温馨感……"

苏婉音眼睛发亮："好！都听你的！林设计师真的很专业！"

林深微微一笑："谢谢苏小姐夸奖。"

陆行舟看着林深的笑容，心中涌起复杂的情绪。

林深在笑，但这个笑容……太标准了。

就像一个面具，完美地掩盖了所有情绪。

"行舟，你觉得呢？"苏婉音问。

陆行舟回过神："什么？"

"场地的调整，你有什么建议？"

陆行舟看向林深。

林深正好也在看他，眼中一片平静。

"我……"陆行舟说，"听林深的就好。"

林深收回视线，继续记录笔记。

陆行舟的心狠狠一抽。

林深……真的变了。
```

### 示例二：花店选花
```
花店里弥漫着淡淡的花香。

苏婉音站在花架前，拿起一支白玫瑰。

"我想要白色的玫瑰，代表纯洁的爱情。"苏婉音笑着说，"行舟，你觉得呢？"

陆行舟看向林深。

林深拿起一支白玫瑰，仔细端详。

"苏小姐，"林深开口，"除了白玫瑰，我建议加一些香槟玫瑰。"

"为什么？"

"白色很纯净，但可能显得有些冷清。"林深专业地解释，"香槟玫瑰的色调更温暖，搭配白色，会有层次感，也更有氛围。"

苏婉音眼睛发亮："好！都听你的！"

林深放下白玫瑰，又拿起几支香槟玫瑰，对比着颜色。

陆行舟站在一旁，看着林深。

林深专注的样子，让陆行舟想起他们第一次见面时的场景。

那天的林深，也是这样专注地设计着什么，眉宇间带着认真和执着。

"行舟，你怎么了？"苏婉音问。

陆行舟回过神："没什么。"

"花的选择，你有什么建议？"

陆行舟看向林深。

"听林深的就好。"陆行舟说。

苏婉音有些奇怪地看了陆行舟一眼，但没有说什么。

林深似乎没有注意到他们的对话，继续挑选着花材。

陆行舟的心中涌起一阵酸楚。

林深在为他们挑选订婚宴的花材。

林深在为他们设计订婚宴的场地。

林深……在亲手筹备陆行舟和别人的订婚宴。

陆行舟的心狠狠地抽痛。
```

### 示例三：深夜加班
```
设计工作室里，只有林深一个人。

电脑屏幕的光照亮他的脸，上面是订婚宴的设计图。

林深专注地调整着细节，完全忘记了时间。

门被推开，陆行舟走了进来。

"这么晚还在工作？"

林深抬起头，看到陆行舟，立刻站起来。

"陆总，您怎么来了？"

"路过，看到你这里还亮着灯。"陆行舟把夜宵放在桌上，"先吃点东西吧。"

"谢谢陆总。"林深打开夜宵，是热粥和小菜。

陆行舟站在一旁，看着林深吃东西的样子。

林深吃得很慢，一口一口地喝着粥，像是机械地在完成一项任务。

"林深……"

林深抬起头："陆总还有事吗？"

"订婚宴……你真的会来吗？"

"当然。"林深放下勺子，"这是我负责的项目，我一定会到场确保万无一失。"

陆行舟的心狠狠一抽。

林深说的是"确保万无一失"，不是"祝福你"，不是"为你高兴"。

"林深，我……"陆行舟想说什么，但不知道该怎么说。

"陆总，我知道您想说什么。"林深平静地说，"您放心，我会把工作做好。这是我的职责。"

"不是工作的问题……"

"那是什么？"林深看着陆行舟。

陆行舟张了张嘴，却回答不上来。

他是什么意思？

他想让林深做什么？

不参加订婚宴？大哭一场？质问他为什么要订婚？

"没什么。"陆行舟最终说，"你慢慢吃，我先走了。"

"好的，陆总慢走。"

陆行舟转身离开，走到门口时，林深突然开口。

"陆总。"

陆行舟停下脚步，回头。

"怎么了？"

"恭喜您。"林深平静地说，"真诚地。"

陆行舟的心狠狠地抽痛。

"谢谢。"陆行舟说，声音有些哑。

陆行舟走出工作室，靠在门外的墙上，久久没有动。

林深的"恭喜"，比任何质问都让他心碎。
```

## 【情感基调】

- **整体基调**：压抑、隐忍、专业
- **情感层次**：
  - 表层：订婚宴的筹备
  - 深层：林深的痛苦
  - 隐层：陆行舟的不安和愧疚

- **情感节奏**：
  - 开场：场地考察
  - 中段：花店选花
  - 高潮：深夜加班
  - 收尾：最终定稿

## 【写作注意事项】

1. **《针锋对决》风格要点**：
   - 林深的专业比歇斯底里更虐心
   - 陆行舟的愧疚要通过细节体现
   - 苏婉音要显得无辜而不讨人厌

2. **细节把控**：
   - 场地考察要专业：宴会厅的布局、灯光、音响
   - 花店场景要唯美：花的种类、颜色、香味
   - 工作室场景要孤独：深夜的灯光、安静的空间、一个人的身影

3. **节奏控制**：
   - 本章是筹备性章节，节奏要从日常到压抑再到累积
   - 情感层次要丰富，不要单一

4. **伏笔设置**：
   - 为第15章的订婚宴做铺垫
   - 为后续的冲突埋下伏笔

5. **避免事项**：
   - 不要让林深显得过于歇斯底里
   - 不要让苏婉音显得刻薄，她是无辜的
   - 不要过度渲染悲剧感，保持克制
