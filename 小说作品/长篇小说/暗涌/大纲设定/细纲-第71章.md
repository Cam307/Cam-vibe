## 【核心情节】

婚礼前一天，林深准备了一个惊喜——他也要向陆行舟求婚。林深拿出戒指，单膝跪地："陆行舟，娶我好吗？"陆行舟深受感动，点头答应。两人拥抱，泪流满面。

## 【对话示例】

### 示例一：林深的准备
```
林深一个人去珠宝店。

"林先生，您来了。"店员笑着说。

"嗯。"林深点头。

"这是您订的戒指。"店员拿出一个盒子。

林深打开，里面是一枚银色的戒指，简洁而优雅。

和陆行舟给他的那一枚，是一对。

"很漂亮。"林深说。

"是的。"店员说，"这是给陆先生的？"

"嗯。"林深点头。

"你们要结婚了？"

"嗯。"林深笑了，"明天就要结婚了。"

"恭喜您。"店员说。

"谢谢。"林深笑了，合上戒指盒。

他要给陆行舟一个惊喜。

一个......双向的惊喜。

陆行舟向他求过婚。

现在，他也要向陆行舟求婚。

这样......才公平。

才......完整。

林深拿着戒指盒，心中充满期待。

今晚......他要向陆行舟求婚了。

向......他最爱的人求婚。
```

### 示例二：阳台上
```
两人站在阳台上，看着夜景。

"明天就要结婚了。"陆行舟说。

"嗯。"林深点头。

"紧张吗？"陆行舟问。

"有一点。"林深说。

"我也是。"陆行舟笑了。

两人沉默了一会儿，看着夜景。

城市灯火辉煌，美得像一幅画。

"林深。"陆行舟忽然说。

"嗯？"

"谢谢你。"陆行舟说。

"谢什么？"林深问。

"谢谢你......愿意嫁给我。"陆行舟说。

林深的心狠狠一跳。

"不用谢。"林深说，"是我愿意的。"

"但还是要谢谢你。"陆行舟说，"因为你......你让我有了家。"

"你也是。"林深说，"你也让我有了家。"

两人相视而笑，心中充满温暖。

"陆行舟。"林深忽然开口。

"嗯？"

"我有话想对你说。"林深说。

"什么话？"陆行舟问。

林深深吸一口气，单膝跪地。

陆行舟愣住了。

"林深，你......"

林深拿出戒指盒，打开。

里面是一枚银色的戒指，简洁而优雅。

和林深手上的那一枚，是一对。

"陆行舟。"林深认真地说，"娶我，好吗？"
```

### 示例三：双向的爱
```
陆行舟的眼泪掉了下来。

"你......你真的......"

"我是认真的。"林深说，"非常认真。"

"我想和你结婚，想和你过一辈子。"

"想每天醒来第一眼看到你，睡之前最后一眼也看到你。"

"想和你一起变老，一起......走完这一生。"

"陆行舟，你愿意娶我吗？"

陆行舟哽咽了。

"我......我愿意。"陆行舟说。

林深站起来，为陆行舟戴上戒指。

"谢谢你，谢谢你......"林深激动地说。

"不用谢。"陆行舟在林深怀中说，"是我愿意的。"

两人紧紧相拥，泪流满面。

"我爱你。"陆行舟哽咽道。

"我也爱你。"林深说。

两人拥吻，在月光下。

双向的求婚。

双向的爱。

双向的......承诺。

"从今以后，我们就是夫妻了。"陆行舟说。

"嗯。"林深点头，"夫妻。"

"一辈子的夫妻。"

"对。"林深笑了，"一辈子的夫妻。"

两人紧紧相拥，在月光下。

明天......

他们就要结婚了。

第71章，完。
```
