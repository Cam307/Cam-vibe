# 细纲-第19章：离开

## 【基本信息】

- **章节标题**：离开
- **目标字数**：3,000字
- **时间**：第18章结束后当天，周二下午至晚上
- **地点**：飞机上、巴黎酒店
- **天气**：晴朗，云层之上
- **人物状态**：
  - 陆行舟：回到公司后，发现自己无法专注于工作
  - 林深：在飞机上，删除所有联系方式

---

## 【上下文约束】

**前情提要**：
第18章结尾，林深在机场等待，但陆行舟到的时候已经晚了。林深已经登机，飞机起飞。陆行舟站在登机口，看着飞机消失在云层中，心狠狠地抽痛。双视角的错过，写得无比虐心。

**核心冲突**：彻底的断联 vs 双方的不舍

---

## 【核心情节】

飞机上，林深做出最后的决定——删除所有联系方式。微信、电话、照片……一切和陆行舟有关的痕迹，全部删除。飞机落地巴黎，林深入住酒店，在这个陌生的城市开始新的生活。而陆行舟回到公司，发现自己无法专注于工作，满脑子都是林深。

---

## 【详细场景分解】

### 场景一：飞机上的决定（约900字）
- **时间**：周二下午4:30
- **地点**：飞机上
- **人物**：林深、邻座乘客
- **情节描述**：
  - 飞机平稳飞行，机舱里很安静
  - 林深坐在靠窗的位置，看着窗外的云海
  - 他打开手机，发现没有信号
  - "等飞机落地，就要彻底删除了。"他对自己说
  - 邻座的大妈和他搭话："小伙子，去巴黎旅游？"
  - "不是，我去学习。"林深笑了笑
  - "哦，那挺好的。巴黎很美，你会喜欢的。"
  - 林深点头，"谢谢。"
  - 他闭上眼睛，深吸一口气
  - 这次离开，不是逃避，而是……重新开始

### 场景二：删除所有联系方式（约900字）
- **时间**：周二下午5:00（巴黎时间上午11:00）
- **地点**：飞机上 → 巴黎机场
- **人物**：林深
- **情节描述**：
  - 飞机开始降落，林深打开手机
  - 信号恢复，消息涌入
  - 苏婉音的消息："林设计师，听说你辞职了？为什么？"
  - 同事的问候："林哥，听说你去巴黎了？保重啊！"
  - 还有……陆行舟的未接来电，十几个
  - 林深看着这些消息，深吸一口气
  - 他打开微信，找到陆行舟的对话框
  - 最后一条消息是陆行舟发的："林深，对不起。"
  - 林深看着这条消息，心中涌起复杂的情绪
  - 最终，他按下删除键
  - "删除联系人？"系统提示
  - 林深按下"确定"
  - 然后，他删除了陆行舟的电话号码
  - 删除了所有和陆行舟有关的照片
  - 删除了所有和陆行舟有关的聊天记录
  - 手机里，再也没有陆行舟的痕迹了

### 场景三：抵达巴黎（约600字）
- **时间**：周二下午6:00（巴黎时间中午12:00）
- **地点**：巴黎机场 → 酒店
- **人物**：林深、酒店前台
- **情节描述**：
  - 飞机落地，林深走出机场
  - 巴黎的阳光很明媚，空气里有淡淡的香味
  - "Bonjour，先生。"酒店前台微笑着说
  - "Bonjour。"林深用蹩脚的法语回应
  - "您的房间在五楼，这是房卡。"
  - "谢谢。"
  - 林深走进房间，窗外是巴黎的街景
  - 铁塔、教堂、老建筑……一切都那么陌生，又那么美好
  - 林深把行李箱放下，躺在床上
  - 他终于离开了
  - 离开了那个城市，离开了那个人，离开了那段没有结果的感情

### 场景四：陆行舟的失落（约600字）
- **时间**：周二晚上8:00
- **地点**：陆行舟办公室
- **人物**：陆行舟
- **情节描述**：
  - 陆行舟回到公司，但无法专注于工作
  - 他时不时拿起手机，想给林深发消息
  - 但发过去的消息，始终没有回复
  - "林深，你平安了吗？"
  - "林深，我们能不能谈谈？"
  - "林深……"
  - 所有的消息，都石沉大海
  - 陆行舟拨打林深的电话
  - "您拨打的电话已关机……"
  - 不，不是关机，是……空号？
  - 陆行舟的心狠狠一抽
  - 他打开微信，想给林深发消息
  - 但系统提示："对方已删除好友……"
  - 陆行舟握着手机，久久没有动

## 【对话示例】

### 示例一：飞机上的决定
```
飞机平稳飞行，机舱里很安静。

林深坐在靠窗的位置，看着窗外的云海。

他打开手机，发现没有信号。

"等飞机落地，就要彻底删除了。"他对自己说。

邻座的大妈和他搭话："小伙子，去巴黎旅游？"

"不是，我去学习。"林深笑了笑。

"哦，那挺好的。巴黎很美，你会喜欢的。"大妈笑着说，"我去过巴黎，埃菲尔铁塔、卢浮宫、香榭丽舍大道……都很美。"

"谢谢。"林深说。

"一个人去？"

"嗯，一个人。"

"那要小心啊，巴黎小偷多。"大妈叮嘱道。

"好，谢谢提醒。"

大妈笑了笑，转头看窗外去了。

林深看着窗外的云海，心中涌起复杂的情绪。

他要去巴黎了。

一个人。

在这个陌生的城市，开始新的生活。

没有陆行舟。

没有那些回忆。

只有他自己。

"林深，加油。"他对自己说。
```

### 示例二：删除所有联系方式
```
飞机开始降落，林深打开手机。

信号恢复，消息涌入。

苏婉音的消息："林设计师，听说你辞职了？为什么？如果是因为订婚宴的事……"

同事的问候："林哥，听说你去巴黎了？保重啊！我们会想你的！"

还有……陆行舟的未接来电，十几个。

林深看着这些消息，深吸一口气。

他打开微信，找到陆行舟的对话框。

最后一条消息是陆行舟发的："林深，对不起。"

林深看着这条消息，心中涌起复杂的情绪。

对不起。

这三个字，他听过太多次了。

第一次，陆行舟因为工作失约，说对不起。

第二次，陆行舟因为忘记纪念日，说对不起。

第三次，陆行舟因为订婚宴上的沉默，说对不起。

每一次，林深都选择了原谅。

但这一次，他不想再听对不起了。

林深按下删除键。

"删除联系人？"系统提示。

林深按下"确定"。

然后，他删除了陆行舟的电话号码。

删除了所有和陆行舟有关的照片。

删除了所有和陆行舟有关的聊天记录。

手机里，再也没有陆行舟的痕迹了。

林深看着手机屏幕，眼泪滑落。

"再见了，陆行舟。"

"这一次，是真的再见。"
```

### 示例三：陆行舟的失落
```
陆行舟回到公司，但无法专注于工作。

他时不时拿起手机，想给林深发消息。

"林深，你平安了吗？"发送成功。

"林深，我们能不能谈谈？"发送成功。

"林深……"发送成功。

但所有的消息，都石沉大海。

陆行舟拨打林深的电话。

"嘟——嘟——"

"您拨打的电话已关机……"

不，不是关机。

陆行舟再拨了一次。

"您拨打的电话是空号，请查证后再拨……"

陆行舟的心狠狠一抽。

空号？

林深……换了号码？

陆行舟打开微信，想给林深发消息。

但系统提示："对方已删除好友，请先添加对方为好友。"

陆行舟握着手机，久久没有动。

林深……删了他？

林深换了号码，删除了微信，断绝了所有联系方式。

陆行舟靠在椅背上，闭上眼睛。

他终于明白，林深是真的要走了。

不是暂时的离开，而是……彻底的消失。

陆行舟的心中涌起强烈的恐慌。

他可能……真的要失去林深了。

这一次，可能真的……无法挽回了。
```

## 【情感基调】

- **整体基调**：决绝、遗憾、新的开始
- **情感层次**：
  - 表层：断联
  - 深层：彻底的告别
  - 隐层：双方的不舍

- **情感节奏**：
  - 开场：飞机上的平静
  - 中段：删除联系方式的决绝
  - 高潮：抵达巴黎的新生
  - 收尾：陆行舟的失落

---

## 【写作注意事项】

1. **《针锋对决》风格要点**：
   - 断联要决绝，不要拖泥带水
   - 陆行舟的失落要通过细节体现
   - 林深的新生要从痛苦中诞生

2. **细节把控**：
   - 飞机场景要真实：云海、颠簸、邻座的对话
   - 巴黎场景要浪漫：阳光、香味、异国风情
   - 删除联系方式的场景要虐心：犹豫、决绝、解脱

3. **节奏控制**：
   - 本章是过渡性章节，节奏要从平静到决绝再到新生
   - 情感层次要丰富，不要单一

4. **伏笔设置**：
   - 为第一卷的尾声做铺垫
   - 为第二卷的三年后埋下伏笔

5. **避免事项**：
   - 不要让林深显得过于歇斯底里
   - 不要让陆行舟显得过于卑微
   - 不要过度渲染悲剧感，保持克制
