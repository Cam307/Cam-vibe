## 【对话示例】

### 示例一：回顾一生
```
"从18岁相识到现在，47年了。"陆行舟说。

"是啊。"林深点头，"47年了。"

"这47年里，我们经历了太多。"

"暗恋、暧昧、错过、分离、重逢......"

"但最终......我们走到了一起。"

"对。"林深笑了，"最终走到了一起。"

"而且......很幸福。"

"很幸福，很幸福。"

陆行舟握住林深的手。

"老伴。"陆行舟说。

"嗯？"

"谢谢你。"陆行舟说。

"谢什么？"林深问。

"谢谢你......这47年，一直在我身边。"陆行舟说。

"谢谢你......没有放弃我。"

"不用谢。"林深笑了，"我也谢谢你。"

"谢我什么？"

"谢谢你......这47年，一直爱着我。"林深说。

"谢谢你......让我幸福。"

两人相视而笑，心中充满感慨。

47年了。

从18岁到65岁，整整47年。

这47年里，他们经历了太多。

但最终......

他们圆满了。

真的圆满了。
```

### 示例二：天伦之乐
```
孩子们带着孙辈来看望。

"爸爸，爷爷。"孩子们叫着。

"哎。"林深和陆行舟应着。

"这是您们的孙子。"儿子说。

"好，好。"林深笑着，抱起孙子。

"这是您们的孙女。"女儿说。

"好，好。"陆行舟笑着，抱起孙女。

一家人聚在一起，其乐融融。

"你们......很幸福。"林深对孩子们说。

"我们幸福是因为你们。"女儿说。

"是你们抚养我们长大，给了我们一个家。"

"不用谢。"陆行舟说，"你们是我们的孩子。"

"我们爱你们。"

"我们也爱你们。"孩子们说。

一家人继续聊天，享受天伦之乐。

林深和陆行舟看着孩子们，心中充满满足。

这一生......

真的很圆满。

有爱人相伴。

有孩子承欢。

还有什么不满足的呢？

"老伴。"陆行舟悄悄对林深说。

"嗯？"

"我爱你。"陆行舟说。

"我也爱你。"林深笑了。

两人相视而笑，继续享受天伦之乐。
```

### 示例三：圆满的感慨
```
两人站在阳台上，看着夜景。

"你说，这一生......圆满吗？"陆行舟问。

"圆满。"林深点头，"很圆满。"

"因为有你在。"

"我也是。"陆行舟笑了，"因为有你。"

两人沉默了一会儿，看着夜景。

"老伴。"陆行舟忽然说。

"嗯？"

"谢谢你。"陆行舟说。

"谢什么？"林深问。

"谢谢你......给了我一个圆满的一生。"陆行舟说。

"没有你，我不会这么圆满。"

林深的心狠狠一跳。

"我也是。"林深说。

"没有你，我也不会这么圆满。"

两人相视而笑，紧紧相拥。

这一生......

真的圆满。

从18岁相识，到25岁重逢，到28岁在一起，到65岁......

整整47年。

47年的相伴。

47年的相爱。

47年的......圆满。

"下一个47年，我们还要一起。"陆行舟说。

"傻瓜。"林深笑了，"哪有下一个47年。"

"那就......下辈子。"陆行舟说。

"下辈子，我们还要在一起。"

"好。"林深点头，"下辈子，我们还在一起。"

两人相拥，在月光下。

这一生，圆满。

下辈子，也要圆满。

第78章，完。
```
