# 细纲-第27章：竞标

## 【基本信息】

- **章节标题**：竞标
- **目标字数**：3,750字
- **时间**：林深回国后一周
- **地点**：陆氏集团、林深办公室
- **天气**：晴朗
- **人物状态**：
  - 林深：回国后，开始新工作，准备参加陆氏项目的竞标
  - 陆行舟：得知林深回国，心神不宁

---

## 【上下文约束】

**前情提要**：
第26章结尾，林深在巴黎三年后，决定回国。宋远不舍但支持他的决定。林深收拾行李，准备回国。第27章开始，林深回国后一周，开始新工作，准备参加陆氏项目的竞标。

**核心冲突**：林深的参与 vs 陆行舟的心神不宁

---

## 【核心情节】

林深回国后，加入了一家设计公司，担任设计总监。他得知陆氏集团有一个大型项目正在招标，决定参与竞标。陆行舟得知林深回国了，心神不宁，到处打听林深的消息。竞标会即将开始，两人将在同一个会议室里见面。

---

## 【详细场景分解】

### 场景一：回国后的新生活（约900字）
- **时间**：林深回国后三天
- **地点**：林深的新办公室
- **人物**：林深、同事
- **情节描述**：
  - 林深回国后，加入了一家知名设计公司
  - "欢迎林设计师加入我们！"同事们都热情地打招呼
  - "谢谢大家。"林深微笑着说，左脸上的伤疤若隐若现
  - 同事们注意到伤疤，但没有多问
  - "林设计师，听说您在巴黎很有名？"同事小王问
  - "还行吧。"林深谦虚地说
  - "那您为什么要回国？"
  - "因为……有些事情需要处理。"林深说
  - 他没有说具体是什么事情，但同事们都能感觉到，林深有故事

### 场景二：得知陆氏项目（约900字）

  - "林设计师？"老板问

## 【对话示例】

### 示例一：得知陆氏项目
```
公司里传来消息，陆氏集团有一个大型项目正在招标。

"这个项目很重要，如果我们能拿下，公司会上一个台阶。"老板在会议上说。

"老板，您决定让谁负责这次竞标？"同事问。

老板看向林深。

"林设计师，你愿意负责这次竞标吗？"

林深愣住了。

陆氏集团。陆行舟的公司。

他要参与陆氏项目的竞标？要和陆行舟见面？

"林设计师？"老板问。

林深深吸一口气。

他逃避了三年，现在……是时候面对了。

"我愿意。"林深说。

"太好了！"老板高兴地说，"我相信你一定能为公司拿下这个项目！"

林深点头，但心中涌起复杂的情绪。

三年了。

他终于要见到陆行舟了。

"林设计师，您还好吗？"同事小王担心地问，"您的脸色……有点苍白。"

"我没事。"林深说，"可能是有点累。"

"那您要好好休息，这次竞标很重要……"

"我知道。"林深说，"我会尽力的。"

散会后，林深坐在办公室里，看着窗外的城市。

这是他离开了三年的城市。

现在，他回来了。

而且，要面对那个他爱了那么多年的人。
```

### 示例二：陆行舟的心神不宁
```
陆行舟在办公室里，无法专注于工作。

"陆总，您还好吗？"秘书担心地问，"您今天看起来……心神不宁的。"

"没事。"陆行舟说，"对了，那个项目的招标……进展如何？"

"已经有几家公司报名了。"秘书说，"其中一家是星河设计，他们的设计总监是刚从巴黎回来的林深。"

陆行舟的手僵住了。

"林深？"

"是的，听说他在巴黎很有名，作品也很优秀。"秘书说，"不过他脸上好像有伤疤……"

陆行舟的心狠狠一抽。

伤疤？

林深有伤疤？

怎么回事？

"陆总？您怎么了？"秘书担心地问。

"没事。"陆行舟说，"你先出去吧。"

秘书点头，走出办公室。

陆行舟坐在那里，久久没有动。

林深回国了。

林深要参与竞标。

这意味着……他要见到林深了？

这三年来，陆行舟无数次想过林深。

想过林深在巴黎过得怎么样，想过林深是否还记得他，想过林深是否还恨他。

但他从来没有想到，有一天，林深会突然出现在他的生活中。

"林深……"陆行舟喃喃道。

他不知道的是，林深也在想他。

也在想即将到来的见面。
```

### 示例三：准备竞标
```
林深开始准备竞标方案，加班加点地工作。

办公室里，只有他一个人还在工作。

电脑屏幕的光照亮他的脸，左脸上的伤疤若隐若现。

"林设计师，您要不要休息一下？"同事小王路过，关心地问。

"不用了，我需要尽快完成方案。"林深说。

"但您已经工作十个小时了……"

"没关系。"林深说，"这个项目很重要，我必须尽力。"

小王叹了口气，"那您至少喝点水，吃点东西。"

"好，谢谢。"

林深喝了一口水，继续工作。

他不知道自己为什么这么拼。

也许是想证明什么——证明他已经不是三年前的那个林深了。

也许是……想见到陆行舟。

三年了。

他终于要见到陆行舟了。

林深闭上眼睛，深吸一口气。

"林深，你可以的。"他对自己说，"一定可以。"

同一时间，陆行舟也在办公室里，心神不宁。

他时不时看手机，想给林深发消息，但最终没有。

林深已经删除了他的联系方式，他怎么联系？

陆行舟只能等待，等待竞标会的那一天。

"林深……"陆行舟看着窗外，"你回来，是要做什么？"

陆行舟不知道的是，林深也在想他。

也在想即将到来的见面。

两个人，在不同的地方，想着同一个问题。

竞标会那天，会发生什么？
```

  - 隐层：双方的不安

- **情感节奏**：
  - 开场：回国后的新生活
  - 中段：得知陆氏项目
  - 高潮：陆行舟的心神不宁
  - 收尾：准备竞标

## 【写作注意事项】

1. **《针锋对决》风格要点**：
   - 重逢前的紧张要通过细节体现
   - 双方的心理活动要真实
   - 竞标的准备要专业

2. **细节把控**：
   - 办公室场景要写出工作氛围
   - 陆行舟的心神不宁要通过动作体现
   - 林深的伤疤要自然地提及

3. **节奏控制**：
   - 本章是铺垫性章节，节奏要从日常到紧张
   - 情感层次要丰富，不要单一

4. **伏笔设置**：
   - 为第28章的重逢做铺垫
   - 为后续的冲突埋下伏笔

5. **避免事项**：
   - 不要让林深显得过于紧张
   - 不要让陆行舟显得过于崩溃
   - 不要过度渲染悬念，保持克制
