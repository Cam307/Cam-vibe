## 【对话示例】

### 示例一：早上
```
阳光洒进房间，两人醒来。

"早安，老公。"林深说。

"早安，老公。"陆行舟笑了。

两人相视而笑，心中充满幸福。

"几点了？"陆行舟问。

"七点。"林深看了看闹钟。

"该起床了。"陆行舟说。

"嗯。"林深点头。

两人一起起床，一起洗漱。

"你做什么早餐？"陆行舟问。

"煎蛋。"林深说。

"那我做粥。"陆行舟说。

两人一起做早餐，配合默契。

林深煎蛋，陆行舟熬粥。

"要香菜吗？"林深问。

"不要。"陆行舟说，"你知道我不吃香菜。"

"我知道。"林深笑了，"逗你的。"

两人相视而笑，早餐很快做好了。

两人一起吃早餐，聊着天。

"今天要做什么？"陆行舟问。

"工作。"林深说，"你呢？"

"我也是。"陆行舟说。

"那晚上见。"

"晚上见。"

两人吃完早餐，一起出门上班。

这就是他们的婚后日常。

平凡而幸福。
```

### 示例二：午休
```
陆行舟来接林深吃饭。

"今天工作怎么样？"陆行舟问。

"还好。"林深说，"有个新项目，还在筹备中。"

"那你......累不累？"

"不累。"林深笑了，"你呢？"

"我也很忙。"陆行舟说，"但只要见到你，就不累了。"

"油嘴滑舌。"林深笑了。

"我是认真的。"陆行舟说。

两人来到餐厅，点好菜。

"你......最近还好吗？"陆行舟问。

"很好。"林深点头，"你呢？"

"我也很好。"陆行舟说。

"那就好。"林深笑了。

两人继续吃饭，聊着工作，聊着生活。

"老公。"林深忽然说。

"嗯？"陆行舟抬头。

"我爱你。"林深说。

陆行舟的心狠狠一跳。

"我也爱你。"陆行舟说。

"一直爱你。"

"我也是。"林深笑了。

两人相视而笑，继续吃饭。

这就是他们的婚后日常。

平淡而甜蜜。
```

### 示例三：晚上
```
两人一起回家。

"今天......很开心。"林深说。

"我也是。"陆行舟说。

"每天都很开心。"

"因为......和你在一起。"

林深的心狠狠一跳。

"我也是。"林深说。

"因为......和你在一起，我也很开心。"

两人相视而笑，继续走着。

回到家，两人一起做晚餐。

"今天做什么？"陆行舟问。

"简单的。"林深说，"炒菜，汤。"

"好。"陆行舟说。

两人一起做晚餐，很快就好了。

两人一起吃饭，聊着天。

"明天......也是这样吗？"陆行舟问。

"什么？"

"一起上班，一起吃饭，一起回家。"陆行舟说。

"嗯。"林深点头，"每一天都是这样。"

"那......以后呢？"

"以后也是。"林深笑了，"一辈子都是这样。"

陆行舟的心狠狠一跳。

"真的？"陆行舟问。

"真的。"林深点头，"一辈子。"

两人相视而笑，心中充满幸福。

这就是他们的婚后日常。

平凡，但......

幸福。

第74章，完。
```

   - 上班的路上

   - 情感层次要丰富

4. **伏笔设置**：
