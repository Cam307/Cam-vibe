## 【对话示例】

### 示例一：三年后
```
"今天是什么日子？"陆行舟问。

"我们的结婚三周年纪念日。"林深笑了。

"对。"陆行舟说，"三年了。"

"是啊，三年了。"林深感慨。

"时间过得真快。"

"是。"陆行舟点头，"但每一天都很幸福。"

"我也是。"林深笑了。

两人相视而笑，心中充满感慨。

三年了。

他们结婚三年了。

这三年里，他们经历了太多。

但最终......他们都幸福了。

"老公。"林深忽然说。

"嗯？"

"谢谢你。"林深说。

"谢什么？"陆行舟问。

"谢谢你......给我一个家。"林深说。

"谢谢你......让我幸福。"

陆行舟的心狠狠一跳。

"不用谢。"陆行舟说。

"是你......让我幸福了。"

两人相视而笑，紧紧相拥。

这三年......

是幸福的三年。
```

### 示例二：回顾照片
```
两人一起翻看相册。

"这是我们的婚礼。"林深指着一张照片。

照片里，两人穿着深蓝色的西装，手牵手。

"我们那时候......真年轻。"陆行舟笑了。

"现在也不老。"林深说。

"是。"陆行舟笑了，"但三年过去了。"

"嗯。"林深点头，翻到下一页。

"这是我们蜜月在巴黎。"

照片里，两人站在埃菲尔铁塔下，拥吻。

"那时候......很幸福。"陆行舟说。

"现在也很幸福。"林深说。

"对。"陆行舟笑了，"现在也很幸福。"

"这是我们结婚一周年。"

"这是我们结婚两周年。"

一张张照片，都是回忆。

都是......幸福的回忆。

"我们......真的很幸福。"陆行舟说。

"是。"林深点头，"真的很幸福。"

两人相视而笑，继续翻看相册。

每一张照片，都是他们爱情的见证。

都是......他们幸福的证明。
```

### 示例三：约定未来
```
两人吃完饭，散步回家。

"以后的每一年，都要一起过。"陆行舟说。

"好。"林深点头，"结婚四周年，五周年......"

"十周年，二十周年......"陆行舟接着说。

"一直到老。"林深说。

"对。"陆行舟笑了，"一直到老。"

"等你白了头发，我也白了头发。"

"我们还在一起。"

"还一起过纪念日。"

"好。"林深笑了，"一言为定。"

"一言为定。"陆行舟说。

两人继续走着，月光洒在他们身上。

"老公。"林深忽然说。

"嗯？"

"我爱你。"林深说。

"我也爱你。"陆行舟说。

"一直爱你。"

"我也是。"林深笑了。

两人相视而笑，心中充满幸福。

他们约定了。

以后的每一年，都要一起过。

一直到老。

第75章，完。
```
