## 【上下文约束】

**前情提要**：
第33章结尾，陆行舟在餐厅看着林深和赵泽相谈甚欢，自己像个局外人。三人午餐结束后，陆行舟独自离开。他无法忍受林深和别人的亲密，但他又没有立场质问。

## 【核心情节】

陆行舟从顾阳口中得知，赵泽是林深的"未婚夫"。这个消息像晴天霹雳，陆行舟彻底崩溃。他想象着林深和赵泽在一起的画面，心如刀割。他找顾阳喝酒，醉后吐真言："他怎么能和别人在一起......我们......我们认识七年了......"

## 【对话示例】

### 示例一：顾阳的消息
```
陆行舟约顾阳出来喝酒。

酒吧很吵，音乐震耳欲聋。

"那个赵泽，到底是什么人？"陆行舟开门见山。

顾阳愣了一下："怎么突然问这个？"

"今天我去林深工作室，看到他们在一起。"陆行舟喝了一口酒，"关系很亲密。"

"哦......"顾阳明白了。

"所以，他到底是什么人？"陆行舟追问。

顾阳犹豫了一下。

"听说是林深的未婚夫。"

"什么？！"

陆行舟手中的酒杯差点掉在地上。

"未婚夫......他们要结婚了？"

"应该是吧。"顾阳说，"他们认识三年了，在巴黎就在一起了。"

陆行舟的脸色瞬间变得苍白。

"三年前......"陆行舟喃喃自语。

那就是林深离开的时候。

那就是林深最痛苦的时候。

那个时候，赵泽就在他身边？

"他们......真的要结婚了吗？"陆行舟问。

"应该是。"顾阳说，"我听宋远说过，他们正在准备婚礼。"

陆行舟闭上眼睛，心中涌起剧烈的痛。

林深......要结婚了？

和别人？

不是和他？

陆行舟的手握紧了，指节泛白。

他失去了。

他彻底失去了林深。
```

### 示例二：醉酒崩溃
```
陆行舟已经喝了很多酒。

顾阳想拦他，但拦不住。

"够了，陆总，你喝多了......"

"我没多！"陆行舟推开顾阳，"给我倒酒！"

"陆总......"

"我说给我倒酒！"陆行舟吼道。

顾阳叹了口气，给他倒了一杯。

陆行舟一口气喝完，然后把酒杯重重放在桌上。

"他怎么能和别人在一起......"陆行舟说，声音沙哑。

"陆总......"

"我们......我们认识七年了......"陆行舟的眼泪掉下来，"七年......"

"我爱他......我真的爱他......"

"但我从来没有说过......"

"我错过了......我全都错过了......"

陆行舟哭得像个孩子。

"我不该订婚......我不该让他走......"

"我应该在机场拦住他......"

"我应该告诉他我喜欢他......"

"但我没有......我什么都没有做......"

"现在我失去他了......"

"他要和别人结婚了......"

"他要幸福了......但不是和我......"

顾阳看着陆行舟，叹了口气。

他从来没见过陆行舟这个样子。

从来都没有。

陆行舟一向骄傲、自信、掌控一切。

但现在......

他只是一个失去爱人的人。

一个彻彻底底的失败者。
```

### 示例三：林深那边
```
林深坐在沙发上看书，赵泽在旁边煮咖啡。

"今天陆行舟好像很难过。"赵泽说。

林深翻了一页书，没有抬头。

"那跟我无关。"

"真的无关吗？"赵泽端着咖啡走过来，"深哥，你今天看到他的时候，手在发抖。"

林深顿住了。

"我......"

"你还爱他，对不对？"赵泽问。

林深沉默了很久。

"爱？"林深笑了，"赵泽，你知道什么是爱吗？"

"爱......爱是......"

"爱是付出，是牺牲，是......是能够放手。"林深说。

"那你能放手吗？"赵泽问。

林深没有回答。

他不知道。

他不知道自己能不能真的放手。

他只知道，他累了。

七年暗恋，三年苦难，如今......无尽的痛苦。

林深真的累了。

"深哥。"赵泽说，"不管你做什么决定，我都支持你。"

"谢谢。"林深说。

"但我想告诉你......"赵泽看着林深，"你值得幸福。"

"不管是和他，还是......和别人。"

林深看着赵泽，最终笑了。

"谢谢你，赵泽。"

"不客气。"赵泽也笑了，"毕竟，我是你的......未婚夫嘛。"

林深愣了一下，然后也笑了。

是啊，未婚夫。

一个假的未婚夫。

但有时候，假的比真的更温暖。
```

## 【情感基调】

- **整体基调**：痛苦、崩溃、对比
- **情感层次**：
  - 表层：陆行舟的醉酒崩溃
  - 深层：林深的复杂情感
  - 隐层：赵泽的秘密身份

- **情感节奏**：
  - 开场：陆行舟的痛苦想象
  - 中段：顾阳带来"未婚夫"的消息
  - 高潮：陆行舟的醉酒崩溃
  - 收尾：林深那边的平静对比

## 【写作注意事项】

1. **《针锋对决》风格要点**：
   - 陆行舟的崩溃要有力，但不要过于歇斯底里
   - 林深的冷淡要有层次
   - 对话要有目的

2. **细节把控**：
   - 陆行舟醉酒时的细节
   - 顾阳的反应和担忧
   - 林深和赵泽的互动细节

3. **节奏控制**：
   - 本章是情感爆发章节，节奏要紧凑
   - 情感层次要丰富

4. **伏笔设置**：
   - 赵泽的"假未婚夫"身份伏笔
   - 林深对陆行舟的真实情感
   - 为后续剧情做铺垫

5. **避免事项**：
   - 不要让陆行舟显得过于可悲
   - 不要让林深显得过于冷漠
   - 保持平衡，每个人都有自己的痛苦
