### 🧠 第一步：先让AI「学会」写好小说

很多人上来就让AI写，出来的东西根本不能看，为什么？因为你的prompt太垃圾了❌

我的做法是：
- 因为自己本身就爱看小说，知道什么样的内容能吸引读者
- 把**1400多篇**知乎盐选爆款小说批量下载，喂给AI
- 让AI反复**总结、提炼、分析**优质小说的：
  - 🖋️ 语言风格
  - 🎵 叙事节奏
  - 👤 人物塑造
  - 🌆 环境描写
- 最终让AI抽象出**结构范式模板**

> ⚠️ 说实话，这一步真的需要些文学功底，你的prompt质量直接决定了AI能不能把好小说的「骨架」拆出来

---

### 🏗️ 长篇小说｜四阶段流水线

我搭了一个超级完整的创作流程系统，四个阶段像工厂流水线一样运作：

```
选题生成 → 细纲生成 → 章节内容创作 → 全局检查评价
```

**每个阶段都有专门的规范文档：**

📌 **阶段一：选题生成**
- 选题生成规范 → 告诉AI什么样的选题有爆款潜力
- 选题输出模板 → 标准化输出格式

📌 **阶段二：细纲生成**
- 人物设定规范 → 角色立体化的标准
- 大纲生成规范 → 故事骨架怎么搭
- 细纲扩充规范 → 把骨架填充出血肉

📌 **阶段三：章节内容创作**
- AI章节创作规范 → 每一章怎么写
- 小说章节输出模板 → 标准化输出
- 自我检查规范 → 写完自动质检

📌 **阶段四：全局检查**
- 每写完5章自动检查逻辑衔接和情节连贯性
- 还有全面检查规范兜底

**操作流程就是：**
喂一篇爆款短篇 → AI按选题规范生成选题 → 挑一个 → 自动走细纲→章节→自检 全流程 🎯

目前已经用这套系统写出了两部完整长篇，其中一部写了100多章！

---

### ⚡ 短篇小说｜改写系统一条龙

短篇更暴力，直接搭了个**四步改写系统**：

```
改写规范 → 输出模板 → 改写检查 → 正文字数检查
```

操作方法：
1. 找一篇爆款短篇小说
2. 喂给AI
3. 让它跑一遍整个改写系统
4. 自动输出一篇完整短篇小说 ✅

就这么简单，把爆款喂进去，改写系统跑一圈，成品出来直接能交稿

---

### 🔧 核心工具：两个必备Skill

最后说一下我用到的关键工具：

1️⃣ **Planning Skill**：帮AI做任务规划和流程拆解，让创作有条不紊

2️⃣ **Ralph Loop Skill**：这个是防断片神器！
- AI写长文容易写到一半就停了（断片）
- Ralph Loop会持续监控执行状态
- 没达到条件就一直跑下去
- 完美解决字数不足、中途断片的问题

> ⚡ 但要谨慎使用，因为它会一直执行，不达目的不罢休，所以一定要设好终止条件！

### 💡 最后说几句大实话

1. **AI是工具不是万能的**，你的审美和判断力才是核心竞争力
2. **prompt工程是核心技能**，同样的AI，不同人用效果天差地别
3. **系统化思维很重要**，不要写一篇是一篇，要搭流程
4. **对标学习是捷径**，先拆解爆款再创作，事半功倍
