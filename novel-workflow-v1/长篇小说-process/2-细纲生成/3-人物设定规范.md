# AI人物设定规范文档

> 版本：v1.0
> 创建日期：2026-02-15
> 用途：基于选题卡片，生成详细的人物设定
> 位置：在大纲生成之前，细纲扩充之前

---

## 一、使用说明

### 1.1 文档用途
在确定选题后，AI必须基于本规范生成**详细的人物设定**，包括所有出场人物、人物关系、人物小传等。

### 1.2 使用流程
```
选题卡片确定
  ↓
AI执行：基于本规范生成人物设定
  ↓
AI输出：完整的人物设定文档
  ↓
进入下一步：大纲生成
```

### 1.3 核心原则
```markdown
1. 人设必须符合番茄爆款风格
2. 人设必须有鲜明记忆点
3. 人设必须有反差（表面vs真实）
4. 人物关系必须清晰
5. 人物小传必须详细
6. 【命名与场景】所有人物姓名及设定中涉及的具体地点/场景名**不得与对标短篇小说中的人名、场景重复**；若选题基于某短篇扩展，创作人设时须使用选题卡片中已确定的新名称，或另行新创，禁止沿用该短篇中的任何角色名或具体场景/地点名。
```
