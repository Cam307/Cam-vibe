# 小说章节输出模板

> 版本：v1.2
> 创建日期：2026-02-15
> 用途：每一章创作完成后，按照本模板输出特定排版的内容

---

## 📘 章节输出标准模板

**AI必须严格按照以下格式输出每一章：**

**正文格式硬性要求：**
- **对话引用格式**：正文中**所有对话必须使用双引号 ""**，不得使用直角引号「」，以符合番茄小说等平台的对话格式规范。示例：正确为 "那个……要演多久啊？" 错误为 「那个……要演多久啊？」
- **正文为可直接复制粘贴的纯叙述内容**：正文中**不得出现**任何场景标题（如 "### 场景2：食堂二楼小餐厅"）、不得出现小标题（如 "#### 开场""#### 发展""#### 高潮""#### 结尾" 等）。仅用**空行**区分段落与场景转换，使正文复制到番茄小说等平台即可使用，无需再删改。
- 正文中**不得使用**「---」分隔线。
- **【字数硬性要求】** 每章**正文内容不少于 4000 字（不含标点）**。字数统计：不含标点符号，不含本章信息、要素统计、承上启下、下章预告等元信息。**输出前必须自行统计正文字数（去除标点）；若不足 4000 字，必须在本章内补写（增加对话、动作、心理、环境或情节细节）直至 ≥4000 字后再输出，未达标不得视为本章完成。**

```markdown
# 第X章：[章节标题]

**【本章信息】**
- 字数：XXXX字（正文不少于4000字，不含标点）
- 创作时间：YYYY-MM-DD HH:MM
- 基于细纲：第X章细纲
- 场景数量：X个
- 出场人物：[列表]

## 正文

[纯叙述正文：环境描写、人物登场、对话、动作、心理、情节推进、情绪爆发、章尾钩子等，全部以连续段落呈现。多场景之间仅用空行分隔，不写"### 场景X：xxx"或"#### 开场/发展/高潮/结尾"。对话一律使用双引号 ""。本章结尾段落须包含钩子。]


## 章尾钩子

[若钩子已写在正文最后一段，此处可简要复述或写「见正文末段」；否则在此写出章尾钩子内容，且对话仍用双引号 ""。]

## 本章要素统计

- **对话占比：** XX%
- **场景数量：** X个
- **出场人物：** [列出所有出场人物]
- **爽点类型：** [无/小爽点/中爽点/大爽点] - [具体说明]
- **钩子类型：** [悬念/反转/危机/身份/情感] - [具体说明]
- **情感推进：** [如有情感推进，简要说明]

## 承上启下

**承接上章：**
- [本章如何承接第X-1章]

**开启下章：**
- [本章如何为第X+1章铺垫]

## 作者备注（可选）

[本章创作心得、特别说明、需要注意的点等]

## 下章预告

[第X+1章的核心情节，吸引读者追更，50-100字]

**【本章完】**
```

- [ ] 排版整洁规范
- [ ] 正文为纯叙述，无 "### 场景X：xxx"、无 "#### 开场/发展/高潮/结尾"

## 💡 输出示例

```markdown
# 第1章：离婚协议

**【本章信息】**
- 字数：XXXX字（正文不少于4000字，不含标点）
- 创作时间：2026-02-15 21:00
- 基于细纲：第1章细纲
- 场景数量：2个
- 出场人物：林浅、顾寒川、苏晴、老陈

## 正文

雷声轰鸣！暴雨如注。窗玻璃上满是水雾，模糊了外面的世界。

客厅里，灯火通明。但气氛冷得掉冰渣。茶几上，一份文件孤零零地放着。《离婚协议书》五个黑字，格外刺眼。

林浅坐在沙发上，背脊挺直，双手放在膝盖上，脸上看不出任何情绪。只有她自己知道，她的心正在一点点冷下去。顾寒川站在窗边，背对着她，看着窗外暴雨，一言不发，仿佛她根本不存在。

"顾总。"苏晴率先开口，声音甜甜的却带着刺，"顾总都等这么久了，林浅姐还不签字吗？"

林浅抬眼，淡淡地看了她一眼。"急什么？"声音平静，听不出任何情绪。苏晴脸上的笑容僵了僵。

"林浅，"顾寒川转过身，声音冷得掉冰渣，"签了它。"他走到茶几旁，修长的手指点在协议上。"签完，拿着一千万，滚出顾家。"

林浅看着那个她爱了三年的男人，冷漠、不耐烦，仿佛她是垃圾一样。她站起来，拿起笔，笔尖落在纸上，刷刷刷，签得飞快。

顾寒川愣了一下。"你……这么干脆？" "怎么？顾总舍不得？"林浅轻笑。"那就签吧。"她抬眼看他，"不过顾寒川，你会后悔的。"顾寒川皱眉冷笑："后悔？我这辈子最后悔的，就是娶了你！"林浅没说话，低头把最后一页也签了。"签完了。"协议被扔回茶几上。她转身就走，没有哭没有闹，甚至没有回头。顾寒川看着她的背影消失在楼梯转角，心里突然空了一块。


暴雨还在下，雷声轰鸣。林浅站在雨中，全身湿透。一辆黑色轿车缓缓驶来，停在路边。车窗摇下，老陈撑着伞下车，冲她微微弯腰："大小姐，董事长在等您。"

林浅笑了。她钻进车里，车门关上，将顾家别墅隔绝在雨幕之外。

"大小姐，"老陈从副驾驶回头，"陆氏集团已经准备好了，明天您可以正式上任。"林浅拿出手机打开相册——一张照片上她和一个中年男人合影，下方写着：陆氏集团董事长陆震天，与独女陆浅。林浅关上手机，嘴角勾起。

就在这时手机响了。来电显示：陆氏董事长。林浅接起。"爸。" "浅浅，"电话那头陆震天的声音有些颤抖，"你弟弟……找到了。"林浅手机一震！"在哪？！" "就在……顾家。"林浅彻底僵住。顾家？！她弟弟在顾家？！

紧接着第二条短信来了：【大小姐，少爷被顾寒川软禁了！！！】林浅的指甲深深掐进掌心。顾寒川，你关我弟弟？好，很好！！！


## 本章要素统计

- **对话占比：** 45%
- **场景数量：** 2个
- **出场人物：** 林浅、顾寒川、苏晴、老陈
- **爽点类型：** 小爽点 - 女主反击，签得飞快
- **钩子类型：** 身份式钩子 + 悬念式钩子 - 女主是陆氏千金 + 弟弟在顾家
- **情感推进：** 女主从被抛弃 → 暗爽（复仇开始）

## 承上启下

**承接上章：**
- 无（第一章）

**开启下章：**
- 女主发现弟弟在顾家，为第2章埋下伏笔

## 作者备注

本章作为开篇，完成了：
1. 强冲突开场（离婚）
2. 女主反击（不哭不闹）
3. 身份暗示（陆氏千金）
4. 章尾强钩子（弟弟在顾家）

希望能吸引读者继续追更。

## 下章预告

第2章：华丽归来

女主将以什么身份归来？
弟弟为什么在顾家？
男主会如何反应？
敬请期待！

**【本章完】**
```

## 📗 本批剧情摘要（每批 3 章全部写完后输出一次）

**每批 3 章正文全部输出完毕后，AI 必须按以下格式输出一次「本批剧情摘要」，供下一批创作时作为上下文使用。**

```markdown
## 本批剧情摘要

**【本批范围】**
- 章节范围：第 X 章 – 第 Y 章（共 3 章）
- 本批标题：[第X章标题]、[第Y-1章标题]、[第Y章标题]

**【主要情节推进】**
1. [情节1，1–2 句]
2. [情节2，1–2 句]
3. [情节3，1–2 句]
（共 2–5 条）

**【人物状态 / 关系变化】**
- 女主：[情绪、心理、处境、与男主/配角关系变化]
- 男主：[情绪、心理、态度、与女主/配角关系变化]
- 重要配角：[如有明显变化则写，否则可略]

**【伏笔与未解悬念】**
- [与本批相关且影响后续的伏笔或悬念 1]
- [伏笔或悬念 2]
（供下一批衔接用）

**【本批爽点 / 钩子简要】**
- 第 X 章：爽点 [无/小/中/大] - [类型或一句话]；章尾钩子 [类型] - [一句话]
- 第 X+1 章：爽点 […] ；章尾钩子 […]
- 第 Y 章：爽点 […] ；章尾钩子 […]

**【时间线 / 场景线】**
- 时间跨度：[本批故事发生的时间范围，如「离婚当日傍晚 – 次日清晨」]
- 主要场景顺序：[场景1] → [场景2] → [场景3] …
```

## 📐 模板使用说明

### 对于AI

1. **每次输出章节时**：
   - 必须使用本模板
   - 必须严格按照模板格式输出
   - 不能遗漏任何要素

2. **正文格式（可直接复制粘贴使用）**：
   - 正文为纯叙述内容：不写 "### 场景X：xxx"、不写 "#### 开场/发展/高潮/结尾" 等任何场景标题或小标题
   - 场景之间、段落之间仅用空行分隔，正文中不得使用「---」分隔线
   - 对话一律使用双引号 ""，不得使用直角引号「」（符合番茄小说平台规范）

3. **章节结构**：
   - 开头：章节信息
   - 正文：按场景组织
   - 结尾：要素统计 + 承上启下 + 作者备注 + 下章预告

4. **必须包含的要素**：
   - 章节信息（字数、时间、人物等）
   - 正文（纯叙述段落，可直接复制粘贴使用，无场景/小标题）
   - 章尾钩子
   - 要素统计
   - 承上启下
   - 下章预告

5. **分批输出**：
   - 每批只输出 3 章；本批 3 章全部写完后，必须再输出一次「本批剧情摘要」（格式见上方「本批剧情摘要」区块）。
   - 下一批创作时，将上一批的「本批剧情摘要」作为上下文提供给 AI。

### 对于用户

1. **查看输出**：
   - AI会按照本模板输出每一章
   - 格式统一，便于查看
   - 要素齐全，便于检查

2. **保存格式**：
   - 建议以Markdown格式保存
   - 每章一个文件
   - 文件名格式：`第X章_[章节标题].md`

3. **发布前处理**：
   - 正文已为可直接复制粘贴的纯叙述（无场景标题、无小标题，对话为双引号），复制「## 正文」与「## 章尾钩子」之间的内容即可用于番茄小说等平台
   - 若需精简：可删除「【本章信息】」「本章要素统计」「承上启下」「作者备注」等，仅保留正文与下章预告
